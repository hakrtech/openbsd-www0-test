OpenBSD/sun3

History and Status:

The tiny bits of OpenBSD that are completely specific to the Sun3 port are basically the same as their NetBSD counterparts but with some additions. The major changes are:

  • default data resource limit raised to 16384 Kbytes so that "make build" does not crash in libc.
  • /dev/leds derived from NetBSD pr#2172 by der Mouse.

Snapshots are not available. If you want to use OpenBSD/sun3 get an Amiga (or any other m68k OpenBSD) snapshot and bug me for the GENERIC kernel and bootstrap programs.
