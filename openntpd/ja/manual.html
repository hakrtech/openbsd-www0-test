マニュアルページ

Web 上のマニュアルページは OpenBSD サイト からご覧になれます。 これらのマニュアルページは OpenNTPD の最新の開発リリースのものです。

  • ntpd - Network Time Protocol デーモン
  • ntpd.conf - Network Time Protocol デーモン 設定ファイル

OpenNTPD で実装されている NTP プロトコルは RFC 1305 および RFC 2030 で記述されています。 NTP プロトコルの参考用の実装 (reference implementation) と、これ以外の文書は ntp.org でご覧になれます。
