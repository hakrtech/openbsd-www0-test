Leitfaden

Ein(e) Leitfaden/Bedienungsanleitung via man pages sind von OpenBSD erhältlich. Diese man pages beschreiben den aktuellsten Entwicklungsstand von OpenNTPD.

  • ntpd - Network Time Protocol daemon
  • ntpd.conf - Network Time Protocol daemon Konfigurationsdatei
Das NTP Protokoll, das in OpenNTPD implementiert ist, wird in RFC 1305 und RFC 2030 beschrieben. Eine Referenzimplementierung und weitere Dokumentation des NTP Protokolls ist unter ntp.org verfügbar.
