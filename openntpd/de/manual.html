Handbuch/Manual pages

Web manual pages sind über OpenBSD verfügbar. Diese manual pages beziehen sich auf den letzten Entwicklungsstand von OpenNTPD.

  • ntpd - Network Time Protocol daemon
  • ntpd.conf - Network Time Protocol daemon Konfigurationsdatei
Das NTP Protokoll, das im OpenNTPD implementiert ist, wird in RFC 1305 und RFC 2030 beschrieben. Eine Referenz Implementierung und andere Dokumentation des NTP Protokolls ist bei ntp.org verfügbar.
