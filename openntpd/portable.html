Portable OpenNTPD

Normal OpenNTPD development produces a very small, secure, and easy to maintain version for the OpenBSD project. The portability team takes that pure version and adds portability code so that OpenNTPD can run on many other operating systems.

The OpenNTPD provided here is known to run on the following Unix operating systems:

  • FreeBSD
  • NetBSD
  • Linux
  • Solaris
  • Mac OS X
  • HP-UX

The portable OpenNTPD follows development of the official version, but releases are not synchronized. Portable releases are marked with a 'p' (e.g. 3.6p1). The official OpenBSD source will never use the 'p' suffix.
