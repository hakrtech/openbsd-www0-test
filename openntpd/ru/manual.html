На Web сайте проекта OpenBSD доступны man страницы для последней версии OpenNTPD.

  • ntpd - сервис NTP (Network Time Protocol)
  • ntpd.conf - описание формата конфигурационного файла для OpenNTPD-демона
Протокол NTP, реализованный в OpenNTPD, описан в RFC 1305 и RFC 2030. Материал о реализации, а так же другая документация о протоколе NTP, доступна на станице ntp.org.
