Pages du manuel

Les pages du manuel Web sont disponibles sur le site d'OpenBSD. Ces pages du manuel correspondent à la dernière version de développement d'OpenNTPD.

  • ntpd - service Network Time Protocol
  • ntpd.conf - fichier de configuration du service Network Time Protocol
Le protocole NTP qu'implémente OpenNTPD est décrit dans la RFC 1305 et la RFC 2030. Une implémentation de référence du protocole NTP ainsi que de la documentation sont disponibles sur ntp.org.
