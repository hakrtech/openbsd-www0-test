Mirroirs FTP et HTTP

Code Source de base d'OpenNTPD 3.7 : publié le 10 Décembre 2004

Le code source de base est conçu spécifiquement pour OpenBSD, pour le rendre simple, propre, minimal, et pouvant être audité. Une autre équipe modifie OpenNTPD vers une révision portable qui fonctionne sur tous les systèmes d'exploitation.

Nous vous le répètons : Pour les autres systèmes, consultez les fichiers de la révision portable !
