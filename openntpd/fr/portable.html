OpenNTPD Portable

Le développement classique de OpenNTPD produit une très petite version pour le projet OpenBSD, sécurisée et facile à maintenir. L'équipe en charge de la portabilité utilise cette version originale et ajoute du code de portabilité, OpenNTPD peut ainsi être utilisé sur de nombreux autres systèmes d'exploitation.

OpenNTPD ici fourni est connu pour fonctionner sur les systèmes d'exploitation Unix suivants :

  • FreeBSD
  • NetBSD
  • Linux
  • Solaris
  • Mac OS X
  • HP-UX

L'OpenNTPD portable suit le développement de la version officielle, mais les révisions ne sont pas en phase. Les révisions portables sont marquées avec le suffixe 'p' (e.g. 3.6.1p1). Vous ne verrez jamais la source officielle OpenBSD utiliser le suffixe 'p'.
