Mirrors

OpenNTPD 3.9 base source code: released May 11, 2006

The base source code is designed specifically for OpenBSD, to make it simple, clean, minimal, and auditable. A separate team converts OpenNTPD to a portable release which runs on all operating systems.

We repeat: For other systems, see the portable release files!
