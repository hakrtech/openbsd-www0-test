Handleidingen

Web handleidingen zijn beschikbaar via OpenBSD. Deze handleidingen reflecteren de nieuwste ontwikkelingsversie van OpenNTPD.

  • ntpd - Network Time Protocol daemon
  • ntpd.conf - Network Time Protocol daemon configuratiebestand
Het NTP protocol geïmplementeerd in OpenNTPD wordt beschreven in RFC 1305 en RFC 2030. Een referentie-implementatie en andere documentatie van het NTP protocol is beschikbaar via ntp.org.
