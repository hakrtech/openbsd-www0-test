Manual pages

Páginas web de manual estão disponíveis a partir do OpenBSD. Estas páginas de manual refletem a última versão de desenvolvimento do OpenNTPD.

  • ntpd - Network Time Protocol daemon
  • ntpd.conf - Network Time Protocol daemon arquivo de configuração
O protocolo NTP aplicado em OpenNTPD é descrito na RFC 1305 e RFC 2030. A implementação de referência e outros documentos do protocolo NTP está disponível a partil ntp.org.
