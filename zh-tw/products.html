OpenBSD 的商業產品

OpenBSD 鼓勵公司行號及個別發展者將 OpenBSD 做為他們的產品或是使用 OpenBSD 來 創作出新的產品。

OpenBSD 的商業支援可以到 我們的支援網頁上尋找相關的廠商。

  • OpenBSD 支援的軟硬體產品

    • 由 OpenBSD 發展而來的產品

  • ePipe by Stallion Technologies
    Stallion Technologies 在一個中小企業撥接網路閘道 ePipe 上搭載 OpenBSD 的嵌入式作業系統 。 這個閘道裝置主要是要取代小型商業公司高價的 VPNs 最末端網段及 應用程式廠商提供給小型商業公司的服務程式。 ePipe 採用 Motorola ColdFire 5307 VL-RISC 處理器，並使用 OpenBSD 內建的加密 軟體提供點對點的安全連線。

  • nCircle Network Security
    nCircle Network Security (前身是 Hiverworld) 超越了入侵偵測技術，進而提供智慧型入侵防範達到全面的網路 安全防護。他所採用的 IP360 專利技術提供了持續、無侵略性的弱點稽核，並結合了 全速的網路流量監控，入侵分析等技術。nCircle 以 OpenBSD 作業系統做為他的 掃描偵測、入侵偵測及控制端安全網路平台。

  • Enterasys Networks
    Enterasys Networks (已併購 Network Security Wizards) 研發了以 OpenBSD 平台為基礎的 Dragon IDS 系列產品。 Dragon 是採用封包為主的入侵偵測系統，並包含超過一千兩百多個入侵特徵比對值 的偵測產品。這些入侵特徵可以辨識出所有層面的網路異常，包括探測、攻擊及確認 遭遇入侵的偵測等功能。Dragon Squire 採用的主機型 IDS 包括主要系統檔的 MD5 值監控及系統記錄檔的分析。Dragon Server 包含廣汎的即時搜證偵查與分析工具， 並且包含以 Nessus 為基礎發展而來的相對應 IDS 事件即時弱點偵測報表。

  • RTMX
    RTMX 販售的是有支援 POSIX 即時作業系統 的 OpenBSD 版本。他們也很善意的提供了相關支援的原始程式碼，這些新的功能 將會很快的整合到 OpenBSD 中。

    • OpenBSD 支援的軟硬體產品

  • PowerCrypt by GTGI
    Global Technologies Group, Inc. 研發的 PowerCrypt, 是一個智慧型， bus-mastering PCI 板，其中整合了 Hifn 7751 加密處理器。系統核心層的支援 是由 OpenBSD 所研發， 由 OpenBSD 社群成員撰寫的驅動程式及 IPsec 原始程式碼所撰寫而成。

  • RedHat's Stronghold Secure Web Server
    RedHat 提供 128-bit 的加強密碼安全 HTTPS 伺服器程式，主要是基於被廣汎使用的 OpenBSD 版 Apache 網頁伺服器，並配合使用 贏得許多獎項的 Stronghold 3 安全網頁伺服器。

  • Open Sound System by 4Front (只支援 i386 平台)
    4Front Technologies 提供一個更好的商業 版本板 OpenBSD/i386 平台的音效卡驅動程式。這些驅動程式相容於 OpenBSD/i386 平台 所搭載的音效卡驅動程式，並且支援許多其他的音效卡，並且較能夠善用個別音效卡的 各種功能。 這個軟體相容於 OpenBSD/i386 3.0 版

  • Core Systems
    Core Systems 販售的 InSite, 是一個容易使用，伺服器端的網頁統計工具，支援 OpenBSD/i386 平台。InSite 是與 WebTrends 相似的產品，但他也提供了可藉由網頁修改調校的介面，可以飛快 地產生報表，並只佔據極少的 CPU 時間。(如果有特別需求的話，核心團隊成員可以 提供在 i386 平台以外的 InSite 版本)。

  • Software2Go Motif (只支援 i386 及 SPARC 平台)
    Software2Go, LLC 有 供 OpenBSD 使用的 Motif 2.1.20 Development 及 Runtime toolkits.
    這個研發中的版本，可供建構圖形使用者介面應用程式的軟體發展者，發展出 Motif 模式的圖形介面觀感及使用方式。 其中 runtime 版本在只有可執行檔的應用程式中，是必備的。
    聯絡: Software2Go, LLC 線上訂購， 817-431-8775 (電話/傳真)

  • Zeus Web Server (只支援 i386 平台)
    Zeus Technology 研發的 Zeus Web Server, 是世界上可擴充性最強的高效能網頁伺服器軟體，目前支援 OpenBSD/i386 2.8 及 2.9 版本。其中一小部份的優異功能是: 非常好的擴充性，即時的網頁介面統計資料，個別 伺服器的頻寬調節，安全的 CGI 執行環境，支援叢集伺服器群，以及 FastCGI 的支援。
