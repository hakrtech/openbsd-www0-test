OpenBSD/vax

History and Status:

The OpenBSD/vax port is currently being worked on. The code has been recently imported into the OpenBSD tree based on work from the NetBSD/vax port.

OpenBSD currently supports these vax models:

  • vax 4000/300
  • vaxstation/microvax 2000
  • vaxstation 3100/20
  • microvax 3300

Along with support for the VS3100 ncr5380 scsi driver.

Projects being worked on:

  • vaxstation 4000/60 (with its ncr53c94-derived scsi controller
  • vaxstation 2000's hdc9224 (floppy/mfm hard disk controller)
  • qbus tk50/tk70 tape drives
  • shac dssi support

The OpenBSD/vax port has not seen a formal release, however snapshots will be made from time to time.

The OpenBSD/vax port maintainer is Brandon Creighton.
