OpenBSD/pmax

OpenBSD/pmax fonctionne sur une large gamme de machines de la famille DECstation et DECserver à base de Digital MIPS. Ces machines fûrent les pendants des VAXstations à base de VAX, jusqu'à ce que la gamme Alpha soit prête.

The OpenBSD/pmax port a été arrêté après la sortie de la version 2.7..

A l'heure actuelle, personne ne travaille à ramener ce port à la vie. Le code source est encore disponible dans le grenier CVS, et du code peut être trouvé sous NetBSD aussi. Si vous souhaitez ranimer le port amiga, faites le fonctionner et contactez Todd Miller et Miodrag Vallat.

Historique :

Le port pmax d'OpenBSD a été démarré par Thomas Graichen juste avant la version 2.0 d'OpenBSD, en se basant sur les sources NetBSD/pmax sources. Il a été alors lentement modelé, devenant ainsi indépendant et fût passé à ECOFF puis à des binaires ELF.

Quelques versions plus tard, Thomas Graichen arrêta de travailler sur le port et personne ne le remplaça. Le port tomba lentement dans l'oubli, sans aucun nouveau code et très peu de corrections de bogues malgré que des versions sortaient encore.

Un effort timide fût effectué après la version 2.7 pour réanimer ce port, mais des problèmes de toolchain spécifiques à MIPS ont empêché le système d'être construit durant longtemps, et le développement s'est arrêté par la suite. Lorsque le problème fût résolu, il a été décidé néanmoins d'arrêter le support de ce port et de supprimer le code de l'arborescence des sources.

Matériel supporté :

Modèles supportés

  • DECstation 2100 (PMIN) et 3100 (PMAX)
  • DECstation 5000/120, 5000/125 et 5000/133 (3MIN)
  • DECstation 5000/200 (3MAX)
  • DECstation 5000/240 (3MAXPLUS)
  • Personal DECstation 5000/20, 5000/25, 5000/33 et 5000/50 (MAXINE)

Périphériques supportés

  • Framebuffers
    • PM monochrome (1024x864x1) ou couleur 8-bit r (1024x864x8) intégré sur PMIN et PMAX
    • PMAG-DV (1024x864x8) intégré sur Personal DECstation
    • TURBOchannel PMAG-AA (mfb) (1280x1024x1)
    • TURBOchannel PMAG-BA (cfb) (1024x864x8)
    • TURBOchannel PMAGB-BA (sfb) (1280x1024x8)
    XFree86 fonctionne sur tous les framebuffers supportés.
  • Ethernet
    • Lance ethernet intégrée
    • Lance ethernet TURBOchannel PMAD-AA ou PMAD-AB
  • Contrôleurs SCSI
    • Contrôleur SCSI (sii) sur PMIN et PMAX
    • Contrôleur SCSI (asc) sur les autres modèles
    • Contrôleur SCSI TURBOchannel PMAZ-AB (asc)
  • Périphériques divers
    • DEC LK-201 ou clavier compatible
    • DEC VSXXX ou souris compatible
    • Port série intégrés sur la plupart des modèles

Obtenir et installer OpenBSD/pmax :

La dernière version supportée d'OpenBSD/pmax fût la version OpenBSD 2.7. Elle n'est plus disponible par ftp. les versions jusqu'à la version 2.3 incluse étaient disponible sur CD.
