SMP

Histoire et Status :

Le projet visant à apporter le support SMP à OpenBSD a été initié en février 2000.

Le support SMP (Symmetric MultiProcessor) a été importé dans la branche de développement principale en juin 2004. A l'heure actuelle, les plates-formes i386, amd64, mvme88k, sparc64 et macppc supportent le SMP. Le port sgi supporte le SMP sur les modéles d'Octane multiprocesseurs, les autres machines ne le supportent pas encore. Le support SMP pour le port hppa est presque terminé, cependant plusieurs bogues continuent d'être présent ce qui ne permet pas un fonctionnement stable. Heureusement d'autres plates-formes, incluant sparc et alpha supporteront le SMP un jour.

Veuillez consultez la page de plate-forme pour le status du support SMP pour votre système.
