Vous pouvez également télécharger un fichier tar.gz contenant tous les correctifs suivants. Ce fichier est mis à jour une fois par jour.

Les correctifs ci-dessous sont disponibles sur CVS via la branche OPENBSD_5_0 stable.

Pour des informations plus détaillées à propos de l'installation de correctifs sur OpenBSD, consultez la FAQ OpenBSD.

  • 001: CORRECTIF DE FIABILITE: 30 Novembre 2011   Toutes les architectures
    Une vulnérabilité a été trouvée dans le serveur de nom de domaine BIND (CVE-2011-4313). Un événement réseau non identifié fait que le résolveur BIND 9 met en cache un enregistrement invalide, et par conséquent les requêtes peuvent crasher le résolveur avec une erreur d'assertion. Un correctif corrigeant le problème au niveau du code source est disponible.

  • 002: CORRECTIF DE SECURITE: 23 Avril 2012   Toutes les architectures
    Une vulnérabilité de dépassement de tas a été trouvée dans le code du parseur ASN.1 de libcrypto (CVE-2012-2110).
    Un correctif corrigeant le problème au niveau du code source est disponible.
