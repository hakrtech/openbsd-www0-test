Historique et Etat Actuel :

Il y a un port HP PA-RISC 2.x en phase de démarrage ayant pour but de supporter les stations de travail ainsi que les serveurs 64 bit HP à base de PA8x00.

Contactez mickey si vous désirez plus d'information.

Projects (sans ordre particulier) :

  • Importer le support hppa64 de gcc3.x
  • Verifier si la version de binutils dans -current est suffisante (elle devrait)
  • Cloner hppa/stand pour le support 64 bit
  • Cloner les fichiers du kernel arch/hppa pour le support 64 bit
  • Pilotes PCI
  • Support des systèmes multi processeurs

Matériel supporté :

Pour l'instant il n'y a pas de matériel supporté. Ci-dessous, la liste des modèles pour lesquels nous prévoyons un support.

Modèles non supportés

  • A base de PA8000 : C160/C180, D270/D370, D280/D380, J280/J282
  • A base de PA8200 : C200/C240, D390, J2280
  • A base de PA8500 : A400/A500, B1000, C360/C3000, J5000/J7000, L1000/L2000
  • A base de PA8600 : B2000/B2600, C3600, J5600/J6000, L3000/rp5400
  • A base de PA8700 : C3700, J6700, rp2400, N4000/rp7400, rp8400
