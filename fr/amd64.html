OpenBSD/amd64

OpenBSD/amd64 fonctionne sur les processeurs AMD de la famille Athlon-64 en mode 64-bits.

Ce port fonctionne aussi sur les futurs processeurs Intel ia32e qui supportent les extensions AMD64. Malheureusement, Intel a abandonné le support du bit NXE (No-EXecute) de la table de pagination. W^X n'est donc pas supporté sur les processeurs Intel.

Cette famille de processeurs fonctionne aussi avec OpenBSD/i386 mais uniquement en mode 32-bit.

Etat actuel :

Ce port peut s'installer et fonctionner avec une difficulté minimale sur le matériel le plus récent.

Le seul problème majeur subsistant à l'heure actuelle est la pauvreté relative du debogueur noyau.

Matériel supporté :

Processeurs

Toutes les versions des processeurs AMD Athlon 64 et leurs clones sont supportés.

Support SMP

A partir de la version OpenBSD 3.6, OpenBSD/amd64 supporte la plupart des systèmes SMP ("Symmetrical MultiProcessor"). Pour le fonctionnement en mode SMP, un noyau SMP séparé (bsd.mp) est fourni avec les ensembles de fichiers d'installation.

Cette fonctionnalité étant nouvelle, les développeurs souhaitent avoir un retour sur le matériel SMP qui fonctionne ou ne fonctionne pas. Il est donc très important d' envoyer un dmesg après l' installation.

Bus

  • Tous les ponts PCI standards
  • Universal Serial Bus (USB)

Interfaces pour bus

  • Ponts standards PCI-PCI, y compris les backplanes d'extension PCI (ppb)

Sources d'entropie

  • Récupération du temps de latence entre les interruptions venant de nombreux périphériques.

Les drivers (A) pour le matériel marqué avec (A) ne sont pas inclus dans l'image de boot des disquettes, mais le sont dans les images de boot des cd-rom.

Le support (*) pour les périphériques marqués avec (*) n'est ni inclus dans le média distribué, ni dans le noyau GENERIC, et requiert une compilation personnalisée pour l'activer.

* Support ajouté après la version 3.7.

Obtenir et installer OpenBSD/amd64

La dernière version disponible et supportée d'OpenBSD/amd64 est OpenBSD 3.7. Voici les instructions d'installation d'OpenBSD/amd64 3.7.

De temps à autre, des snapshots sont mis à disposition, à cet endroit ainsi que sur quelques miroirs. Et voici les instructions d'installation des snapshots OpenBSD/amd64 .

Plusieurs média d'installation sont fournis :

  • Démarrage à partir du CD
    Le démarrage à partir du CD fournit une image de disquette El Torito de 2.88MB incluant pratiquement tous les pilotes OpenBSD. Cette image inclut aussi un support USB minimal (périphériques de stockage et clavier).
    Pour la liste la plus récente des pilotes disponibles sur cette image, consultez le fichier de configuration noyau RAMDISK_CD.
  • Floppy A (floppy37.fs)
    Cette image de disquette de 1.44MB contient les pilotes les plus communs. Elle a été conçue pour le PC le plus usuel. En règle générale, les pilotes manquants sont ceux nécessitant des microcodes conséquents pour fonctionner tels que les pilotes de certaines cartes SCSI rares, des cartes gigabit ethernet, ou des périphériques RAID.
    Pour la liste la plus récente des pilotes disponibles sur cette image, consultez le fichier de configuration noyau RAMDISK.
