OpenBSD/sun3

OpenBSD/sun3 fonctionne sur les machines Sun3 à base de Motorola 68020, excepté la variante 3/E et le 3/50.

The OpenBSD/sun3 port a été arrêté après la sortie de la version 2.9.

A l'heure actuelle, personne ne travaille à ramener ce port à la vie. Le code source est encore disponible dans le grenier CVS, et du code peut être trouvé sous NetBSD aussi. Si vous souhaitez ranimer le port amiga, faites le fonctionner et contactez Miod Vallat.

Historique :

Le port Sun3 d'OpenBSD a été dérivé à partir du port NetBSD/sun3, quand les ports sun3 et sun3x étaient encore des entités séparées. Personne n'a démontré de l'intérêt pour OpenBSD/sun3 pour un bon moment, même si le code était en général gardé en synchronisation avec NetBSD.

Finalement, Kenneth Stailey a commencé à travailler sur le port mais vu qu'il vivait aux Etats-Unis, il ne pouvait pas exporter une version qu'il aurait lui-même faite. Il a donc perdu tout intérêt à travailler sur ce port.

Au début de l'année 2000, Miod Vallat a tenté de ramener ce port à un état correct, de faire en sorte que le code soit compilable à nouveau, et de concevoir un nouveau média d'installation. Le port fût mis à disposition avec un client X Window complet ainsi que des utilitaires bien qu'aucun serveur X ne fonctionnait.

Le port a vécu durant quelques versions, mais malheureusement des problèmes matériels et le manque de temps ont empêché de maintenir le code en synchronisation avec NetBSD, et le code sun3 n'était prêt pour passer de l'ancien système de mémoire virtuelle Mach à uvm. Il a ainsi raté la sortie de la version 3.0 et il y avait espoir que le code soit corrigé à temps pour la sortie de la version 3.1. Malheureusement, ce ne fût pas le cas et finalement il a été décidé de supprimer le code de l'arborescence des sources.

Matériel supporté :

Modèles supportés

  • 3/60, 3/60LE
  • 3/75, 3/140, 3/150, 3/160 et 3/180
  • 3/110
  • 3/260 et 3/280
OpenBSD ne fonctionnera pas sur la famille des machines 3/160 avec seulement 2MB de mémoire, et même très difficilement sur les machines 4MB quelque soit le modèle.

Périphériques supportés

  • Ports série intégrés
  • Souris et clavier Sun type 3
  • Vidéo intégrée ou VME video (bwtwo, cgtwo, cgfour)
  • Ethernet intégré ou VME (intel, lance)
  • Contrôleur SCSI sun-3 intégré ou VME, actuellement avec quelques limitations
