OpenBSD 3.2 : 1er novembre 2002


Pour commander un CDROM 3.2, cliquez ici.
  • Trois CDROMs dans une belle boîte.
  • Tous les composants d'installation pour SIX architectures : i386, alpha, macppc, vax, sparc, sparc64.
  • Les architectures suivantes ne sont disponibles que via une image ISOFS du système de fichiers téléchargeable sur le FTP : amiga, hp300, mvme68k, mac68k.
  • Les CDs sont bootables sur i386, alpha, macppc, sparc, et sparc64.
  • Une superbe jaquette à la fois drôle et artistique contenant les instructions d'installation. Les informations contenues sur cette jaquette rendent OpenBSD légèrement plus simple d'installation qu'une installation via un serveur FTP.
  • Une arborescence complète du code source (prête pour une usage via AnonCVS).
  • Les versions binaires de la dernière version stable de XFree86 pour toutes les architectures.
  • La dernière version du code source de XFree86 dotée de quelques modifications de notre part pour le rendre encore mieux.
  • Notre propre arborescence de ports qui a énormement évoluée depuis OpenBSD 3.1. Presque tous les packages fonctionnent sous la plupart des architectures.
  • Plusieurs binaires pré-compilés pour les architectures les plus communes, s'installant très facilement depuis le CDROM.
  • Comme d'habitude, des autocollants inclus!
  • Et plein d'autres choses...
