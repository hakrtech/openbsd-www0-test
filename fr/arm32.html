OpenBSD/arm32

Histoire et statut :

Le port arm32 n'est pas supporté dans OpenBSD ; du code est disponible dans NetBSD.

Il n'y a actuellement personne pour travailler sur un port arm32. Si quelqu'un a le matériel et est prêt à le mettre à jour, merci de contacter tech@openbsd.org pour démarrer.
