Disposition d'un miroir :

La structure des fichiers de chaque serveur FTP/AFS est en gros :

  • .../OpenBSD/2.7/
    Où vous trouverez notre dernière version (Pour chaque architecture, X11 est inclus si possible).
    • amiga/
    • hp300/
    • i386/
    • mac68k/
    • mvme68k/
    • mvme88k/
    • pmax/
    • powerpc/
    • sparc/
    • src.tar.gz
       Sources des programmes utilisateurs, que vous placerez dans /usr/src (ceci n'inclut pas les sources du noyau)
    • srcsys.tar.gz
       Les sources du noyau, que vous placerez dans /usr/src/sys
    • ports.tar.gz
       Les sources des ports, que vous placerez dans /usr/ports
    • X11.tar.gz
       Les sources de XFree86 X11

  • .../OpenBSD/snapshots/
    Pour nos architectures principales, nous construisons des mini versions dont la stabilité et la qualité sont inconnues environ tous les moins. C'est l'endroit où il faut rechercher ces versions.

  • .../OpenBSD/src/
    Il s'agit d'une version des sources par cvs faite tous les jours pour ceux qui veulent accéder aux fichiers individuellement. Normalement, nous recommandons l'usage de CTM ou AnonCVS plutôt que de faire un mirroir de cette partie.

  • .../OpenBSD/tools/
    Autres petites choses.
  • .../OpenBSD/2.6/
    Disposition similaire à la version 2.7 décrite ci-dessus...pour la nostalgie.
  • .../OpenBSD/2.5/
    Disposition similaire à la version 2.7 décrite ci-dessus...pour la nostalgie.
  • .../OpenBSD/2.4/
    Disposition similaire à la version 2.7 décrite ci-dessus...pour la nostalgie.
  • .../OpenBSD/2.3/
    Disposition similaire à la version 2.7 décrite ci-dessus...pour la nostalgie.

Les personnes désirant mettre en place de nouveaux miroirs FTP/ANONCVS seront intéressées par l'estimation de la place disque de ce qui est maintenu à ftp://ftp.openbsd.org:/pub/OpenBSD/.
