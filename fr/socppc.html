OpenBSD/socppc

OpenBSD/socppc fonctionne sur des machines basées sur la plateforme Freescale MPC8349E System-on-Chip (SoC) qui utilise Das U-Boot comme chargeur de démarrage. Actuellement cette plateforme supporte seulement le Thecus N1200, qui est un boitier NAS avec un seul disque dur basé sur le MPC8347 fonctionnant à 266MHz. Il a 128Mo de mémoire, un port Ethernet 1000baseT Ethernet, un switch intégré Ethernet 100baseTX, deux ports USB 2.0 et un port eSATA évidemment pour le disque SATA.

OpenBSD/socppc est compatible binairement avec OpenBSD/macppc; tous les paquetages existants pour OpenBSD/powerpc fonctionnent parfaitement sur OpenBSD/socppc.

Le mainteneur actuel de ce portage est Mark Kettenis (kettenis@openbsd.org). D'autres sont les bienvenus pour contribuer !

    • Historique du port

    OpenBSD/socppc est massivement basé sur OpenBSD/macppc. Le coeur e300 trouvé sur le MPC8349E est un dérivé du processeur MPC603e utilisé dans les premiers systèmes Apple Macinstosh à base de PowerPC, donc la plupart du code de bas niveau est identique.

    Etat actuel

    Le support matériel est presque complet. Les périphériques USB 1 ne sont seulement supportés que derriére un hub USB 2.0 et le support du switch intégré sur le Thecus N1200 reste manquant.

    Matériel supporté

      Pour une liste compléte des composants et des périphériques pour cette architecture, regardez intro(4/socppc).

    • cpu(4) Processeur Freescale MPC8347.
    • ipic(4) Contrôleur d'interrupetions programmable intégré (IPIC).
    • sociic(4) Contrôleur I2C.
    • fintek(4) Senseur de température Fintek F75375.
    • ricohrtc(4) Horloge temps réel Ricoh RS5C372.
    • com(4) Console port série à 115200 baud.
    • tsec(4) Contrôleur Ethernet three-speed (TSEC).
    • socpcic(4) Contrôleur PCI.
    • pciide(4) Contrôleur SATA supportant wd(4).
    • ehci(4) Contrôleur USB supportant la plupart des périphériques usb(4).
