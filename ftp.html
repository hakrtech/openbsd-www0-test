OpenBSD

The OpenBSD 2.0 Release:

The most recent OpenBSD release is 2.0. You can find these files in OpenBSD/2.0 on each listed FTP server.

The OpenBSD 2.1 Release is IMMINENT. The files for it will show up as soon as possible.

Binary Snapshots:

As well, snapshots will be made available from time to time on the ftp mirrors. The snapshots are available on each of the FTP servers listed below, typically in the directory OpenBSD/snapshots. If you plan to use a snapshot, you should also read the general description about OpenBSD snapshots.

NOTE: please do not fetch binaries directly from cvs.openbsd.org but instead use one of the mirror sites.

The file structure of each FTP server is roughly:

  • .../OpenBSD/2.0/
    -- where you find our most recent release. X11 is included if possible.

  • .../OpenBSD/snapshots/
    -- for major architectures, we tend to build mini releases of unknown stability and quality about every month or so.p

  • .../OpenBSD/src/
    -- a checkout of our daily source tree for those who want to access individual files. Normally we recommend the use of CTM or AnonCVS instead of mirroring this location.

  • .../OpenBSD/tools/
    -- various other little bits..

If you add a new FTP mirror site, please contact the FTP maintainer.
