There are currently 3 OpenBSD 2.1 advisories. All these problems are solved in OpenBSD 2.2. Some of these problems still exist in other operating systems.

  • rfork
  • procfs
  • signals

These are the current OpenBSD 2.2 advisories. All these problems are solved in OpenBSD current. Some of these problems still exist in other operating systems.

  • Intel P5 f00f lockup
  • sourceroute
  • ruserok
  • mmap

Other security advisories that have (in the past) affected OpenBSD can be found at the Secure Networks archive.

If you find a new security problem, you can mail it to deraadt@openbsd.org.
