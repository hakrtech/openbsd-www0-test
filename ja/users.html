ユーザ

"ユーザ" という用語には、いくつかの言外の意味が含まれています。エンドユーザは、 しばしば、システムの他のユーザと会い、アイディアや問題とその解決策等を共有し、そして、 飲食を共にしながら、システムについて議論したいと考えています。これをするための最も良い 方法のひとつは、私たちの世界中のユーザグループのいずれかに 参加することでしょう。

また、この用語のもうひとつの言外の意味が、"誰が何のためにシステムを使用しているのか ?" でしょう。そしてそれがこのページの残りの主題なのです。これらの会社や 組織は、OpenBSD の厳密なコード監査やセキュリティ第一の開発モデルを信頼しているのです。 彼らは、ファイアウォールや侵入検知システム、あるいは汎用のサーバを構築するために、 システムを使用しているのです。 大学の研究者や IT 部門の開発者は、しばしば、セキュリティや安定性に 関して、よく似た要求を持っていて、そのような観点から OpenBSD を選びます。また、多くの インターネットサービスプロバイダは、OpenBSD のセキュリティ機能が、 非常に強固であることを知っています。

もしあなたが、このページへの掲載をご希望される場合には、 press@openbsd.org まで、情報をお送りください。

注: セキュリティ上の理由から、会社は、彼らや彼らのクライアントたちの名前を 掲載しないようにとご依頼いただくことも可能です。その場合、彼らは "秘密の会社" として 掲載されることになるでしょう。

    私たちは、現在、OpenBSD が 2 台、Solaris が 1 台と Linux を 2 台、サーバとして運用しています。 私たちは、OpenBSD で、メイン web サイト (7 年前の Sparcstation 5 の上で幸せに動作しています)、 そして、メールサーバ兼ファイアウォールを運用しています。
    私たちの開発エリア内には、さらに 2 台の OpenBSD コンピュータがあります。このうちの 1 台は、ネットワークモニタ (snort を使用) として稼働していて、間もなくファイアウォールの 隣に設置されるでしょう。また、もう 1 台のサーバは、OpenBSD の CVS ミラー (CVSROOT=anoncvs@anoncvs.mx.openbsd.org:/cvs) として稼働しています。
    私たちは、特にクリエイティブなものを実行しているわけではなく、OpenBSD が古い目のハードウェア でもスムーズに走り、私たちが他のオペレーティングシステムに対して抱いている、ほとんどの懸念や 疑いから私たちを解放してくれる、最高のものとして使用しているのです。
    私たちは、スペイン語の OpenBSD のメーリングリスト (openbsd@tlali.iztacala.unam.mx) も主催しています。

インターネットサービスプロバイダ

どのような ISP でも、そのひとつの目標として、顧客のサイトやアカウントを侵入の心配のない 安全な状態に保つというのがあります。OpenBSD のセキュリティの記録は、自身を雄弁に物語って いますが、このため、多くの ISP は、この理由だけでも OpenBSD を使用します。しかしながら、 他のものは、ほとんどではないにせよ、多くのサービスのために OpenBSD を使用しています。

  • Anonix
    Anonix 社は、匿名の電子メール、シェルおよび web ホスティングサービスを提供する ISP です。これらすべてと、DNS および課金システムは OpenBSD 上で運用されています。
    「我々は、セキュリティやきれいなところ、階層化されたアプローチに、その自信を感じます。 基本インストールは巨大な不要パッケージを入れてくれたりせず、我々は、我々のシステム上の すべてがそこに属しているとの確信を持つことができる。」

  • Appalachian Web Solutions
    Appalachian Web Solutions 社は、彼らの企業ファイアウォール およびその舞台裏のセキュリティ機能に OpenBSD を使用する、 カロライナを拠点とするホスティングならびに web デザインの会社です。 「商用、オープンソース両方のすべての選択肢を完全に評価した後では、 我々のファイアウォールならびに他のセキュリティサービス用に OpenBSD を使用するということは、簡単に決定することができた。」

  • BS Web Services
    ドイツの ISP である BS Web Services 社は、プライマリおよびセカンダリの DNS (djbdns)、 主要な web ホスティング (Apache) と主要なメールサービス (qmail-ldap) のために OpenBSD サーバを使用しています。また、ミッションクリティカルな LDAP 認証バックエンド も MySQL データベースとともに、OpenBSD (OpenLDAP) 上で運用しています。 ホストの管理者である、Henning Brauer は次のように述べています。
    「OpenBSDは、これらのマシンではちょっとしたチューニングが必要だったが、それは特に maxprocs と maxfiles とをより大きくするというもので、これによって普通のハードウェアでも、驚くほどの負荷 状況でも処理を続けてくれるようになる。我々は、AMD Athlons (ほとんどは新しい Thunderbird) と AMD K6-III を使用している。我々は、実験用とプリントサーバ、およびあらゆる種類の他の目的のため のサーバとして、OpenBSD で運用されているいくつかの内部のマシンを持っている。我々は、特に我々の ファイアウォールのような、さらにもう少し多くのマシンを OpenBSD に移行させる計画だ。ただ、運悪く、 我々はいまでもクローズドソースのソフトウェアを使用しているのだが、これについては Linux エミュ レーションを試してみようと考えている。特に DoS 攻撃のような高負荷時の OpenBSD の挙動は、 我々が以前使用していた Linux とは比較にならない。だって、Linux は速度低下が著しかったのに対して、 OpenBSD では (同じハードウェアでも !) 気にせずに済んだのだから。」

    この ISP では、長年の間、フリーのシェルサーバを実行するために OpenBSD を使用 していました (このサービスは、「フリー」で提供するコストの上昇のため、 2001 年 11 月に閉鎖されました)。彼らは、他のシステムでも OpenBSD を使用しています。 一基の AMD Athlon 650 を使用していたシェルサーバは、最終的には 101,796 ユーザを 処理していました。Hobbiton の Leif Pedersen は「我々は、他のオペレーティング システムで、日常茶飯事のようなセキュリティ問題を経験していたので、OpenBSD を 試みたのだ」と言いました。「それ以来、オペレーティングシステムのセキュリティが 問題となることもなく、さらにシステムがより安定するというおまけもついた。」

  • Hurontario.net
    カナダ・オンタリオ州のヘッドウォーター地域の Hurontario.net では、 何台かの自社用あるいは顧客用のマシン上で、OpenBSD を使用しています。

  • Infotime
    オランダにある Infotime 社では、OpenBSD のサーバを使用して、 web ホスティングサービス、ドメイン名登録サービスを提供しています。 彼らは、サービスを提供する上で、OpenBSD が最も信頼性が高く、 安全なオペレーティングシステムであるということがわかったそうです。

  • inTEXT Communications
    inTEXT Communications 社は、ファイアウォール、VPN、同じく、さまざまな ハイエンドのセキュリティシステムに OpenBSD を使用する、ネットワーク セキュリティの企業です。inTEXT Communications Inc (1994 創業) は、 カナダ ブリティッシュコロンビア州バンクーバーにあり、pharmaceutical firm 社などの、いくつかの高名な企業に OpenBSD を納入しています。

  • IOActive
    IOActive 社では、WWW の開発者やハッカーのために、実験に使えるテストサーバを提供しています。 ワシントン州のシアトルにあるサービスプロバイダでは、ファイアウォール、 VPN および地域のビジ ネスのための IDS システムにも OpenBSD を使用しています。オーナの Josh Pennell は「OpenBSD は速くて信頼できる。それに、それを使っていることで、夜、私も安眠できる。」と言っています。 「他のことで、これが大好きな理由は、インストール後にマシンを安全な場所に設置するだけで、 もう仕事が半分終ったようなものだということで、このことは私の会社のおびただしい時間を節約して くれている。私の中では、OpenBSD は、オープンソースでセキュアなオペレーティングシステムの デファクトスタンダードだ。他の誰もが、これに追いつこうとしているわけだ。」

  • Phoenix Communications
    Phoenix Communications 社はテキサス州ダラスにある ISP であり、OpenBSD をファイアウォール ならびにその他のインフラストラクチャに使用しています。

  • Poppe Tyson Europe Poppe Tyson Europe 社では、プライマリ DNS、100 個以上のメールボックスのための メールサーバ、および 50 以上のサイトのための web サイト開発サーバに OpenBSD を使用しています。

  • qpalzm.com services
    qpalzm.com services 社では、web ホスティングおよびシェルアカウントを提供するために OpenBSD を使用しています。web サイトは、プログラミング、ゲーム、IRC や他のテクノバブル のために毎日更新しています。オンラインの MUD も利用できます。また、OpenBSD を使用して、 JavaScript や DHTML に興味を持つ人たちの便宜を図った JavaScript メーリングリストも運用しています。 さらに言えば、qpalzm.com の忙しい WWW、FTP、メールサーバは、200MHz の Pentium Pro で OpenBSD がみごとに動いています。
  • RTMX Networking Services
    このノースカロライナにある ISP では、リサーチトライアングルパーク西部の彼らの コミュニティの 1,000 以上の電子メールユーザや web、DNS 用の複数のサーバに OpenBSD を使用しています。AMD K-6、MicroSPARC-II と PowerPC のシステムが混在 して使用されており、さらに、これらに加えて、顧客用サブネットのサーバ群がつな がってきます。また、RTMX.NET は、OpenBSD の WWW および ftp サイトをミラーし、 さらに、匿名 CVS リポジトリ (CVSROOT=anoncvs@openbsd.groupbsd.org:/cvs) も 提供してくれていますが、これらはすべて、彼らの 47GB のディスクスペースと 専用の T1 接続のおかげです。
