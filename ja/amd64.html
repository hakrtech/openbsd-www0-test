OpenBSD/amd64

OpenBSD/amd64 は AMD の Athlon-64 ファミリのプロセッサの 64 ビットモードで動作します。

OpenBSD/amd64 はまた、近い将来にリリース予定の、AMD64 の拡張をサポートする Intel IA32e プロセッサ上でも動作しますが、Intel はページテーブルの NXE (No-EXecute) ビットのサポートを省略しているので、Intel の CPU 上では W^X はサポートされません。

OpenBSD/i386 でもこれらのプロセッサ上で動作可能ですが、 32 ビットモードであることに注意してください。

現在の状況:

現状のほとんどのハードウェア上で、最低の難易度でインストールできて、 動作することが期待されています。

現時点における唯一の大きな欠点は、カーネルデバッガが やや貧弱なことでしょう。

OpenBSD/amd64 の取得とインストール:

折に触れて snapshot が作成されて、 こちらに置かれ、 同時にいくつかの ミラーサイトにも置かれています。 また、同様に OpenBSD/amd64 の snapshot のインストール方法 も置かれています。

以下のように、いくつかのインストールメディアが用意されています。

  • CD からのブート

    ほとんどすべての OpenBSD のドライバを含む El Torito 2.88MB フロッピーイメージで用意された CD からブートします。 これは同様に最小限の USB サポートも含みます (ストレージデバイスとキーボード)。
    このイメージで利用可能なドライバの最新のリストは、カーネル設定ファイル RAMDISK_CD を参照してください。

  • フロッピー A (floppy34.fs)

    この 1.44MB のフロッピーイメージは、ほとんどの共通的なドライバを含みます。 これは、ほとんどの典型的な PC をカバーする目的で設計されています。一般的に、 最高級 SCSI カードやギガビット Ethernet カード、RAID ドライバなどのような 大きな実行用のマイクロコードを必要とするドライバが含まれていません。 このイメージで利用可能なドライバの最新のリストは、カーネル設定ファイル RAMDISK を参照してください。
