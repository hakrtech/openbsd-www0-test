OpenBSD/amd64

OpenBSD/amd64 は AMD の Athlon-64 ファミリのプロセッサの 64 ビットモードで動作します。

OpenBSD/amd64 はまた、AMD64 拡張機能をサポートする Intel IA32e プロセッサ上でも動作しますが、Intel はページテーブルの NXE (No-EXecute) ビットのサポートを省略しているので、Intel の CPU 上では W^X はサポートされません。

OpenBSD/i386 でもこれらのプロセッサ上で動作可能ですが、 32 ビットモードであることに注意してください。

現在の状況:

ほとんどの現存するハードウェアに、インストールしかつ動作させることは かなり容易なこととされています。 しかしながら、ACPI サポートがされていません ( つまり halt -p やサスペンドや休止が動きません )。 またいろいろな標準的な PC ハードウェアがサポートされていません。

現時点における唯一の大きな欠点は、カーネルデバッガである ddb が、やや貧弱なことでしょう。

サポートされているハードウェア:

プロセッサ

すべての AMD Athlon 64 プロセッサとその互換品が サポートされています。

SMP サポート

OpenBSD 3.6 より、OpenBSD/amd64 ではほとんどの SMP (Symmetrical MultiProcessor) システムをサポートしています。 SMP の運用をサポートするため、独立した SMP カーネル (bsd.mp) が インストール用のファイル群に含まれています。

SMP マシン上で実行する場合に、この新しい機能が、どのハードウェアでは実行可能で、 どれでは実行できないのか、開発者は確認したいでしょうから、インストール後に dmesg を送ることがさらに重要になります。

# 3.8 以降に追加されたサポートです。

OpenBSD/amd64 の取得とインストール:

OpenBSD/amd64 がサポートされている最新版は OpenBSD 3.8 です。 OpenBSD/amd64 3.8 のインストール手順 はこちらを参照してください。

折に触れて snapshot が作成されて、 こちらに置かれ、 同時にいくつかの ミラーサイトにも置かれています。 また、同様に OpenBSD/amd64 の snapshot のインストール方法 も置かれています。

以下のように、いくつかのインストールメディアが用意されています。

  • CD からのブート
    ほとんどすべての OpenBSD のドライバを含む El Torito 2.88MB フロッピーイメージで用意された CD からブートします。 これは同様に最小限の USB サポートも含みます (ストレージデバイスとキーボード)。
    このイメージで利用可能なドライバの最新のリストは、カーネル設定ファイル RAMDISK_CD を参照してください。
  • フロッピー A (floppy38.fs)
    この 1.44MB のフロッピーイメージは、ほとんどの共通的なドライバを含みます。 これは、ほとんどの典型的な PC をカバーする目的で設計されています。一般的に、 最高級 SCSI カードやギガビット Ethernet カード、RAID ドライバなどのような 大きな実行用のマイクロコードを必要とするドライバが含まれていません。 このイメージで利用可能なドライバの最新のリストは、カーネル設定ファイル RAMDISK を参照してください。
