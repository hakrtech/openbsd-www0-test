FTP

OpenBSD の FTP ミラーはマスター・ファンアウト・サイトを中心に構成されています。 ここから2次ミラーサイトに送られ、それ以外のサイトはそこからミラーするように なっています。第2ミラーサイトはネットワーク接続がよく、 OpenBSD 開発コミュニティにも知られている管理者が運営している サイトから選ばれました。もしあなたが ftp ミラーを運営したい場合は、 どうかいちばん近い2次ミラーサイトからミラーしてください。 マスター・ファンアウト・サイトからのミラーは非常に迷惑です。

構成・レイアウト:

各サーバのファイルの階層構造はおおまかに次のとおりです。

  • /pub/OpenBSD/
    OpenBSD FTP 集のある HTTP および FTP ミラーの絶対パスは 有効でなければなりません。 もし必要であれば、この場所にリンクがはれます。

  • /pub/OpenBSD/3.7/
    ここに最も最近のリリースが置かれています。
    • alpha/
    • amd64/
    • cats/
    • hp300/
    • hppa/
    • i386/
    • luna68k/
    • mac68k/
    • macppc/
    • mvme68k/
    • mvme88k/
    • packages/ あなたの -stable システムにインストールするための バイナリパッケージ (プリコンパイルされた ports)
    • sgi/
    • sparc/
    • sparc64/
    • tools/ 他のオペレーティングシステム（Windows, Linux およびそれに準ずるもの）上で ブートフロッピーを作ったり、パーティションサイズを変更したりその他のことをするためのツール
    • vax/
    • zaurus/
    • src.tar.gz
       /usr/src に置かれるユーザランドソースコード。 (これにはカーネルソースコードは含まれていません)
    • sys.tar.gz
       /usr/src/sys に置かれるカーネルソースコード。
    • ports.tar.gz
       /usr/ports に置かれる ports のソースコード。
    • XF4.tar.gz
       X Window System (X11) ソースコード。

  • /pub/OpenBSD/snapshots/
    主なアーキテクチャ用にだいたい毎月ビルドされる、 安定性/品質を保証しないミニリリースの試みです。 これを、私たちはテストリリースと位置付けています。

  • /pub/OpenBSD/src/
    個々のファイルにアクセスしたい人のための、CVS でチェックアウトされた 毎日のソースツリーです。通常は、CTM か、 ここに位置するミラーの代わりに AnonCVS の使用をお勧めします。これは、すべての FTP ミラーが、この場所を ミラーしているわけではないからです。

  • /pub/OpenBSD/doc/
    他の OpenBSD 関連のドキュメントと同様の、PDF およびプレーンテキストでの OpenBSD の FAQ です。

  • /pub/OpenBSD/tools/
    その他もろもろ。
  • /pub/OpenBSD/3.6/
    上述した 3.7 リリースと同様のレイアウト。懐古趣味用。
  • /pub/OpenBSD/3.5/
    上述した 3.7 リリースと同様のレイアウト。懐古趣味用。
  • /pub/OpenBSD/3.4/
    上述した 3.7 リリースと同様のレイアウト。懐古趣味用。

新しく FTP/ANONCVS ミラーを立てようと する人でしたら、次の ftp://ftp.openbsd.org/pub/OpenBSD/ で使用される ディスク使用量の見積りに興味があることでしょう。
現状のポリシーでは、このページにリストされたミラーは、すべてのアーキテクチャについて 少なくとも最新の 2 リリース (現時点では 3.6 ならびに 3.7)、および OpenSSH/、OpenNTPD/、OpenBGPD/、patches/ および tools/ ディレクトリの下をバイナリの形で用意することが求められています。
また、利用可能なディスクスペースに依存しますが、スナップショットやより古いリリース、 カレントのソースツリーなど、ミラーはさらに多くのコンテンツを保持することも可能です。
マスタ ftp サイトにおける、現時点でのディスクスペースの使用量は次のようになっています。 


                      現時点    可能性
pub/OpenBSD/3.4       10.8GB             (OpenBSD 3.4 リリース)
pub/OpenBSD/3.5        9.8GB             (OpenBSD 3.5 リリース)
pub/OpenBSD/3.6       17.4GB             (OpenBSD 3.6 リリース)
pub/OpenBSD/3.7       17.3GB    18.5GB   (OpenBSD 3.7 リリース)
pub/OpenBSD/snapshots 15.9GB  さらに巨大! (現在のスナップショットで、
                                          通常は最新リリース以上のサイズとなります)
pub/OpenBSD/src         .5GB      .6GB   (FTP 可能なソースツリーです。ANONCVS から
                                          夜毎チェックアウトされているだけのもの
                                          ですので、たぶん、これを得るための FTP
                                          ミラーの使用は避けるべきで、最寄りの
                                          ANONCVS サーバから単にアップデートする
                                          のが良いでしょう)
このように、完全なミラーは約 75GB ものディスクスペースを必要としますが、最新の 2 リリースに限定すれば、約 50GB ほど必要なだけで済みます。 また、マスター ftp サイトは、ミラーが保持する必要のない、次のようなディレクトリも 用意していることに注意してください。 

pub/OpenBSD/distfiles  4.7GB さらに巨大! (ports サブシステムの makefile が使用する
                                          配布ファイルです。他の理由で必要でもない
                                          限りは、たぶん、これらのファイルをミラー
                                          すべきではないでしょう。ports の
                                          makefile はこれらのためにデフォルトで
                                          ftp.openbsd.org と ftp.usa.openbsd.org
                                          とをチェックします)

もし、新規に FTP、HTTP、AFS や RSYNC ミラーサイトを追加する場合には、 www@openbsd.org に連絡してください。
