OpenBSD 向けの製品

OpenBSD は企業や独立した開発者が、OpenBSD とともに使用する製品や OpenBSD 自身をベースにした製品を創り出すことを応援します。

OpenBSD の商用サポートは、私たちのサポート ページにリストされたベンダから提供されています。

  • OpenBSD をベースにした製品
  • OpenBSD のためのハードウェア/ソフトウェア
  • OpenBSD をプリインストールしたマシン

    • OpenBSD をベースにした製品

  • Stallion Technologies 社の ePipe
    Stallion Technologies は、中小ビジネス向けのダイアルアップ ネットワークゲートウェイ製品である ePipe に、 組み込み OS として OpenBSD を使用しています。このゲートウェイボックスは、 スモールビジネス向けの VPN や アプリケーションサービスプロバイダ による、スモールビジネス向けのホスティングされたアプリケーションにおける、 ラスト 1 マイル の高コストを打ち破るための製品として企画されました。 ePipe は Motorola ColdFire 5307 VL-RISC プロセッサ上で走り、 エンドトゥエンドのセキュアな接続を実現するために、 OpenBSD に組み込まれた暗号化ソフトウェアを使用しています。

  • nCircle 社の Network Security
    nCircle 社の Network Security (以前の Hiverworld) は、インテリジェントな侵入防御を提供するため、 単なる侵入検知を超えた、包括的なネットワークセキュリティを可能に しています。特許出願中の IP360 ソリューションは、回線速度の上限の トラフィックモニタリングと侵入解析とを結合した、継続的で非侵入性の 脆弱性評価を提供します。nCircle 社は、スキャニングセンサおよび 侵入センサ、すべてをセキュアな状態で実行するコンソールエンジン、 ネットワークアプライアンス用の OS として OpenBSD を選定しました。

  • RTMX 社
    RTMX 社は、POSIX のリアルタイム機能が 追加された OpenBSD を完全に補完したバージョンを販売しています。 彼らは恵み深くもこれらの拡張機能のソースコードを寄贈してくれましたので、 まもなく、これらの変更は OpenBSD に統合されるでしょう。

  • syscall Network Solutions 社
    Syscall Network Solutions 社では、OpenBSD ベースの syswall セキュリティアプライアンスを製造しています。 syswall セキュリティアプライアンスは、ステートフルなパケットフィルタリングタイプの ファイアウォールであり、侵入検知システムと VPN 機能が動的 IP アドレスでも使用できます。 この製品は、この会社の IT セキュリティ専門家による、管理セキュリティサービスや 管理セキュリティモニタリング用に特に設計されたものです。また、この製品は、 PPPoE ならびに DHCP をサポートしていますので、DSL や ケーブルルータとしても 利用可能なものです。

    • OpenBSD のためのハードウェア/ソフトウェア

  • GTGI 社の PowerCrypt
    Global Technologies Group, Inc. 社は、Hifn 7751 暗号化プロセッサを組み込んだ、 インテリジェントなバスマスター PCI ボードである PowerCrypt を生産しています。カーネルレベルのサポートは、 OpenBSD コミュニティのメンバーによって書かれたデバイスドライバや IPsec コードとともに、OpenBSD で開発されたものです。

  • RedHat 社の Stronghold Secure Web Server
    RedHat 社は、彼らが受賞した Stronghold 3 Secure Web Server で、OpenBSD 上の、広く称賛された Apache Web サーバを ベースにした、128 ビットの強固な暗号化 HTTPS デーモンを 提供しています。

  • 4Front 社の Open Sound System (i386 のみ)
    4Front Technologies 社は、 OpenBSD/i386 のオーディオデバイスドライバを実質的にきれいに仕上げた 商用バージョンを提供しています。これらの置き換え用ドライバは、OpenBSD/i386 が提供しているものと互換性がありますが、この製品はより多くのカードを サポートし、そのカードが持つ機能を最大限に活用可能です。 このパッケージは、 OpenBSD/i386 3.x 用のものが入手可能です。

  • Core Systems 社
    Core Systems 社は、OpenBSD/i386 用の、 使用法の簡単なサーバサイドの Web 統計ユーティリティである InSite, を販売しています。InSite は WebTrends のような製品と同様のものですが、 わずかな CPU 時間で、すぐにレポートを生成できる Web インターフェース で設定することも可能な製品です (リクエスト次第では、Core 社は i386 以外 のプラットフォーム用の InSite を提供してくれるかも知れません)。

  • Software2Go 社の Motif (i386 および SPARC のみ)
    Software2Go, LLC 社は、OpenBSD 用の Motif 2.1.20 の開発用とランタイム用のツールキットを販売しています。
    開発用ディストリビューションは、Motif のルックアンドフィールを持つ、グラフィカル ユーザインターフェースアプリケーションを開発するソフトウェア開発者用です。また、 ランタイム用ディストリビューションは、バイナリのみのアプリケーションを実行する際に必要となります。
    Software2Go, LLC 社への連絡方法: オンライン注文、 817-431-8775 (TEL/FAX)

  • Zeus 社の Web Server (i386 のみ)
    Zeus Technology 社は、 OpenBSD/i386 2.8 および 2.9 用に、世界で最もスケーラブルで高性能な Web サーバである Zeus Web Server を提供しています。その特徴的な機能の一部を紹介しますと、高度なスケーラビリティ、 リアルタイムの Web ベースの統計機能、サーバごとの帯域幅制御機能、セキュアな CGI 用サンドボックス、クラスタリングサポートや FastCGI サポートなどがあります。
