デフォルトのインストールでも、4 年連続リモートセキュリティホールなし！

OpenBSD プロジェクトは、フリーでマルチプラットホーム対応の 4.4BSD-ベースの UNIX-系 OS を提供します。われわれのプロジェクトは、移植性、標準化、正しさ、積極的なセキュリティと統合化された暗号に力点をおいています。OpenBSD は、SVR4 (Solaris), FreeBSD, Linux, BSD/OS, SunOS、 HP-UX用の多くのプログラムについて、バイナリエミュレーションをサポートします。

OpenBSD はわれわれの FTP サイトから自由に入手できるし、お値ごろな 2枚組CDセットもあります。最新リリースは OpenBSD 2.8 で、2000年12月1日に出荷開始（リリース発表を読みませう）。 CD (そしてTシャツ) のご注文はこちら...

われわれの次のリリースである OpenBSD 2.9は先行予約受付中です。出荷開始は6月1日の予定です。

OpenBSD にはOpenSSH が含まれます。これは SSH1 と SSH2 を両方ともサポートします!

OpenBSD はボランティアによって開発されています。プロジェクトの開発資金は、 CD やTシャツの売り上げと、寄付によってまかなわれています。 組織や個人が 寄付をしてくれるおかげで、OpenBSD が存在を続けて、みんなの好きなように使い、再利用するのがフリーでありつづけることが保証されるわけです。
