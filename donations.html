Donations

OpenBSD has no major sponsors -- the project's infancy was funded by a large cash donation from Theo de Raadt (ie. "savings"). Those funds have since dried up, and currently the project is funded by CD and tshirt sales, as well as generous donations made by various people. For more information...

We have a list of hardware that we would like to see donated to the project. We would appreciate if it people would look through the list and compare it to personal or corporate junk piles. Amongst other reasons, the list exists so that project members can own devices that should be supported, so that future releases ship with better device driver support.
