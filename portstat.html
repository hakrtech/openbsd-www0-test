Port Tree Status

This page lists the date and last change for each OpenBSD port by category. The ports team tries to verify that each port works as expected, but given the number of machines and machine configurations supported there can be no guarantee that any given port will work on your system. If you find a port that does not work please let us know by mailing to ports@OpenBSD.ORG. Be sure to include:

  • the port name and version
  • your OpenBSD version, including any patches you may have applied.
  • the output of uname -a.
  • a complete description of the problem.
