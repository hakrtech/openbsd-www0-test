Produkte für OpenBSD

OpenBSD ermutigt Firmen und unabhängige Entwickler Produkte für die Benutzung unter OpenBSD zu kreieren.

Kommerzieller Support für OpenBSD ist von Anbietern verfügbar, die auf unserer Support Seite aufgelistet sind.

  • Produkte, die auf OpenBSD basieren
  • Hardware/Software Produkte für OpenBSD
  • Vorinstallierte OpenBSD Maschinen

    • Produkte, die auf OpenBSD basieren

  • ePipe von Stallion Technologies
    Stallion Technologies benutzt OpenBSD als 'embedded' Betriebssystem in seinem ePipe dial-up network gateway für kleine und mittlere Geschäfte. Die gateway box soll die hohen Kosten der last Mile für kleinere Business VPNs und Application Service Provider , die Applikationen für kleinere Geschäfte bereitstellen. ePipe läuft auf einem Motorola ColdFire 5307 VL-RISC Prozessor und benutzt OpenBSD's eingebaute Verschlüsselungssoftware um sichere Verbindungen zu liefern.

  • NFR
    NFR benutzen OpenBSD als ihre Hauptentwicklungsplattform für ihre "Network Flight Recorder" Software, die zum Entdecken ovn Eindringlingen in Rechner dient. Das NFR Paket ist als normale software für OpenBSD verfügbar, oder als eingebettetes Produkt namens NFR Intrusion Detection Appliance, die intern unter OpenBSD läuft. Auf ihrer Web Seite teilt NFR den Kunden leider nicht mit, dass die Software in ihrem IDA OpenBSD ist.

  • Anzen Computing
    Anzen Computing verkauft Anzen Flight Jacket, ein umfassenderes System zum Entdecken von Eindringlingen, das auf NFR basiert. Unabhängig davon haben auch sie OpenBSD als ihr hautpsächlich verwendetes System gewählt. Desweiteren berichtet ein technisches Papier von Anzen, dass OpenBSD unter einer Vielzahl von Betriebssystemen dasjenige ist, unter dem NFR am schnellsten läuft.

  • Hiverworld
    Hiverworld bietet eine andauernde Verwundbarkeitseinschätzung in einem Gerät, das OpenBSD benutzt und sie arbeiten an einem 100 MBit und Gigabit IDS (ebenfalls mit OpenBSD), das später in diesem Jahr erscheinen soll.

  • Network Security Wizards
    Network Security Wizards stellt das Dragon IDS für die OpenBSD Plattform her. Dragon ist eine Paket für die Entdeckung von Eindringlingen, das mit 1200 Signaturen umgehen kann. Diese Signaturen können alle Phasen onv Netzwerkmissbrauch inklusive Probes, Angriffe und tatsächliches Eindringen identifizieren. Dragon kann in voll ausgelasteten 100 Mb/s Netzwerken operieren, und viele Dragon Benutzer operieren be etwa 200 Mb/s in Gigabit Ethernet Segmenten. NSW verkauft auch 2U-Maschinen, auf denen OpenBSD und Dragon läuft, für die Leute, die einfach nur Plug&Play machen wollen.

  • RTMX
    RTMX verkauft eine Version von OpenBSD in das eine komplette Sammlung von POSIX Realtime Features integriert wurde. Sie haben großzügigerweise den Source Code für diese Erweiterungen zur Verfügung gestellt und diese Änderungen werden bald in OpenBSD integriert sein.

    • Hardware/Software Produkte für OpenBSD

  • PowerCrypt von GTGI
    Global Technologies Group, Inc. produziert PowerCrypt, eine intelligente bus-masterfähige PCI Karte , die auf dem Hi/fn 7751 Verschlüsselungs Prozessor basiert. Kernel-level Unterstützung wurde von OpenBSD, entwickelt, mit Gerätetreibern und IPSec Code geschrieben von Mitgliedern der OpenBSD Gemeinschaft.

  • Covalent's Raven SSL
    Covalent Technologies Inc. bietet RSA lizensierte 128-bit starke Kryptografie für den weit verbreiteten Apache Web Server unter OpenBSD mit ihrem preisgekrönten Raven SSL Modul. (OpenBSD enthält einen SSL-fähigen httpd, aber die Benutzung innerhalb der USA benötigt eine Lizenz von RSA, die Raven bereitstellt. Es enhält weiterhin Support und einfache Tools, um noch "frische" https-Administratoren zu unterstützen.

  • Lan Media Corporation
    LMC verkauft mehrere PCI-basierte synchrone serielle Karten für T1 und HSSI/DS3, die in OpenBSD nach der Version 2.5 unterstützt werden.

  • Open Sound System (i386 only)
    Eine substantiell verbesserte kommerzielle Version des OpenBSD/i386 Audio Gerätetreibers ist verfügbar von http://www.opensound.com/. Die Ersatztreiber sind kompatibel mit den Audio Treibern mit denen OpenBSD/i386 ausgeliefert wird,aber sie unterstützen viele zusätzliche Karten und all ihre Features. Dieses Paket ist für OpenBSD/i386 2.6 erhältlich.

  • Core Systems
    Core Systems verkauft InSite, ein einfach zu benutzendes Server-seitiges Webstatistik-Werkzeug für OpenBSD/i386. InSite ist Produkten wie WebTrends ähnlich, ist aber auch durch ein Web-Interface zu konfigurieren, um Berichte während des Betriebes zu erstellen, und es verbraucht dabei kaum Rechenzeit. (Auf Nachfrage ist Core in der Lage InSite auch für andere Plattformen ausser i386 bereitzustellen.)

  • Software2Go Motif (nur i386 und SPARC)
    Software2Go, LLC hat Motif 2.1.20 Entwicklungs und Runtime Toolkits für OpenBSD.
    Die Entwicklungsdistribution ist für Software Entwickler, die grafische Oberflächen mit dem Motif look-and-feel entwickeln. Die Runtime Distribution ist notwendig, um nur-lauffähige Applikationen zu entwickeln, also ohne Quellcode.
