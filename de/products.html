Produkte für OpenBSD

OpenBSD ermutigt Firmen und unabhängige Entwickler Produkte für die Benutzung unter OpenBSD zu kreieren oder auch solche ,die auf OpenBSD basieren. Sieh dir die Produkte weiter unten einfach an.

Kommerzieller Support für OpenBSD ist von Anbietern verfügbar, die auf der Support Seite augeführt sind. Eine Gallerie der User führt Firmen und Projekte auf, die OpenBSD in verschiedenen Varianten eisetzen.

  • PowerCrypt von GTGI
    Global Technologies Group, Inc. produziert PowerCrypt, eine intelligente, bus-master-fähige PCI Karte basierend auf dem Hi/fn 7751 Verschlüsselungsprozessor. Kernel-level Unterstützung wurde entwickelt von OpenBSD, mit Gerätetreibern und IPSec Code geschrieben von Mitgliedern der OpenBSD Gemeinschaft.

  • ePipe von Stallion Technologies
    Stallion Technologies benutzt OpenBSD als 'embedded' Betriebssystem in seinem ePipe dial-up network gateway für kleine und mittlere Geschäfte. Die gateway box soll die hohen Kosten der last Mile für kleinere Business VPNs und Application Service Provider , die Applikationen für kleinere Geschäfte bereitstellen. ePipe läuft auf einem Motorola ColdFire 5307 VL-RISC Prozessor und benutzt OpenBSD's eingebaute Verschlüsselungssoftware um sichere Verbindungen zu liefern.

  • NFR
    NFR benutzen OpenBSD als ihre Hauptentwicklungsplattform für ihre "Network Flight Recorder" Software, die zum Entdecken ovn Eindringlingen in Rechner dient. Das NFR Paket ist als normale software für OpenBSD verfügbar, oder als eingebettetes Produkt namens NFR Intrusion Detection Appliance, die intern unter OpenBSD läuft. (Wir glauben, dass es leider noch OpenBSD 2.3 ist). Auf ihrer Web Seite teilt NFR den Kunden leider nicht mit, dass die Software in ihrem IDA OpenBSD ist.

  • Anzen Computing
    Anzen Computing verkauft Anzen Flight Jacket, ein umfassenderes System zum Entdecken von Eindringlingen, das auf NFR basiert. Unabhängig davon haben auch sie OpenBSD als ihr hautpsächlich verwendetes System gewählt. Desweiteren berichtet ein technisches Papier von Anzen, dass OpenBSD unter einer Vielzahl von Betriebssystemen dajenige ist, unter dem BFR am schnellsten läuft.

  • Covalent's Raven SSL
    Covalent Technologies Inc. bietet RSA lizensierte 128-bit starke Kryptografie für den weit verbreiteten Apache Web Server unter OpenBSD mit ihrem preisgekrönten Raven SSL Modul. (OpenBSD enthält einen SSL-fähigen httpd, aber die Benutzung innerhalb der USA benötigt eine Lizenz von RSA, die Raven bereitstellt. Es enhält weiterhin Support und einfache Tools, um noch "frische" https-Administratoren zu unterstützen.

  • Network Security Wizards
    Network Security Wizards stellt das Dragon IDS für die OpenBSD Plattform her. Dragon ist eine Paket für die Entdeckung von Eindringlingen, das mit 1200 Signaturen umgehen kann. Diese Signaturen können alle Phasen onv Netzwerkmissbrauch inklusive Probes, Angriffe und tatsächliches Eindringen identifizieren. Dragon kann in voll ausgelasteten 100 Mb/s Netzwerken operieren, und viele Dragon Benutzer operieren be etwa 200 Mb/s in Gigabit Ethernet Segmenten.

  • Hiverworld
    Hiverworld verkauft eine OpenBSD-basierte Hardwarelösung, die eine konstante Intranet-Verwundbarkeit Schätzung abgibt, und das Produkt heisst HiverMute Network Security Appliance.

  • Lan Media Corporation
    LMC verkauft mehrere PCI-basierte synchrone serielle Karten für T1 und HSSI/DS3, die in OpenBSD nach der Version 2.5 unterstützt werden.

  • RTMX
    RTMX verkauft eine Version von OpenBSD die zusätzlich eine komplette Menge an POSIX Echtzeitfähigkeiten hat. Sie haben freundlicherweise den Quellcode für diese Erweiterungen bereitgestellt, und diese Änderungen werden bald in OpenBSD einfliessen.

  • Software2Go Motif (i386 and SPARC only)
    Software2Go, LLC hat Motif 2.1.20 Entwicklungs und Runtime Toolkits für OpenBSD.
    Die Entwicklungsdistribution ist für Software Entwickler, die grafische Oberflächen mit dem Motif look-and-feel entwickeln. Die Runtime Distribution ist notwendig, um nur-lauffähige Applikationen zu entwickeln, also ohne Quellcode.

  • Core Systems
    Core Systems verkauft InSite, ein einfach zu benutzendes, server-seitiges Webstatistik Utility für OpenBSD/i386. InSite ist Produkten wie WebTrends sehr ähnlich, aber kann durch ein Web-basiertes Interface so konfiguriert werden, im laufenden Betrieb Statistiken zu erzeugen und dabei die CPU nur sehr gering zu beanspruchen. Auf Nachfrage kann Core InSite möglicherweise auch für andere Plattformen bereitstellen.

  • Open Sound System (nur i386)
    Eine deutlich erweiterte Version des OpenBSD/i386 Audio Treibers ist bei http://www.opensound.com/ erhältlich. Diese Treiber ersetzen die Standard-Treiber, sind aber kompatibel dazu, und unterstützen viele zusätzliche Karten und unterstützt all deren Features. Dieses Paket ist für OpenBSD/i386 2.6 erhältlich.

  • Vorinstallierte OpenBSD Systeme in Oregon, USA
    Chris Cappuccio verkauft Systeme, auf denen OpenBSD bereits installiert ist und die als Server optimiert sind, oder auch als Router oder Workstation. Dabei sind auch spezielle Netzwerkoptionen und ein installiertes X-Windows.

  • Pre-installed OpenBSD Systeme in Großbritannien
    Knowledge Matters Ltd. verkauft Systeme, die mit OpenBSD laufen, für eine ganze Reihe von Anwendungen. Das Hauptaugenmerk liegt dabei auf Internet-Services, Routern und Utility-Plattformen. Knowledge Matters ist in London, England zu Hause.
