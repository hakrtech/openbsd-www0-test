Produkte für OpenBSD

OpenBSD ermutigt Firmen und unabhängige Entwickler Produkte zu entwickeln, die unter OpenBSD nutzbar sind, oder auf OpenBSD basieren.

Kommerzieller Support für OpenBSD ist von Anbietern verfügbar, die auf unserer Support Seite aufgelistet sind.

  • Produkte, die auf OpenBSD basieren
  • Hardware/Software Produkte für OpenBSD
  • Vorinstallierte OpenBSD Maschinen

    • Produkte, die auf OpenBSD basieren

  • ePipe von Stallion Technologies
    Stallion Technologies benutzt OpenBSD als 'embedded' Betriebssystem in seinem ePipe dial-up network gateway für kleine und mittlere Geschäfte. Die gateway box soll die hohen Kosten der letzten Meile für kleinere Business VPNs und Application Service Provider, die Applikationen für kleinere Geschäfte bereitstellen drücken. ePipe läuft auf einem Motorola ColdFire 5307 VL-RISC Prozessor und benutzt OpenBSD's eingebaute Verschlüsselungssoftware um sichere Verbindungen zu liefern.

  • nCircle Network Security
    nCircle Network Security (vormals Hiverworld) bietet umfassende Netzwerksicherheit und geht dabei über Intrusion Detection hinaus, hin zu Intelligenter Intrusion Prevention. Ihr Patentantrag ist noch in der Schwebe, doch ihre IP360 Lösung liefert dauerhafte nicht-invasive Verwundbarkeitsuntersuchungen kombiniert mit Traffic Monitoring und Intrusion Analysis mit der vollen Geschwindigkeit des Netzwerks. nCircle hat sich für seinen Scanning Sensor, Intrusion Sensor und seine Console Engine für OpenBSD als Betriebssystem entschieden, die alle auf einem sicheren Netzwerkgerät laufen.

  • RTMX
    RTMX verkauft eine Version von OpenBSD, die eine vollstädigen Zusatz an POSIX real-time Features hat. Sie haben diese uns diese Änderungen großzügig zur Verfügung gestellt, und die Änderungen werden bald in OpenBSD integriert werden.

  • syscall Network Solutions
    Syscall Network Solutions produziert die syswall Sicherheitsanwendung basierend auf OpenBSD. Die syswall Sicherheitsanwendung beinahltet eine stateful packet-filtering Firewall, ein Intrusion Detection System und VPN Fähigkeiten sogar mit dynamischen IP Addressen. sie wurde speziell designed, um Managed Security Services und Managed Security Monitoring von unseren IT Security Experten zu bieten. Die syswall kann also als DSL- oder Kabel-Router benutzt werden, da sie PPPoE und DHCP unterstützt.

    • Hardware/Software Produkte für OpenBSD

  • PowerCrypt von GTGI
    Global Technologies Group, Inc. produziert PowerCrypt, eine intelligente bus-masterfähige PCI Karte, die auf dem Hifn 7751 Verschlüsselungs Prozessor basiert. Kernel-Level Unterstützung wurde von OpenBSD entwickelt, Gerätetreiber und IPsec Code wurde von von Mitgliedern der OpenBSD Gemeinschaft geschrieben.

  • RedHat's Stronghold Secure Web Server
    RedHat bietet einen 128-Bit starken Kryptographie HTTPS Daemon basierend auf dem weit verbreiteten Apache Web Server auf OpenBSD mit ihrem preisgekrönten Stronghold 3 Secure Web Server.

  • Open Sound System von 4Front (nur i386)
    4Front Technologies bietet eine wesentlich verbesserte kommerzielle Version der OpenBSD/i386 Audio Treiber. Diese Ersatztreiber sind kompatibel mit denen, die mit OpenBSD/i386 ausgeliefert werden, aber sie unterstützen zusältzliche Karten und auch deren komplette Features. Dieses Paket ist für OpenBSD/i386 3.x erhältlich.

  • Core Systems
    Core Systems verkauft InSite, ein einfach zu benutzendes Server-seitiges Webstatistik-Werkzeug für OpenBSD/i386. InSite ist Produkten wie WebTrends ähnlich, ist aber auch durch ein Web-Interface zu konfigurieren, um Berichte während des Betriebes zu erstellen, und es verbraucht dabei kaum Rechenzeit. (Auf Nachfrage ist Core in der Lage, InSite auch für andere Plattformen als i386 bereitzustellen.)

  • Software2Go Motif (nur i386 und SPARC)
    Software2Go, LLC hat Motif 2.1.20 Entwicklungs und Runtime Toolkits für OpenBSD.
    Die Entwicklungsdistribution ist für Software Entwickler, die grafische Oberflächen mit dem Motif look-and-feel entwickeln. Die Runtime Distribution ist notwendig, um nur-lauffähige Applikationen zu entwickeln, also ohne Quellcode.
    Kontaktiere: Software2Go, LLC Online Bestellung, 817-431-8775 (phone/fax)

  • Zeus Web Server (nur i386)
    Zeus Technology stellt den Zeus Web Server her, die weltweit skalierbarste "high performance web server" Software für OpenBSD/i386 2.8 und 2.9. Einige seiner ausgewöhnlichen Features: sehr skalierbar, webbasierte Statistiken in Echtzeit, pro-Server Bandbreitenbeschränkungen, sicheres CGI Sandboxing und Clusterunterstützung.
