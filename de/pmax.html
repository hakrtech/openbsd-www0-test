OpenBSD/pmax

OpenBSD/pmax läuft auf einer grossen Menge der Digital MIPS-basierten DECstation und DECserver Familie. Diese Maschinen waren der Gegenpart der VAX-basierten VAXstations, bis die Alpha Linie fertig war.

Der OpenBSD/pmax port wurde nach der Version 2.7 nicht weiterentwickelt.

Zur Zeit arbeitet niemand aktiv daran den port erneut zum Leben zu erwecken. Der Source Code ist weiterhin per CVS Attic verfügbar, ausserdem ist welcher in NetBSD zu finden. Wenn du interessiert daran bist, den pmax port wiederzubeleben, bringe ihn zum Laufen und wende dich danach an Todd Miller und Miod Vallat.

Geschichte:

Der pmax port von OpenBSD wurde von Thomas Graichen kurz vor der Veröffentlichung von 2.0 begonnen, basierend auf den NetBSD/pmax Sourcen. Er wurde dann langsam in Form gebracht, es konnte auf dem port selbst weiterentwickelt werden, und dann auf ECOFF umgestellt werden, den ELF binaries.

Ein paar Versionen später beendete Thomas Graichen seine Arbeit an diesem port, und niemand folgte ihm. Der port geriet langsam in Vergessenheit, ohne neuen Code und kaum Bugfixes, obwohl er noch mit veröffentlicht wurde.

Nach der Version 2.7 wurde eine grosse Anstrengung gemacht, den port wiederzubeleben, aber MIPS-spezifische toolchain-Probleme verhinderten, dass sich das System selbst erzeugen konnte, und die Entwicklung begann erneut einzuschlafen. Als das Problem schließlich bereinigt wurde, war die Entscheidung schon gefallen, die Arbeit an diesem port einzustellen und den Code aus dem source tree zu entfernen.

Unterstützte Hardware:

Unterstützte Modelle

  • DECstation 2100 (PMIN) und 3100 (PMAX)
  • DECstation 5000/120, 5000/125 und 5000/133 (3MIN)
  • DECstation 5000/200 (3MAX)
  • DECstation 5000/240 (3MAXPLUS)
  • Personal DECstation 5000/20, 5000/25, 5000/33 und 5000/50 (MAXINE)

Unterstützte Peripheriegeräte

Frame Buffer
  • On-board PM Monochrom (1024x864x1) oder 8-bit Farbe(1024x864x8) auf PMIN und PMAX
  • On-board PMAG-DV (1024x864x8) auf Personal DECstation
  • TURBOchannel PMAG-AA (mfb) (1280x1024x1)
  • TURBOchannel PMAG-BA (cfb) (1024x864x8)
  • TURBOchannel PMAGB-BA (sfb) (1280x1024x8)
XFree86 funktioniert auf allen unterstützten Frame Buffern.
  • Ethernet
    • On-board Lance Ethernet
    • TURBOchannel PMAD-AA oder PMAD-AB Lance Ethernet
  • SCSI Controller
    • On-board SCSI Controller (sii) auf PMIN und PMAX
    • On-board SCSI Controller (asc) auf anderen Modellen
    • TURBOchannel PMAZ-AB SCSI Controller (asc)
  • Verschiedene Geräte
    • DEC LK-201 oder kompatible Tastatur
    • DEC VSXXX oder kompatible Maus
    • On-board serielle Ports auf den meisten Modellen

    Wo man OpenBSD/pmax herbekommt und wie man es installiert:

    Die letzte unterstützte Version von OpenBSD/pmax war OpenBSD 2.7. Es ist nicht länger von den FTP-Sites erhältlich, aber Versionen bis zu 2.3 waren auf CD verfügbar.
