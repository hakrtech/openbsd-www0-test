Installationsmedien

Es gibt verschiedene Installationsmöglichkeiten:
  • CD boot:

    Beim booten von CD wird ein El Torido 2.88MB Floppy Image geladen, dass nahezu alle OpenBSD Treiber enthält. Das beinhaltet auch minimale USB-Unterstützung (umass und ukbd Geräte).
    Die neueste Liste von Treibern für dieses Image findet sich im RAMDISK_CD Config File.

  • Floppy A:

    Dieses 1.44MB Floppy Image enhält die gebräuchlichsten Treiber. Es ist dazu bestimmt, auf dem typischen PC zu laufen. Als generelle Regel gilt dabei, dass solche Treiber, die viel Microcode laufen lassen müssen, nicht dabei sind, genauso wie seltene SCSI Treiber, Gigabit Ethernet Karten, oder RAID Geräte.
    Die neueste Liste von Treibern für dieses Image findet sich im RAMDISK Config File.

  • Floppy B:

    Dieses Medium soll den Leuten helfen morderne Serversystem aufsetzen. Es enthält Unterstützugn für die meisten SCSI und RAID Treiber, Gigabit Ethernet Karten, und so weiter. Aufgrund von Platzmangel fehlen einige Treiber. Insbesondere die Unterstützung für die meisten PCMCIA Geräte fehlt, genauso wie für einige wirklich alte Hardware.
    Die neueste Liste von Treibern für dieses Image findet sich im RAMDISKB Config File.

  • Floppy C:

    Dieses 1.44MB Floppy Image enthält viele Cardbus und PCMCIA Gerätetreiber. Dieses Medium ist zum Installieren von Laptops gedacht. In den meisten F6auml;llen wird auch Floppy A funktionieren, aber einige ungewöhnliche Laptops können von dieser Floppy profitieren.
    Die neueste Liste von Treibern für dieses Image findet sich im RAMDISK_CD Config File.

Nimm einfach die Methode, die für dich am besten ist.

Installation mit anderen Betriebssystemen:

Der OpenBSD/i386 port kann so installiert werden ,dass er sich die Festplatte/n mit anderen Betriebssystemen teilt, z.B. mit MS-DOS, Windows oder Linux, indem er das MS-DOS MBR/Partitions-Schema benutzt und optional einen Boot-loader zum Auswählen, welches BS nun gebootet werden soll. In den Installtionshinweisen gibt es weitere Details.

Binärkompatibilität mit anderen Betriebssystemen:

Der OpenBSD/i386 port kann Linux, FreeBSD, BSD/OS, SVR4 (inklusive Solaris), und IBCS2 binaries ausführen.

Unterstützte Hardware:

Der OpenBSD/i386 port funktioniert auf einer großen Menge von Standard-PC-Hardware und Klonen davon, inklusive einer grossen Menge an Prozessoren und IO-Bussystemen. In den allermeisten Fällen laufen Installation und Betrieb ohne große Schwierigkeiten. Die Fälle, in denen Probleme auftauchen, sind typischerweise ältere proprietäre PC's, Laptops oder spezialisierte Server Maschinen, die auf einem bestimmten BIOS beruhen, um bestimmte Implementationunterschiede zu überdecken.

OpenBSD unterstützt zur Zeit kein SMP, sprich hat keine Multiprozessorfähigkeit, kann aber auf solchen Maschinen installiert werden, obwohl dann eben nur ein Prozessor genutzt wird.

Die meisten Laptops werden unterstützt. Wir haben eine Seite die die Details für die gängigen Laptops auflistet.

Die Liste der unterstützten Hardware bezieht sich auf OpenBSD-current. Sie weicht leicht von der ab, die für die letzte veröffentlichten Version gilt.

Nicht unterstütze Hardware

Nicht unterstütze Hardware kann in der Zukunft unterstützt werden, eine Spende von Hardware, oder das Einbringen von Zeit zur Unterstützung oder Dokumentation kann das ganze natürlich beschleunigen!

  • Der "Micro Channel" MCA bus , der in vielen IBM PS/2 Modellen benutzt wird.
  • NCR 5380-basierte und 53400-basierte SCSI host Adapter
  • QIC-40 und QIC-80 tape drives (das sind die Tape drives, die man am floppy disk Controller anschliessen kann)
  • Multiprocessor Motherboards (obwohl sie gut funktionieren, aber dann nur einen Prozessor benutzen). Es existiert bereits ein Projekt , um das zu ändern.
  • Sony und Panasonic proprietäre CD-ROM interfaces
  • Parallel-port ZIP Laufwerke (SCSI und ATAPI ZIP ALufwerke arbeiten bestens)
  • Intel i82556 (EtherExpress PRO/100A) und i82596 (EtherExpress PRO/10 PCI) Ethernet Adapter
  • Hewlett-Packard PC-LAN+ (HP27xxx) Ethernet Adapter
  • SoundBlaster Live! (EMU10k1) Sound Geräte
  • Yamaha DS1-based Sound Geräte
  • Aureal Vortex Soundkarten
  • Winmodems
  • Infrared devices, wie sie meist in Laptops eingebaut sind
  • Firewire
  • 3C990 Hardware Krypto Beschleuniger (3Com will keinerlei Spezifikationen öffentlich machen)
  • Mylex, und Intel RAID Controller
  • PCMCIA:
    • Memory cards
    • Die meisten Multifunktions Karten

Wenn du gerne einige neue Features vor der nächsten offiziellen Version testen willst, probiers mal mit einem der Snapshots. Für die i386-Architektur gibt es von Zeit zu Zeit neue Snapshots.
