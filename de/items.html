OpenBSD 3.6: Wird ab dem 1. November 2004 versandt


Um eine 3.6 CDROM zu bestellen, klicke hier.
  • Drei CDROMs im HardCover.
  • Die kompletten Installationskomponenten für SECHS Architekturen: i386, vax, amd64, macppc, sparc, sparc64.
  • Die folgenden Architekturen sind nur via FTP Download erhältlich: alpha, cats, hp300, hppa, luna88k, mac68k, mvme68k, mvme88k.
  • Die CDs sind bootbar auf i386, amd64, macppc, sparc und sparc64.
  • Ein abgefahrenes und überraschend künstlerisches CD Inlet, das die Installationsanweisungen enthält. Die Information auf diesem Stück Papier machen OpenBSD viel leichter installierbar, als wenn du eine FTP Installation machen würdest.
  • Ein kompletter Quellcode-Baum (bereit für die Benutzung von AnonCVS).
  • Die neuesten zuverlässigen XFree86 Binaries für alle Architekturen
  • Der neueste XFree86 Quellcode mit kleinen Änderungen von uns, um es schöner und sicherer zu machen.
  • Unser eigener ports tree, der sich seit OpenBSD 3.4 unglaublich verbessert hat. Fast alle Pakete arbeiten auf fast allen Architekturen.
  • Verschiedene fertige Packete für die am weitesten verbreiteten Architekturen, die ganz einfach von der CDROM installiert werden können.
  • Wie immer sind Aufkleber dabei!

OpenBSD 2.3: 19. Mai, 1998


Um eine 2.3 CDROM zu bestellen,hier clicken.
  • Zwei CDROMs im Hardcover.
  • Die kompletten Installationskomponenten für ZEHN Architekturen: alpha, amiga, arc, hp300, i386, mac68k, mvme68k, pmax, powerpc, sparc.
  • Die CDs sind auf i386, sparc, powerpc und alpha bootbar.
  • Inklusive einer CD-Beilage, auf der die Installationsanleitungen sind. Durch diese Installationshinweise ist die Installation von CD sogar einfacher, als die über FTP.
  • Ein kompletter Quellcode-Baum (fertig für AnonCVS).
  • Die letzten XFree86 binaries für alle Architekturen (bis auf PowerPC), so dass jetzt XFree86 viel leichter zu installieren ist, als bei OpenBSD 2.2 .
  • Der letzte XFree86 source code mit kleinen Modifikationen von uns,um ihn hübscher zu machen :-)
  • Unser eigener kleiner ports tree, der seit OpenBSD 2.2 stark verbessert wurde.
  • Ein grosses Archiv von distfiles, die vom oben erwähnten ports tree benutzt werden, damit die binaries einfacher erzeugt werden können.
  • Die folgenden fertigen Pakete für die meisten Architekturen, die einfach von CDROM installiert werden können:
    bash-2.01.1, emacs-20.2, gmake-3.76.1, id-utils-3.2, ircii-2.8.2-epic3.004, pine-3.96, screen-3.7.4, tcsh-6.07.02, unzip-5.3.1.
  • Zwei NEUE OpenBSD 2.3 Aufkleber inklusive!
