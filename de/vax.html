OpenBSD/vax

Geschichte und Status:

OpenBSD/vax wird momentan in Bezug auf die Harwareunterstützung ausgebaut, genauso die Befehlsemulation, die Grafik-Konsolen-Unterstützung und 'shared libraries'. Der architekturspezifische code basiert grösstenteils auf dem NetBSD/vax port, so dass alles in allem eine recht komfortable OpenBSD Umgebung dabei rauskommt.

OpenBSD läft zur Zeit auf diesen Vax Modellen:

  • vaxstation/microvax 2000
  • vaxstation 3100/20
  • vaxstation 3100/76
  • vaxstation 4000/60
  • microvax 3300
  • vax 4000/300

SCSI sollte auf den folgenden Maschinen arbeiten:

  • microvax 3300
  • vaxstation 4000/60
  • vaxstation 4000/vlc?
  • die meisten der 3100 Serie von vaxstations und microvaxes

Projekte, an denen gearbeitet wird:

  • vaxstation 2000's hdc9224 (floppy/mfm Festplatten Controller)
  • qbus tk50/tk70 Bandlaufwerke
  • flexiblere Installationsoptionen
  • shac dssi Unterstützung
  • VAX 4000-10X Unterstützung (Dokumentation und Hardware gesucht)

Bekannte Probleme:

  • optimizer bugs im gcc
  • Es gibt Aussetzer im lance ethernet
  • ein paar 'signal delivery problems'

Die momentan aktuelle Version ist OpenBSD/vax 2.8; es gibt aber von Zeit zu Zeit snapshots.

Der OpenBSD/vax port maintainer ist Brandon Creighton.
