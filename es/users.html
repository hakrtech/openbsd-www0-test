Usuarios

El término «usuarios» tiene varias connotaciones. Los Usuarios finales suelen desear conocer a otros usuarios del sistema, con el propósito de intercambiar ideas, problemas y soluciones, y para discutir sobre el sistema mientras comen o beben algo. Uno de las mejores maneras de hacer esto es mediante uno de nuestros Grupos de Usuarios Internacionales.

Otra connotación del término es la de «aquellas entidades que usan el sistema y para qué lo usan», y ése es el asunto que trataremos en el resto de esta página. Estas compañías y organizaciones confían en el modelo de desarrollo basado en la seguridad como hecho primordial y en las rigurosas auditorías de código de OpenBSD. Utilizan el sistema para construir sistemas de detección de intrusos, cortafuegos, o servidores con propósitos varios. Los investigadores de las universidades o los desarrolladores de los departamentos de tecnología informática a menudo necesitan de estas características de seguridad, y escogen OpenBSD.

Si le gustaría estar en la lista de esta página, envíe la información a press@openbsd.org.

NOTA: Por motivos de seguridad, las empresas pueden pedirnos que ocultemos sus nombres o los de sus clientes. En tal caso aparecerían como "Undisclosed Company" (Compañia No Revelada).

  • Amnistía Internacional
    Amnistía Internacional es una organización a nivel mundial que trabaja para promover los derechos humanos tal y como aparecen en la Declaración Universal de Derechos Humanos y en otras normativas internacionales. Amnistía Internacional usa OpenBSD para los cortafuegos de sus redes y para sus Redes Privadas Virtuales entre sus diferentes secciones alrededor del mundo.

  • Network Security Technologies, Inc.
    Esta firma de seguridad informática y de redes usa OpenBSD para la detección de intrusiones de alta velocidad, redes virtuales privadas (VPNs), y aplicaciones de almacenamiento de datos. Network Security Technologies Inc. está ubicada dentro del área metropolitana de Washington DC, y usa OpenBSD para varias agencias gubernamentales y militares no reveladas.

  • Adobe Systems
    Este gigante del software usa OpenBSD en varios de sus cortafuegos para red y sus sistemas de pruebas de redes.

  • Consejo Electoral del Estado de Sonora, México
    El Consejo Estatal Electoral del Estado de Sonora usa OpenBSD para proteger sus sistemas; éstas aplicaciones están disponibles a través de Internet, dando resultados electorales tanto a usuarios internos como externos.

  • Calyx Internet Access Corp.
    Esta compañía usa OpenBSD para la ejecución de todos los servicios con misiones críticas, incluidos WWW, FTP, correo electrónico, tráfico de VPN, y monitorización de redes en sus centros de datos en Nueva York, Los Ángeles y Amsterdam. Sitios aún más grandes de web como mitsubishicars.com, snapple.com, tanqueray.com y otros no representan ningún reto para OpenBSD.

  • Alteon Networks
    El manufacturador de etherner gigabit usa máquinas OpenBSD con fines variados, como campos de prueba y pasarelas.

  • CORE SDI S.A.
    Una compañía especializada en Seguridad de la Información con sede en Buenos Aires, Argentina, que usa OpenBSD como plataforma principal para las operaciones y desarrollo de productos relacionados con la seguridad de información. «La fortaleza, portabilidad y el compromiso para con la seguridad de OpenBSD, así como la capacidad de funcionamiento en diferentes plataformas de hardware, da como resultado un sistema operativo ideal para entornos en los que la seguridad y alta disponibilidad son importantes», dice Iván Arce, ingeniero-jefe de CORE SDI'S.

  • IOActive
    IOActive pone a disponibilidad de desarrolladores de WWW y "hackers", un sitio para experimentar con servidores de prueba. Este proveedor de servicios de Seattle también instala sistemas VPN e IDS, y cortafuegos con OpenBSD para comercios y empresas regionales. «OpenBSD es rápido, fiable, y yo duermo un algo mejor por la noche sabiendo que lo estoy usando,» dice el propietario, Josh Pennell. «Otra cosa que me gusta mucho de OpenBSD es que más de la mitad del trabajo para asegurar el sistema ya está hecho nada más instalarlo, ahorrando a mi compañía una cantidad de tiempo enorme. Para mí, OpenBSD es la norma "de facto" para sistemas operativos seguros de código abierto. Los demás sólo intentan ponerse a su altura».

  • The University of Minnesota
    Esta universidad usa OpenBSD en estaciones de trabajo Sun Sparc para tareas de monitorización de redes y planificación de capacidad. Hacen pesquisas en 53.000 interfaces diferentes (mayo de 1.999) por medio de SNMP, registrando ingresos de datos SNMP de más de 250MB a un disco concatenado para procesarlos cada mes.

  • Empire Net
    Un proveedor de servicios de Internet que usa OpenBSD en sistemas basados en AMD, Intel y Sun, para enrutamiento, cortafuegos, IPsec/VPN, limitación de bando de ancha, hospedaje de web, servidores de bases de datos, monitorización de redes, detección de intrusos, servidores de correo, servidores para copias de seguridad, servidores de cache, y estaciones de trabajo. Uno de sus enrutadores gestiona el tráfico entre un T3 y ocho puertos fast ethernet, y también entre varias VLANs 802.1Q para separar las redes entre los clientes y las compañías con espacio alquilado.

  • RTMX Incorporated
    Esta comercial produce un derivado de OpenBSD, solución de sistema extendido a tiempo real IEEE POSIX. RTMX ha hecho una reimplementación total de extensiones a tiempo real IEEE POSIX, previamente desarrolladas, usando OpenBSD como fuente. Aproximadamente todas las extensiones IEEE POSIX han sido añadidas. Actualmente todas las características de OpenBSD se pueden usar, pero sólo en procesadores de tipo m68k, MIPS, y PowerPC. Las versiones para Pentium y Alpha aparecerán pronto. RTMX Inc. también es una casa con licencia de desarrollo para OSF Motif, y ha portado Motif 1.2.3 a la mayoría de plataformas soportadas por éste.

  • RTMX Networking Services
    Este proveedor de servicios de Internet de Carolina del Norte usa OpenBSD en múltiples servidores de Web, DNS y más de 1.000 usuarios de correo electrónico en su comunidad. Usa una mezcla de sistemas con AMD K-6, MicroSPARC-II y PowerPC, con más servidores de subrred clientes. RTMX.NET tiene una réplica de las páginas de OpenBSD, WWW, y de las de FTP; también ofrece acceso a un repositorio de CVS anónimo (CVSROOT=anoncvs@openbsd.groupbsd.org:/cvs), todo gracias a los 47GB de espacio en disco y una conexión dedicada T1.

  • Poppe Tyson Europe
    Usa OpenBSD como DNS principal, servidor de correo para más de 100 buzones, y como servidor para el desarrollo de su sitio Web para más de 50 sitios.

  • The Center for Information Technology Integration (CITI)
    Este laboratorio en la Universidad de Michigan usa OpenBSD como la base para muchos proyectos de investigación intensiva. OpenBSD se usa para desarrollar y analizar los protocolos y contenidos de tarjetas inteligentes, tanto en aislamiento como en aplicaciones reales. Existen planes para la expedición de tarjetas que contengan fichas seguras para la introducción del usuario en el sistema y adquisición de pase con kerberos. OpenBSD también se usa como plataforma de pruebas para el programa de ordenadores móviles en CITI. Internamente, «El Cofre del Paquete» es un máquina OpenBSD que captura y graba en cd-rom cada paquete en la ethernet de 10Mbps local. El contenido del paquete se cifran conforme con los requerimientos de privacidad. Esta práctica se utiliza para la detección de intrusos. Aparte de esto, unas cuantas personas del departamento usan OpenBSD como su sistema operativo principal.

  • The University of Alberta
    Usa OpenBSD en hardware SPARC e Intel para servidores proxy, servidores Kerberos, servidores de impresión, monitorización de servicios, escaneados de seguridad preventivos, y respuestas a incidentes. Se usa OpenBSD en procesadores Intel para cortafuegos y Lan-to-Lan VPN (Redes Privadas Virtuales de Red Local a Red Local) para las subrredes con implementaciones de seguridad de la universidad, detrás de las cuales residen todos los nuevos sistemas administrativos de la universidad. OpenBSD se usa para pasarelas de autenticación frente a laboratorios públicos y conexiones públicas ethernet en unos 40 sitios en todo el campus (150 plazas aproximadamente) para ayudar a hacer el acceso público a internet seguro. El Departamento de Ciencias Informáticas usa laboratorios OpenBSD de 20 plazas para las prácticas de los estudiantes.

  • Crown.Net
    Este proveedor de servicios de Internet funciona casi totalmente con una mezcla de OpenBSD/sparc y OpenBSD/i386. Sus dos servidores de Web, servidor de correo, DNS primarios y secundarios, y servidores Radius funcionan todos con OpenBSD/sparc, y nuestro servidor de shell y varios servidores clientes funcionan con OpenBSD/i386.

  • FSC Internet Corp.
    Una gran firma de desarrollo de Seguridad Informática e Internet ubicada en Toronto que ha usado OpenBSD y su soporte de IPsec para construir una VPN segura y flexible para un cliente multimillonario. «Estamos encantados con el funcionamiento y resultados de OpenBSD, su fiabilidad, y su actitud proactiva hacia la seguridad,» dice el portavoz de la compañía. «Tenemos la intención de usar OpenBSD en muchos proyectos en el futuro. Creemos firmemente que las soluciones de código abierto como OpenBSD pueden ofrecer mejor los altos niveles de seguridad que requieren nuestros clientes -- el software de código cerrado nunca recibe el nivel de revisión de código al que está sometido OpenBSD.»

  • SoftQuad Software Inc.
    Este creador de software de edición para HTML y XML usa OpenBSD para sus pasarelas, FTP, y servicios de web.

  • Hobbiton.org
    Este proveedor de servicios de Internet usa OpenBSD para su servidor gratuito de shell, así como otros sistemas. El servidor de shell, un simple AMD K6/233, gestiona bastante más de 10.000 usuarios. «Probamos OpenBSD después de tener problemas constantes de seguridad con otros sistemas operativos», dice Leif Pedersen, de Hobbiton. «Desde entonces, la seguridad en el sistema operativo no ha dado ningún problema y, además, los sistemas han sido más estables.»

  • Prague Institute of Chemical Technology, República Checa
    La universidad usa OpenBSD en PCs para dar acceso de WWW, correo y shell a sus miembros y estudiantes, y en una SPARC IPX para un servidor de hora y DNS secundario. Las estaciones en administración también tienen PCs con OpenBSD.
