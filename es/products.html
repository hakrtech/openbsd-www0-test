Productos para OpenBSD

OpenBSD anima a las compañías y a los desarrolladores independientes a que programen productos que puedan usarse con OpenBSD, o que estén basados en OpenBSD.

Existe un servicio de asistencia comercial de asistencia para OpenBSD disponible en nuestra lista de distribuidores de nuestra página de asesoría.

  • Sistemas OpenBSD preinstalados

    • Productos basados en OpenBSD

  • ePipe by Stallion Technologies
    Stallion Technologies usa OpenBSD como sistema operativo integrado en su pasarela de conexión de redes para medianas y pequeñas empresas. Esta pasarela tiene como fin el abaratamiento de los altos costes que supone la conexión de Redes Virtuales Privadas para las pymes, y para proveedores de servicios de aplicaciones que ofrezcan aplicaciones hospedadas a las pymes. ePipe funciona sobre un procesador Motorola ColdFire 5307 VL-RISC, y usa el software de cifrado integrado en OpenBSD para obtener conexiones seguras en ambas partes de la comunicación.

  • NFR
    NFR usa OpenBSD como plataforma principal de desarrollo para su software de detección de intrusos "Network Flight Recorder". El paquete NFR se encuentra disponible como un producto independiente para OpenBSD o como un producto integrado que se llama NFR Intrusion Detection Appliance, y que ejecuta OpenBSD desde dentro. En sus páginas, NFR no informa a sus clientes de que el sistema operativo dentro de IDA es OpenBSD.

  • Anzen Computing
    Anzen Computing vende Anzen Flight Jacket, un sistema de detección de intrusos más completo que está basado en NFR. Esta compañía también ha escogido de modo independiente OpenBSD como su sistema empaquetado principal. De acuerdo con un informe técnico de Anzen, OpenBSD es el sistema operativos más rápido para NFR.

  • Hiverworld
    Hiverworld provee un examen continuo de vulnerabilidades en una aplicación que funciona sobre OpenBSD, y están trabajando sobre un IDS Gigabit y 100Mbit (utilizando OpenBSD) que lanzarán más adelante durante este año.

  • Network Security Wizards
    Network Security Wizards produce Dragon IDS para la plataforma OpenBSD. Dragon es un paquete basado en un sistema de detección de intrusos que puede operar con 1.200 firmas. Las firmas identifican todas las fases de la red que estén siendo mal usadas, incluidos sondeos, ataques y posibles peligros reales. Dragon opera en redes 100Mb/s con saturación total, y muchos clientes de Dragon operan en 200Mb/s sobre segmentos Gigabit Ethernet. NSW también vende máquinas 2U con OpenBSD + Dragon preinstalados, para aquellos que sólo quieran enchufarlo y que funcione.

  • RTMX
    RTMX vende una versión de OpenBSD que contiene un complemento entero de funcionalidades en tiempo real POSIX añadido. Han donado el código fuente para estas extensiones, y estos cambios se integrarán pronto en OpenBSD.

    • Productos de Hardware/Software para OpenBSD

  • PowerCrypt by GTGI
    Global Technologies Group, Inc. produce PowerCrypt, una placa PCI bus-mastering, que incorpora el procesador de cifrado Hi/fn 7751. El soporte es a nivel del núcleo del sistema y ha sido desarrollado por OpenBSD, con los controladores de dispositivos y el código de IPSec escrito por miembros de la comunidad de OpenBSD.

  • C2Net's Stronghold SSL server
    C2Net ofrece una licencia para RSA para un dæmon HTTPS con criptografía fuerte de 128 bits, basado en el Sevidor de Web de Apache sobre OpenBSD con su premiado servidor Stronghold 3 (OpenBSD contiene un SSL para httpd, pero su uso en los EE.UU. requiere la licencia de RSA, que el lo que C2Net ofrece. Este producto también incluye soporte y herramientas fáciles para la asistencia a los administradores nuevo de https).

  • Covalent's Raven SSL
    Covalent Technologies Inc. ofrece para OpenBSD criptografía de 128 bits de RSA con licencia para el más que concocido Servidor de Web Apache, con su premiado módulo Raven SSL Module (OpenBSD contiene httpd con capacidad para SSL, pero su uso en los EE.UU. requiere una licencia de RSA, que Raven puede ofrecer. También incluye soporte y herramientas fáciles para ayudar a los nuevos administradores de http).

  • Lan Media Corporation
    LMC vende varias tarjetas de seriales sincrónicas de PCI para T1 y HSSI/DS3, que tienen soporte en OpenBSD desde la versión siguiente a la 2.5.

  • Open Sound System (sólo i386)
    Una versión comercial de los contoladores de dispositivos de audio de OpenBSD/i386 disponible en http://www.opensound.com/. Estos controladores substitutos son compatibles con los controladores de audio que vienen con OpenBSD/i386, pero tienen soporte para tarjetas adicionales y aprovechan al máximo las características de estas tarjetas. Este paquete se encuentra disponible para OpenBSD/i386 2.6.

  • Core Systems
    Core Systems vende InSite, una utilidad de estadísticas de web para servidores, fácil de usar, para OpenBSD/i386. InSite es parecido a otros productos como WebTrends, pero también se puede configurar a través de una interfaz de web para generar informes al vuelo, usando muy poco tiempo de CPU (Core puede proveer InSite para otras plataformas que no sean i386 por encargo).

  • Software2Go Motif (sólo i386 y SPARC)
    Software2Go, LLC ofrece "Motif 2.1.20 Development and Runtime", unos juegos de herramientas de tiempo real y desarrollo en Motif para OpenBSD.
    La distribución en desarrollo es para desarrolladores de software que hagan aplicaciones de interfaces gráficas de usuario con el estilo de Motif. La distribución de tiempo real se necesita para distribuir aplicaciones solo de binarios.
    Contacto: Software2Go, LLC Online Ordering, 817-431-8775 (phone/fax)
