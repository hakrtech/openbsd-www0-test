Productos para OpenBSD

OpenBSD anima a las compañías y a los desarrolladores independientes a que programen productos que puedan usarse con OpenBSD, o que estén basados en OpenBSD. Le invitamos a ver la lista de productos que sigue.

Existe un servicio de asistencia comercial de asistencia para OpenBSD disponible en nuestra lista de distribuidores de nuestra página de asesoría. También puede encontrar una galería de usuarios con compañías y proyectos relacionados con OpenBSD.

  • ePipe by Stallion Technologies
    Stallion Technologies usa OpenBSD como sistema operativo integrado en su pasarela de conexión de redes para medianas y pequeñas empresas. Esta pasarela tiene como fin el abaratamiento de los altos costes que supone la conexión de Redes Virtuales Privadas para las pymes, y para proveedores de servicios de aplicaciones que ofrezcan aplicaciones hospedadas a las pymes. ePipe funciona sobre un procesador Motorola ColdFire 5307 VL-RISC, y usa el software de cifrado integrado en OpenBSD para obtener conexiones seguras en ambas partes de la comunicación.

  • NFR
    NFR usa OpenBSD como plataforma principal de desarrollo para su software de detección de intrusos "Network Flight Recorder". El paquete NFR se encuentra disponible como un producto independiente para OpenBSD o como un producto integrado que se llama NFR Intrusion Detection Appliance, y que ejecuta OpenBSD desde él (desafortunadamente, creemos que es OpenBSD 2.3). En sus páginas, NFR no informa a sus clientes de que el sistema operativo dentro de IDA es OpenBSD.

  • Anzen Computing
    Anzen Computing vende Anzen Flight Jacket, un sistema de detección de intrusos más completo que está basado en NFR. Esta compañía también ha escogido de modo independiente OpenBSD como su sistema empaquetado principal. De acuerdo con un informe técnico de Anzen, OpenBSD es el sistema operativos más rápido para NFR.

  • Covalent's Raven SSL
    Covalent Technologies Inc. ofrece para OpenBSD criptografía de 128 bits de RSA con licencia para el más que concocido Servidor de Web Apache, con su premiado módulo Raven SSL Module (OpenBSD contiene httpd con capacidad para SSL, pero su uso en los EE.UU. requiere una licencia de RSA, que Raven puede ofrecer. También incluye soporte y herramientas fáciles para ayudar a los nuevos administradores de http).

  • Network Security Wizards
    Network Security Wizards produce el sistema para detección de intrusos "Dragon Intrusion Detection". Este paquete puede operar con 1.200 firmas. Las firmas identifican todas las fases de mala utilización de la red, incluidas pruebas y ataques. Dragon funciona en redes de 100Mb/s totalmente saturadas, y muchos de los clientes de Dragon operan en segmentos de Ethernet Gigabit de 200Mb/s.

  • Hiverworld
    Hiverworld vende una solución para hardware basada en OpenBSD, HiverMute Network Security Appliance, que hace valoraciones constantes sobre vulnerabilidades en una Intranet.

  • Lan Media Corporation
    LMC vende varias tarjetas de seriales sincrónicas de PCI para T1 y HSSI/DS3, que tienen soporte en OpenBSD desde la versión siguiente a la 2.5.

  • RTMX
    RTMX vende una versión de OpenBSD que lleva añadida un complemento completo de características de tiempo real POSIX. RTMX nos ha cedido el código fuente para estas extensiones y estos cambios serán integrados pronto en OpenBSD.

  • Software2Go Motif (sólo i386 y SPARC)
    Software2Go, LLC ofrece "Motif 2.1.20 Development and Runtime", unos juegos de herramientas de tiempo real y desarrollo en Motif para OpenBSD.
    La distribución en desarrollo es para desarrolladores de software que hagan aplicaciones de interfaces gráficas de usuario con el estilo de Motif. La distribución de tiempo real se necesita para distribuir aplicaciones solo de binarios.
    Contacto: Software2Go, LLC Online Ordering, 817-431-8775 (phone/fax)

  • Core Systems
    Core Systems vende InSite, una utilidad de estadísticas de web para servidores, fácil de usar, para OpenBSD/i386. InSite es parecido a otros productos como WebTrends, pero se puede configurar a través de una interfaz de web para generar informes al momento, haciendo uso de muy poco tiempo de UPC (si se requiere, Core puede ofrecer InSite para otras plataformas que no sean i386).

  • Open Sound System (sólo i386)
    Una versión comercial substancialmente mejorada de los controladores de dispositivos de audio de OpenBSD/i386, que se encuentra disponible en http://www.opensound.com/. Estos controladores son compatibles con los controladores de audio que vienen con OpenBSD/i386, pero tienen soporte para muchas tarjetas adicionales y aprovechan al máximo las características de estas tarjetas. Este paquete está disponible para OpenBSD/i386 2.6.

  • Sistemas OpenBSD preinstalados en Oregon, EE.UU.
    Chris Cappuccio vende sistemas con OpenBSD preinstalado y optimizado para servidores, estaciones de trabajo, o enrutadores. Incluye opciones especiales de red y de configuración de "X Window System".

  • Sistemas OpenBSD preinstalados en el Reino Unido
    Knowledge Matters Ltd. vende systemas con OpenBSD funcionando para varias aplicaciones. Su mercado principal es el de los servicios de Internet, enrutadores y plataformas para utilidades. Knowledge Matters está ubicado en Londres (Inglaterra).
