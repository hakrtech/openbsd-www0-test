OpenBSD/sun3

El porte de OpenBSD/sun3 funciona en la actualidad en todas la máquinas Sun3 basadas en la 68020, excepto las 3/E.

Cronología y Estado Actual:

El porte para Sun3 de OpenBSD deriva del porte NetBSD/sun3, cuando los portes para sun3 y sun3x todavía eran diferentes. Las pequeñas partes de OpenBSD que son totalmente específicas del porte para Sun3 son básicamente las mismas que sus correspondientes en NetBSD.

Hardware con Soporte:

OpenBSD/sun3 funciona en los siguientes modelos Sun3:
  • Sun-3/50
  • Sun-3/60, 3/60LE
  • Sun-3/75, 3/140, 3/150, 3/160 y 3/180
  • Sun-3/110
  • Sun-3/260 y 3/280
Otros modelos de Sun3 NO tienen soporte, en especial los basados en la 68030 (sun3x). OpenBSD no funcionará en máquinas 3/160 con sólo 2MB de memoria, y funcionará a duras penas en máquinas con 4MB.

Dispositivos con soporte:

  • Puertos serie integrados
  • Ratón y teclado Sun type 3
  • Vídeo VME o integrado (bwtwo, cgtwo, cgfour)
  • Ethernet VME o integrada (intel, lance)
  • Controladora SCSI sun-3 VME o integrada, en la actualidad con algunas limitaciones
  • Las tarjetas xylogics VME las reconoce, pero aún no funcionan
La controladora SCSI sun-2 todavía no tiene soporte.

La versión completa de OpenBSD/sun3 se encuentra disponible por ftp; las versiones de prueba ("snapshots") se ponen a disposición cada cierto tiempo.
Antes de intentar instalar OpenBSD/sun3 debería leer las notas de instalación para esta versión.

El primer arranque del sistema tardará bastante tiempo ya que tiene que generar una clave DSA para ssh. Puede llegar a tardar unos 15 minutos en una 3/160, pero este periodo de tiempo puede variar mucho dependiendo de la máquina. Este proceso sólo se lleva a cabo una sola vez, y no hay porqué preocuparse.

Puede usar cvs para obtener el X Window System. Kenneth Stailey escribió unos parches para OpenBSD/sun3 que se debín haber integrado en la corriente principal del código fuente de XFree86. Se encuentran disponibles los clientes y las bibliotecas de X, pero todavía no existe un servidor de X.

Proyectos sobre los que se está trabajando (sin ningún orden):

  • Dispositivos "call up" (/dev/cua*).
  • Dispositivo SCSI sun-2 (sc).
  • Migración del porte del sistema de memoria VM a UVM (en progreso).
  • Reprogramación del gestor de arranque y de los procedimientos de instalación para que compartan tanto código con el porte para Sparc como sea posible.

El mantenedor del porte de OpenBSD para Sun3 es Miograg Vallat <miod@openbsd.org>.
