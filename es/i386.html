i386

Instalar con Otros Sistemas Operativos:

El porte de OpenBSD/i386 se puede instalar para que comparta los discos del sistema con otros sistemas operativos como MSDOS, Windows o Linux, usando el esquema de la partición del MBR de MSDOS, y un gestor de arranque opcional. Una vez hecho, las operaciones normales no tienen ningún problema, pero la configuración sí que debe hacerse con cuidado ya que la instalación de Windows 95 es famosa por el poco cuidado que tiene con la información en particiones MBR ya existentes. Para más información lea las Notas de Instalación.

Compatibilidad de Binarios con Otros Sistemas Operativos:

El porte para i386 de OpenBSD puede ejecutar binarios de Linux, FreeBSD, BSD/OS, SVR4 (incluido Solaris), y IBDS2.

Hardware Soportado:

El porte para i386 de OpenBSD funciona en un amplio rango de PCs y clónicos de éstos, con una gran variedad de procesadores y arquitecturas con bus de E/S. Se puede instalar y ejecutar con una dificultad mínima sobre la mayoría de productos actuales. Los casos en los que se puede encontrar problemas son por lo general en PCs antiguos con hardware propietario, portátiles o sevidores especializados que dependen de una BIOS a medida para ocultar diferencias de implementación.

Actualmente OpenBSD no tiene soporte para procesadores múltiples (SMP), pero funcionará usando un sólo procesador sobre una placa de un sistema de multiprocesadores.

Existe soporte para la mayoría de portátiles. Disponemos de una página en donde se detallan las características específicas de los portátiles más comunes.

Hardware No Soportado

El hardware que no tenga soporte puede tenerlo en un futuro; ¡su donación de tiempo, hardware o documentación puede acelerar este soporte!

  • Adaptec 2940U2W (aic7890) (pero el 2940UW sí tiene soporte)
  • "Micro Channel" MCA bus usado en muchos modelos IBM PS/2
  • Adaptadores SCSI para huésped basados en NCR 5380 y 53400
  • Controladores de almacenamiento QIC-40 and QIC-80 (Son los controladores de almacenamiento que conectan con el controladore de de la disquetera)
  • Placas base con Multiprocesadores (Aunque funcionan correctamente usando sólo un procesador) Existe un proyecto para desarrollar este soporte.
  • Interfaces propietarias de CDROM Sony y Panasonic
  • Controladores para ZIP de puerto paralelo (los ZIP SCSI y ATAPI funcionan bien)
  • Controladores y Adaptadores Cardbus
  • PCMCIA:
    • 3COM 3c574
    • Tarjetas de memoria
    • La mayoría de tarjetas multifuncionales

Si está buscando probar las nuevas características de la revisión de pre-lanzamiento, puede probar uno de los "snapshots". Para la arquitectura i386 existen "snapshots" disponibles de vez en cuando".
