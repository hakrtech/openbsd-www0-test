Documento: Abriendo el depósito de código fuente con CVS anónimo

Abriendo el depósito de código fuente con CVS anónimo, Charles D. Cranor y Theo de Raadt
CVS anónimo es una mecanismo avanzado de distribución de código fuente que nosotros creamos para permitir a los proyectos de software de código abierto distribuir código fuente e información acerca del código a usuarios de Internet. Construido encima del sistema de control de revisiones Sistema de Versiones Concurrentes (CVS por sus siglas en inglés), CVS anónimo permite el acceso seguro de sólo lectura a un depósito CVS de código fuente. El mismo documento en formato PDF anoncvs-paper.pdf
