为什么 OpenBSD 的开发人员使用 CVS

OpenBSD 项目组选择使用并发版本系统 (CVS)。 使用 CVS，大约 70 个开发人员可以更新 OS (大约 1.2GB，140,000 个文件)，ports/packages, 和 X Window System 的源代码。 很多 OpenBSD 用户通过匿名 CVS 服务器下载 OS 源代码，它比软件升级协议 (sup(1)) 更加可靠。

CVS 还能容易的创建和维护分支，从而我们可以提供发行版本 + 补丁的源代码， 以及最新的开发版本源代码，执行并行开发。使用 CVSweb 还可以给所有人提供方便而有导航能力的 WEB 访问接口。

CVS 与安全外壳 (ssh(1)) 的接口能提供可靠的用户和主机认证。它很好的满足了 OpenBSD 项目的安全目标。
