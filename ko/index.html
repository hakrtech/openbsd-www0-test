기본 설치시에 6년 가까이 동안 원격 취약점이 한개만 존재합니다.

OpenBSD 프로젝트는 Free이고, 멀티-플랫폼 4.4BSD에 기반을 둔 UNIX계열의 운영체제를 제공합니다.
우리들은 이식성, 표준화, 정확성, 미리 대처하는 보안 과 통합된 암호에 노력들을 기울이고 있습니다.
OpenBSD는 SVR4 (Solaris), FreeBSD, Linux, BSD/OS, SunOS 그리고 HP-UX에서 사용되어지는 대부분의
프로그램들을 바이너리 에뮬레이션으로 지원합니다.

OpenBSD 는 우리의 FTP 사이트들로 부터 자유롭게 이용이 가능하고, 또한 저렴한 가격인 2개의 CD세트도 이용이 가능합니다.
현재 릴리즈는 2002년 5월 19일에 나온 OpenBSD 3.1 입니다. OpenBSD 3.2 는 11월 1일 릴리즈 예정이고, 주문 예약이 시작되었습니다.(제품은 주문의 순서대로 보내집니다.)
CD(그리고 T셔츠)주문이 가능합니다...

OpenBSD 에는 OpenSSH 가 포함되어 있습니다. OpenSSH 는 SSH1과 SSH2를 모두 지원합니다.!

OpenBSD 는 자원봉사자들로부터 개발 되고 있습니다. 프로젝트 개발과 릴리즈 기금은 기부를
받는 것 뿐만 아니라,CD들과 T-셔츠들의 판매로 이루어지고 있습니다. 조직들과 개인들의 기부 덕분에,
OpenBSD 는 계속해서 존재할 것이며, 모든이가 이용과 적합한 재사용을 위해서, 변함없이 Free로 남을것입니다.
