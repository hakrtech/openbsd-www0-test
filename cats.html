OpenBSD/cats

OpenBSD/cats runs on the Simtec Electronics StrongARM 110 Evaluation Board.

A mailing list for ARM-based ports is available at arm@openbsd.org. To join the OpenBSD/arm mailing list, send a message body of "subscribe arm" to majordomo@openbsd.org. Please be sure to check our mailing list policy before subscribing.

The current port maintainer is Dale Rahn (drahn@openbsd.org). Others are definitely welcome to contribute! We are also evaluating other possible ARM platforms.

History:

OpenBSD/cats was started based on NetBSD -current code of January 2004.

Current status:

The Simtec StrongARM 110 Evaluation Board is the only supported board at the current time.

X11:

The port is too young to think about graphics, however vga console text mode is supported.

Projects (in no particular order):

  • Shared libraries (in progress).
  • Better booting support from ABLE firmware.
  • Kernel Symbols (currently unavailable due to boot process limitations).

Supported peripherals:

Currently only the following hardware has been tested:

  • Onboard PCI modules (pci)
    • Acer Labs M5229 UDMA IDE
  • PCI-Bridges (pchb)
    • Acer Labs M1533 ISA
  • Video Cards
    • S3 PCI VGA
  • Ethernet
    • Novell NE1000, NE2000 (ISA, and PCI variants thereof) (ne) (B)
    • VIA Rhine/RhineII/RhineIII Ethernet adapters (vr), including:
      • Addtron AEF-360TX
      • Hawking PN102TX
      • D-Link DFE530TX
    • DEC DECchip 21143 (Tulip clones) (dc)
    • Intel i8255x-based (except the i82556) PCI adapters (fxp), including:
      • Intel EtherExpress PRO/10+
      • Intel EtherExpress PRO/100, PRO/100B, and PRO/100+
      • Intel EtherExpress PRO/100+ "Management Adapter"
      • Intel EtherExpress PRO/100 Dual Port
      • Intel PRO/100 VE, PRO/100 VM, and PRO/100 S
    • Intel 21145-based PCI adapters (dc)
  • SCSI and IDE Host Adapters
    • Symbios Logic 53c810, 53c825 (siop)
    • Adaptec AIC-7770-based SCSI host adapters (including the Adaptec AHA-274x and AHA-284x families). (ahc)
    • Adaptec AHA-[23]94x[W] cards and some on-board PCI designs using the AIC7870 and AIC7880 chips. (ahc)
    • Adaptec AHA-[23]9[34]xU[2W] cards and on-board PCI designs using the AIC789[01], AIC7895, AIC789[67] chips. (ahc)
    • Adaptec AIC-789[29] chips and products like the AHA-29160 based upon it which do 160MB/sec SCSI. (ahc) (However, the 7899G card is currently not supported with more than one device attached)
    • All PCI IDE (pciide) adapters supported by the i386 platform
  • Drives
    • All IDE devices supported on the i386 platform.
    • All ATAPI drives supported on the i386 platform.
    • All SCSI drives supported on the i386 platform.
  • USB Controllers/Hubs

    USB support may not be working correctly.

    • OHCI: Acer Labs M5237 USB (ohci)
  • USB Devices
    • Alps Electric Apple USB Keyboard (ukbd)
    • Solid Year Keyboard and Mouse (ukbd) (ums)
    • Logitech M4848 (ums)
    • Kensington USB/PS2 Wheel Mouse (ums)
    • USB Zip Drives (umass)
  • Other Devices
    • Serial Ports: the onboard serial port should work as fcom serial console or as a standard serial port.

Getting and installing OpenBSD/cats:

Snapshots are made available from time to time, in this location as well as in a few mirrors. Due to the very young age of this port, there are currently no installation instructions yet.
