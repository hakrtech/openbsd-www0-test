Sistema de Rastreamento de Bugs

O Sistema de Rastreamento de Bugs do OpenBSD foi desativado.

Relatos sobre novos bugs ainda devem ser enviados para bugs@openbsd.org, preferencialmente utilizando a ferramenta sendbug(1).

Para maiores detalhes sobre reportar bugs podem ser vistos aqui.

Os status dos relatos de bugs podem ser consultados nos arquivos da lista bugs@openbsd.org.
