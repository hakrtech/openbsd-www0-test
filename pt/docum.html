  • Info - Sistema de Navegação de documentação.
  • Bison - "Yet Another Compiler".
  • CPP - O preprocessador GNU C.
  • Emacs - O extensivo editor de texto auto-documentado.
  • Forms mode - Um modo de edição baseado em "form".
  • G++ & GCC - Os compiladores GNU C++ e C.
  • G77 - O compilador GNU Fortran.
  • GDB - O depurador de C.
  • Gawk - A implementação GNU do AWK.
  • Gnus - O leitor/compositor de news GNU.
  • Gprof - O perfilador GNU.
  • libstdc++ - A biblioteca padrão C++ GNU.
  • Make - A implementação GNU do Make.
  • rx & regex - As bibliotecas de expressões regulares.
  • Tar - GNU Tar.
  • Termcap - A biblioteca termcap, que permite que aplicações possam lidar com qualquer tipo de terminal.
  • Texinfo - Com somente um arquivo fonte, crie um manual impresso (através do TeX) ou um arquivo Info (através do texinfo).
  • VIP - Um modo de emulação do Vi para Emacs.
  • Viper - Um novo emulador de Vi para Emacs.
  • VM - Uma nova interface de correio (compatível com pastas Elm e mailx).
  • Calc - Calculadora GNU Emacs -- calculadora numérica e de símbolos baseada em RPN.
  • Elisp - Como estender o Emacs.
  • Babyl - O formato (BABYL) do ~/RMAIL, usado pelo rmail do Emacs.
  • Perl - Documentação do Perl.
  • GNATS - O sistema de rastreamento de erros GNU.
  • groff - Implementação GNU do troff.
  • screen - Gerenciado de terminais virtuais.

Outros Sistemas Operacionais do tipo UNIX

  • NetBSD - SO 4.4BSD Multiplataforma.
  • FreeBSD - Avançado SO 4.4BSD para i386 e posteriores.
  • Sistema Operacional MIT Exokernel - Utiliza as ferramentas e bibliotecas padrão do OpenBSD, além de implementar um interface para as chamadas de sistema do OpenBSD.
  • Flux - novo projeto Mach em Utah.
  • Linux - Sistema Operacional do tipo Unix do Linus Torvalds para i386 e maiores.
  • Sistema Operacional GNU da Free Software Foundation baseado no kernel Mach/Hurd.
  • Minix - Clone Unix V7 do Andy Tanenbaum.
  • Minix - cloca Unix V7 do Andy Tanenbaum
  • Amoeba - SO distribuído por Andy Tanenbaum.
  • Sprite - O Sistema Operacional Sprite.
