Various commercial products for OpenBSD are available from the following companies:

  • NFR
    NFR uses OpenBSD as their primary development platform for their Network Flight Recorder intrusion detection software. They make demo CD that includes NFR, running OpenBSD.

  • Anzen Computing
    Anzen Computing sells Anzen Flight Jacket, a more comprehensive intrustion detection system based on NFR.. they have chosen OpenBSD as their primary packaged system. As well, they report that OpenBSD is the fastest platform to run NFR on...

  • Hiverworld
    Hiverworld sells an OpenBSD-based hardware solution which does constant Intranet vulnerability assesment, called the HiverMute Network Security Appliance.

  • Lan Media Corporation
    LMC sells several PCI-based synchronous serial cards, for T1 and HSSI/DS3, which are supported in the OpenBSD, starting after the 2.5 release.

  • RTMX
    RTMX sells a version of OpenBSD which has a full complement of POSIX realtime features added to it.

  • Motif
    (i386 only)
    Software2Go,LLC has Motif 2.1.10 Development Edition for OpenBSD/i386 2.3. Unfortunately this has not been updated for newer releases...

  • Open Sound System
    (i386 only)
    A substantially spiffed-up commercial version of the OpenBSD/i386 audio device drivers is available from http://www.opensound.com/. These replacement drivers are compatible with the audio drivers OpenBSD/i386 ships with, but they support many additional cards and take full advantage of these cards' features. This package is available for OpenBSD/i386 2.5

  • Pre-installed OpenBSD systems
    Chris Cappuccio sells systems with OpenBSD installed and optimized for a server, workstation, or a router. Includes special networking options and X Windows setup.
