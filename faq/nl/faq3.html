3.1 - Een OpenBSD CD set kopen

Een OpenBSD CD set aankopen is in het algemeen de beste manier om van start te gaan. Bezoek de bestelpagina om uw kopie aan te kopen: OpenBSD bestelpagina.

Er zijn veel goede redenen om een OpenBSD CD set te bezitten:

  • CD verkoop steunt de voortdurende ontwikkeling van OpenBSD.
  • Ontwikkeling van een multi-platform besturingssysteem vereist voortdurend investering in apparatuur.
  • Uw steun in de vorm van een CD set aankoop heeft een werkelijke impact op de toekomstige ontwikkeling.
  • De CD set bevat binaries (en broncode) voor de meest populaire ondersteunde platformen.
  • De CD's zijn bootable op verscheidene platformen en kan gebruikt worden om een machine te bootstrappen zonder een voorafbestaand geïnstalleerd besturingssysteem.
  • De CD's zijn nuttig om te bootstrappen zelfs als u verkiest een snapshot te installeren.
  • Installeren vanaf CD is sneller! Installeren vanaf CD spaart ook de netwerkmiddelen.
  • OpenBSD CD's gaan altijd gepaard met heel leuke stickers. Uw systeem is niet helemaal compleet zonder. U kan deze stickers enkel bekomen door een CD set te kopen of door hardware te schenken.
  • OpenBSD CD sets bevatten een assortiment aan nuttige en populaire packages. De CD set is compleet genoeg om een volledige werk- en ontwikkelomgeving klaar te zetten zonder netwerkverbinding.

Als u een release versie van OpenBSD installeert, gebruikt u best een officiële CD set.

3.3 - Voorziet OpenBSD een ISO image voor download?

Vanaf OpenBSD 4.2, voor bepaalde platformen, ja!

Gebruikers van de alpha, amd64, hppa, i386, macppc, sparc en sparc64 platformen kunnen nu een ISO image downloaden en installeren die kan gebruikt worden om een CD-ROM te maken die OpenBSD volledig kan starten en installeren.

Merk op dat deze ISO niet hetzelfde is als de officiële CD set. Deze images zijn voor één platform en bevatten geen voorgecompileerde packages, stickers of kunstwerk zoals de officiële CD set.

Zoals voorheen is de installatie via het ISO bestand voor veel mensen echter niet de optimale installatiemethode. Het is meestal nog steeds sneller en eenvoudiger om de opstartmedia te downloaden en dan gewoon de benodigde onderdelen te installeren. Voor hen die echter een aantal installaties wensen te doen of niet kunnen uizoeken hoe ze tien bestanden op een CD-ROM plaatsen of een lokale FTP server opzetten, zijn er ISO's beschikbaar.

Het OpenBSD project stelt de ISO images die gebruikt worden om de officiële CD's te maken, niet ter beschikking voor download. De reden is gewoon dat we graag hebben dat u de CD sets koopt om de voortdurende ontwikkeling van OpenBSD te helpen steunen. De officiële OpenBSD CD-ROM layout is copyright Theo de Raadt. Theo laat niet toe dat mensen de images van de officiële OpenBSD CD's verder verspreiden. Als stimulans voor mensen om de CD set te kopen, worden er enkele extra's bijgevoegd in het pakket (artwork, stickers enz).

Merk op dat alleen de CD layout copyrighted is, OpenBSD zelf is vrij. Niets belet dat iemand anders OpenBSD downloadt en zijn eigen CD maakt.

Voor hen die een bootable CD voor hun systeem nodig hebben, zijn er bootdisk ISO images (cd44.iso genoemd) beschikbaar voor een aantal platformen die toelaten om de rest van het systeem via FTP te installeren. Deze ISO images zijn slechts enkele megabytes groot en bevatten enkel de installatie tools, niet de eigenlijke bestandensets.

3.4 - Downloaden via FTP, HTTP of AFS

Er zijn tal van internationale mirror sites die FTP en HTTP toegang aanbieden tot OpenBSD releases en snapshots. AFS toegang is ook beschikbaar. U gebruikt best steeds de site het dichtst bij u. Voor u een release of snapshot begint af te halen, gebruikt u misschien even ping(8) en traceroute(8) om na te gaan welke mirror site zich het dichtst bij u bevindt en of die mirror ook goed genoeg werkt. Natuurlijk is uw OpenBSD release CD altijd dichter bij dan gelijk welke mirror. Informatie over toegang vindt u hier:

OpenBSD FTP pagina.

3.5 - Verkrijgen van de Current Broncode

De broncode van OpenBSD is vrij verspreidbaar en beschikbaar zonder kosten. In het algemeen is de beste manier om van start te gaan met een recente source tree om de broncode van de meest recente CD te installeren en vervolgens AnonCVS te configureren om hem regelmatig te updaten. Informatie over AnonCVS, inclusief hoe het in te stellen, is hier beschikbaar:

OpenBSD AnonCVS pagina.

Nog een ander alternatief is om de broncode van het web af te halen. U kan dat doen via cvsweb op: http://www.openbsd.org/cgi-bin/cvsweb/.
