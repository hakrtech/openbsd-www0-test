3.1 - OpenBSD CD の購入

通常は OpenBSD CD を購入することからはじめるのが適当でしょう。 CD は、注文ページから購入することができます。 http://www.openbsd.org/orders.html

OpenBSD CD を購入することには、次のような多くの利点があります。

  • CD の売上は、現在行われている OpenBSD の開発資金として利用されます。
  • マルチプラットフォーム対応のオペレーティングシステムの開発には、 開発機器への定期的な投資が必要です。
  • あなたの CD を購入は、 将来の開発を支えるのです。
  • CD には、対応プラットフォームすべてのバイナリ (とソース) が含まれています。
  • CD は、いくつかのプラットフォームで起動メディアとして利用することができます。そのため、予め オペレーティングシステムがインストールされていないマシンの起動に利用することが可能です。
  • CD はスナップショット版をインストールする際の 土台として使うことができます。
  • CD からのインストールは高速です! また、CD からのインストールは、 ネットワークの帯域も節約してくれます。
  • OpenBSD CD には素敵なステッカーがついてきます。これなくしては 完成されたシステムとは言えません。このステッカーは、CD セットを購入するか、 ハードウェアを寄贈しなければ手に入りません。

リリース版の OpenBSD をインストールするなら、 CD を使いましょう。

3.2 - OpenBSD T シャツの購入

はい、OpenBSD では、おしゃれな T シャツの販売も行っています。 実際のものは http://www.OpenBSD.org/tshirts.html をご覧ください。 ぜひ買ってくださいね :)

3.3 - OpenBSD の ISO イメージはダウンロード可能でしょうか ?

いくつかの他のオープンソースのオペレーティングシステムは、一般に CDROM の ISO イメージとして 配布されています。これは、OpenBSD がどのように配布されているかということではありません。

OpenBSD プロジェクトでは、公式 CD のマスタとして使用された ISO イメージを ダウンロードして利用できるようにはしていません。この理由は簡単で、 あなたに CD セットを購入していただき、OpenBSD の開発を支援して欲しいと 考えているからです。公式の OpenBSD CD-ROM のレイアウトは Theo de Raadt 氏が 著作権を所有しており、Theo 氏は公式 CD のイメージの再配布を禁止しています。 CD セットを購入してくれる人へのサービスの一つとして、いくつかの特典 (アートワークやステッカーなど) もついています。

ただし、著作権の対象となっているのは CD レイアウトのみであり、 OpenBSD そのものはフリーであることに注意してください。どなたでも OpenBSD をダウンロードして、独自に CD を作成しても一向に構いません。 何らかの理由で CD イメージをダウンロードしたいなと思ったら、 メーリングリストアーカイブを探すと良いでしょう。もちろん、 インターネット上にある OpenBSD ISO イメージは、Theo de Raadt 氏の著作権を冒すものか、公式とは異なるもののいずれかです。 非公式イメージの提供元が信頼できるかどうかは、あなたが決めてください。

わたしたちは、無料で OpenBSD をダウンロードしたい人々に FTP インストールを推奨しています。 起動可能な CD を必要とする人たちのために、起動ディスクの ISO イメージ (名称は cd34.iso) が、システムの残りの部分を FTP 経由でインストール することができる、多くのプラットフォーム向けに用意されています。これらの ISO イメージは数メガバイト程度のサイズしかなく、実際のファイル一式ではなくて、 インストールに必要なツールだけを含むものです。

3.4 - FTP や AFS 経由でのダウンロード

世界中には、OpenBSD のリリース版やスナップショット版を FTP で提供している、 たくさんのミラーサイトが存在します。AFS によるアクセスも提供されています。 これらのサイトを使う場合は、最も近いサイトを選びましょう。リリース版や スナップショット版をダウンロードする前に、そのミラーサイトが最も近く、 十分な転送速度を持っているかどうかを ping(8) や traceroute(8) で調べてください。もちろん、OpenBSD のリリース版 CD は、どのミラーサイト よりもあなたの近くにあります。アクセスに関する情報は、次の場所にあります。

http://www.openbsd.org/ftp.html

3.5 - 最新のソースコードの入手

OpenBSD のソースは、無料で自由に入手・再配布することが可能です。 一般に最新のソースツリーを入手する最適の方法は、最新の CD から ソースをインストールして、AnonCVS を使って定期的に更新するように 設定することです。AnonCVS の設定法などに関する詳細は、 次の場所にあります。

http://www.openbsd.org/anoncvs.html

あるいは FAQ 8, CVS をご覧ください。

AnonCVS を利用できる十分なネットワーク帯域が確保できない、あるいは インターネットに UUCP 経由で接続しているような場合は、AnonCVS の代わりに CTM を使って最新のソースファイルを維持することが可能です。その場合は、 最新のリリース版 CD がより重要になります。CTM の設定法などの詳細は、 次の場所にあります。

http://www.openbsd.org/ctm.html

もう一つの方法は、web からソースコードを入手する方法です。 これは、次の場所にある cvsweb を使います。 http://www.openbsd.org/cgi-bin/cvsweb/
