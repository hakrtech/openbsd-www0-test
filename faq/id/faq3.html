3.1 - Membeli CD OpenBSD

Membeli CD OpenBSD umumnya cara paling bagus untuk dimulai. Kunjungi halaman pemesanan untuk mendapatkan salinan CD anda: http://www.openbsd.org/id/orders.html.

Ada beberapa alasan bagus untuk memiliki sendiri CD OpenBSD:

  • Penjualan CD mendukung pengembangan lanjut OpenBSD.
  • Pengembangan dari sebuah sistem operasi multi-platform membutuhkan investasi tetap untuk peralatan.
  • Dukungan anda dalam bentuk membeli CD mempunyai pengaruh nyata untuk pengembangan di masa depan.
  • CD berisi binari (dan kode sumber) untuk semua platform terdukung.
  • CD bootable pada beberapa platform, dan dapat digunakan untuk bootstrap sebuah mesin tanpa sebuah sistem operasi terinstall sebelumnya.
  • CD sangat berguna untuk bootstrapping meski jika anda memilih untuk menginstall sebuah snapshot.
  • Menginstall dari CD adalah lebih cepat! menginstall dari CD menghemat pemakaian sumber daya koneksi jaringan internet.
  • CD-CD OpenBSD selalu disertai stiker yang cantik. Sistem anda tidak akan lengkap tanpa stiker ini. Anda hanya bisa mendapatkan stiker ini dengan membeli CD atau menyumbangkan peranti keras.

Jika anda menginstall sebuah versi rilis OpenBSD, anda sebaiknya menggunakan sebuah CD.

3.2 - Membeli Kaos/T-Shirt OpenBSD

Ya, OpenBSD memiliki kaos untuk kepuasan berpakaian anda. Dapat anda lihat kaos kami di http://www.OpenBSD.org/id/tshirts.html. Selamat menikmati :)

3.3 - Apakah OpenBSD menyediakan ISO untuk didownload?

Beberapa sistem operasi open source lainnya umumnya didistribusikan sebagai ISO CD-ROM. Hal ini bukan cara OpenBSD didistribusikan.

Proyek OpenBSD tidak membuat ISO yang bisa di download sebagai master dari CD resmi. Alasannya sederhana yaitu kami ingin agar anda membeli CD, membantu dana pengembangan OpenBSD yang sedang berlangsung. Layout CD-ROM resmi OpenBSD hakciptanya pada Theo de Raadt. Theo tidak mengijinkan orang mendistribusikan ulang ISO dari CD resmi OpenBSD. Sebagai sebuah insentif untuk orang yang membeli CD resmi, beberapa bonus disertakan dalam paket CD (desain seni, stiker, dll).

Menjadi perhatian bahwa hanya layout CD yang dipatennkan, OpenBSD nya tetap gratis. tanpa membatasi orang lain untuk mendownload OpenBSD dan membuat CD mereka sendiri. Jika untuk beberapa alasan anda ingin mendownload ISO CD, coba cari dalam arsip milis untuk sumber yang mungkin. Tentu saja, setiap ISO OpenBSD yang ada di Internet salah satunya melanggar hakcipta Theo de Raadt atau bukan ISO resmi. Sumber ISO tidak resmi mungkin bisa atau tidak untuk dipercaya; ini terserah pada anda untuk menentukan bagi anda sendiri.

Kami sarankan bagi yang ingin download OpenBSD secara gratis gunakan opsi instalasi FTP. Untuk yang membutuhkan CD bootable bagis sistem mereka, ISO bootdisk (bernama cd34.iso) disediakan untuk sejumlah platform yang kemudian akan memperbolehkan menginstall sisanya lewat FTP. ISO ini hanya beberapa megabyte, dan berisi hanya alat instalasi, bukan berkas instalasi utama.

3.4 - Download lewat FTP atau AFS

Terdapat sejumlah situs miror internasional yang menawarkan akses FTP untuk rilis dan snapshot OpenBSD. Akses AFS juga tersedia. Anda sebaiknya selalu menggunakan situs yang terdekat dengan anda. Sebelum anda mulai mengambil sebuah rilis atau snapshot, mungkin anda gunakan dulu ping(8) dan traceroute(8) untuk menentukan situs miror mana yang terdekat dengan anda dan apakah situs terdekat berkinerja pantas. Tentu saja, CD OpenBSD rilis anda selalu terdekat dari miror manapun. Keterangan untuk mengakses di sini:

http://www.openbsd.org/id/ftp.html.

3.5 - Mendapatkan Kode Sumber Terkini

Kode sumber OpenBSD secara gratis didistribusikan ulang dan tersedia tanpa pengutan biaya sama sekali. Umumnya cara terbaik untuk memulai dengan struktur kode sumber terkini adalah menginstall kode sumber dari CD terbaru dan kemudian konfigurasi AnonCVS untuk memperbaruinya secara berkala. Keterngan tentang AnonCVS, termasuk bagaimana mengaturnya , tersedia di sini:

http://www.openbsd.org/id/anoncvs.html.

atau lihat FAQ 8, CVS

Jika anda tidak mempunyai cukup bandwidth untuk melakukan AnonCVS, atau jika akses Internet anda lewat UUCP, anda masih bisa menjaga kode sumber OpenBSD tetap terkini dengan emnggunakan CTM selain menggunakan AnonCVS. Jika hal itu sistuasi yang anda hadapi, lalu memulai dengan rilis CD terbaru akan lebih penting. Keterangan tentang CTM, termasuk bagaiamna mengaturnya, tersedia di sini:

http://www.openbsd.org/id/ctm.html.

Alternatif lain yang ada adalah dengan mendapatkan kode sumber OpenBSD dari web. Anda dapat melakukannya lewat cvsweb pada: http://www.openbsd.org/cgi-bin/cvsweb/.
