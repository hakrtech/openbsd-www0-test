3.1 - Koupě OpenBSD CD

Zakoupení CD s OpenBSD je nejlepší způsob jak začít. Navštivte tuto stránku, abyste si mohli objednat vaši kopii: OpenBSD objednací stránka.

Je zde mnoho dobrých důvodů proč mít vlastní CD s OpenBSD:

  • Prodej CD podporuje vývoj OpenBSD.
  • Vývoj multi-platformního systému vyžaduje neustálé investice do vybavení.
  • Vaše podpora ve formě zakoupení CD má skutečné dopady na budoucí vývoj.
  • CD obsahuje binárky(a zdroje) pro většinu populárních a podporovaných platforem.
  • CD jsou bootovatelné na několika platformách a mohou být využity pro start instalace bez předchozího nainstalovaného systému.
  • CD je užitečné pro start instalace dokonce chcete-li instalovat snapshot.
  • Instalace z CD je rychlejší! Instalace z CD šetří síťovou konektivitu a zdroje.
  • CD s OpenBSD vždy obsahuje velmi pěkné nálepky. Bez nich systém není kompletní. Tyto nálepky můžete získat pouze zakoupením CD nebo věnováním hardwaru.
  • CD obsahují sadu užitečných a oblíbených balíčků. CD jsou dostatečně kompletní k vytvoření plného pracovního a vývojového prostředí bez potřeby jakéhokoliv připojení k síti.

Pokud instalujete release verzi OpenBSD, tak by jste měli použít oficiální CD set.

3.3 - Poskytuje OpenBSD ISO image ke stažení?

Od verze OpenBSD 4.2,pro vybrané platformy,ano!

Uživatelé alpha, amd64, hppa, i386, macppc, sparc a sparc64 platforem nyní mohou stáhnout a instalovat ISO image,který může být použit pro vytvoření CD-ROM,které může bootovat a instalovat vše z OpenBSD.

Upozorňujeme,že toto ISO není stejné jako oficiálníl CD set. Tyto image jsou pro jednotlivé platformy a neobsahují žádné před-kompilované balíčky, nálepky nebo grafiku jako oficiální CD set.

Stejně jako předtím,instalace z ISO souboru NENÍ optimální instalační metoda pro hodně lidí.Většinou je rychlejší a lehčí stáhnout bootovací médium a poté doinstalovat jen potřebné části.Přesto jsou ISO dostupné pro lidi, kteří si přejí udělat více instalací najednou nebo neví jak umístit 10 souborů na CD-ROM nebo jak vytvořit lokální FTP server.

OpenBSD projekt také neposkytuje ke stažení ISO soubory použité k vytvoření oficiálních CD.Důvod je jednoduchý,chceme aby jste si koupili CD set a tím podpořili další vývoj OpenBSD.Struktura oficiálního CD je copyright Theo de Raadta.Theo nepovoluje lidem dále distribuovat image oficiálních CD.Jako odměna pro lidi kupující oficiální CD jsou zde přiloženy některé věci navíc (grafika, nálepky atd.).

Pouze struktura oficiálního CD je chráněna copyrightem,OpenBSD samotné je volně k využití.Nic nebrání komukoliv dalšímu stáhnout OpenBSD a vytvořit vlastní CD.

Pro ty,kteří potřebují bootovací CD pro jejich systém jsou zde bootdisk ISO image (pojmenované cd43.iso) a jsou dostupné pro většinu platforem. Umožňují,aby zbytek systému byl nainstalován přes FTP.Tyto ISO image jsou pouze několik MB velké a obsahují pouze instalační nástroje, ne aktuální sety k instalaci.

3.4 - Stažení přes FTP, HTTP nebo AFS

Je zde spousta mezinárodních mirrorů nabízejících FTP nebo HTTP přístup k OpenBSD release nebo snapshots.AFS přístup je také dostupný.Vždy by jste měli použít stránku,která je Vám nejblíže.Než začnete stahovat release nebo snapshot,tak by jste měli použít ping(8) a traceroute(8) ke zjištění,který mirror je nejblíže k vám a zda tento mirror má vyhovující odezvu.Pochopitelně,že oficiální CD je vždy bližší než jakýkoliv mirror. Přístupové informace jsou zde:

OpenBSD FTP stránka.

3.5 - Získání aktuálního zdrojového kódu

Zdrojový kód OpenBSD je volně redistribuovatelný a dostupný zdarma. Nejlepší cestou jak začít s aktuálním zdrojovým kódem je instalace zdrojů z nejaktuálnějšího CD a poté nastavit AnonCVS pro aktualizaci. Informace o AnonCVS,včetně toho jak jej nastavit jsou dostupné zde:

OpenBSD AnonCVS stránka.

Další možností je získání zdrojového kódu z webové stránky.Toto můžete udělat pomocí cvsweb na adrese: http://www.openbsd.org/cgi-bin/cvsweb/.
