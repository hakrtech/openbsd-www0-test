3.1 - Comprare un CD di OpenBSD

Comprare un CD di OpenBSD è generalmente il modo migliore di iniziare. Visita la pagina degli ordini per comprare una copia: OpenBSD ordering page.

Ci sono molte buone ragioni per possedere un CD di OpenBSD:

  • La vendita di CD aiuta lo sviluppo futuro di OpenBSD.
  • Lo sviluppo di un sistema operativo multi-piattaforma richiede un costante investimento in hardware.
  • Il vostro suppporto nella forma dell'acquisto di un CD ha un vero impatto sul futuro sviluppo.
  • Il CD contiene i binari (e i sorgenti) per tutte le piattaforme supportate.
  • Il CD è avviabile su numerose piattaforme, e può essere usato per avviare una macchina senza alcun sistema operativa pre-installato.
  • Il CD è utile per il bootstrp anche se scegli di installare uno snapshot.
  • Installare da CD è più veloce! L'installazione da CD risparmia le risorse di connettività di rete.
  • I CD di OpenBSD sono sempre accompagnati da splendidi adesivi. Il tuo sistema non è completo senza di essi. Puoi avere quegli adesivi solo comprando un CD o donando hardware.

Se stati installando un release ufficiale di OpenBSD, dovresti usare un CD.

3.3 - OpenBSD fornisce un'immagine ISO per il download?

Alucni altri sistemi operativi open source sono tipicamente distribuiti come immagini ISO per CD-ROM. OpenBSD non è distribuito così.

Il progetto OpenBSD non rende disponibili per il download le immagini ISO usate per masterizzare i CD ufficiali. Il motivo è semplicemente che ci piacerebbe che compraste il set di CD, contribuendo al continuo sviluppo di OpenBSD. Il layout ufficiale del CD-ROM di OpenBSD è copyright di Theo de Raadt. Theo non permette a nessuno di ridistribuire le immagini dei CD ufficiali di OpenBSD. Come incentivo all'acquisto del set di CD, vengono inclusi alcuni extra (immagini, adesivi, etc.).

Nota che solo il layout del CD è protetto da copyright, OpenBSD stesso è libero. Nulla impedisce a chiunque di scaricare OpenBSD e creare il proprio CD. Se per qualche motivo volete scaricare un'immagine CD, provate a cercare negli archivi della mailing list. Naturalmente, qualunque immagine ISO di OpenBSD disponibile su internet viola il copyright di Theo de Raadt o non è ufficiale. La fonte di un'immagine non ufficiale può essere più o meno affidabile; sta a voi giudicare.

Suggeriamo a chi vuole scaricare gratuitamente OpenBSD, di optare per l'installazione via FTP. Per chi ha bisogno di un CD avviabile per il loro sistema, immagini ISO di dischi di boot (chiamate cd38.iso) sono disponibili per numerose piattaforme e permetteranno di installare il resto del sistema via FTP. Queste immagini ISO sono grandi solo pochi megabyte e contengono solo gli strumenti per l'installazione, non gli effettivi set di file.

3.4 - Scaricare tramite FTP, HTTP o AFS

Ci sono parecchi siti mirror internazionali che danno accesso via HTTP e FTP alle release e agli snapshot di OpenBSD. E' disponibile anche l'accesso via AFS. Dovreste sempre usare il sito più vicino. Prima di cominciare a prendere una release, potreste usare ping(8) e traceroute(8) per determinare quale sito mirror è più vicino e se questo ha performance adeguate. Naturalmente il tuo CD di OpenBSD è sempre più vicino di qualunque mirror. Le informazioni per l'accesso sono qui:

OpenBSD FTP page.

3.5 - Ottenere il codice sorgente corrente

Il codice sorgente di OpenBSD è liberamente disponibile e ridistribuibile senza oneri. Generalmente il modo migliore di iniziare è installare i sorgenti dall'ultimo CD e quindi configurare AnonCVS per aggiornarlo regolarmente. Informazioni sull'AnonCVS, compreso come configurarlo, sono disponibili qui:

OpenBSD AnonCVS page.

Un'altra possibilità ancora è prendere il codice sorgente dal web. Lo potete fare tramite cvsweb presso: http://www.openbsd.org/cgi-bin/cvsweb/.
