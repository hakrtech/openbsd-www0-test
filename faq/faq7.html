7.1 - How do I remap the keyboard?

By using kcon(1), "kcon - keyboard control and remapping for the pcvt driver"

Example: 

    bsd# kcon -m gb

This will load the keycap file for Great Britain.

7.2 - Is there gpm or the like in OpenBSD?

No. But if you are willing to make a port. Please share. :)

7.3 - Clearing the console each time a user logs out.

To do this you must add a line in /etc/gettytab. Change the section currently: 

    P|Pc|Pc console:\
        :np:sp#9600:

Add the line ":cl=\E[H\E[2J:" at the end, so it ends up looking like this: 

    P|Pc|Pc console:\
        :np:sp#9600:
	:cl=\E[H\E[2J:

7.4 - Accessing the Console Scrollback Buffer

For OpenBSD 2.6 the ability to scrollback on the console was added for i386 machines. This allows you to see information that has already scolled past your screen. To be able to move up and down in the buffer simply use the key combination "[shift]+[pgup]" to move up, and "[shift]+[pgdn]". The default scollback buffer (the amount of pages that you can move up and view) is 8, but this can be changed via the scon(1) utility. For example, to change the number of pages you can scrollback to 10, simple do: 

    bsd$ scon -b 10

The smallest amount of pages is 2 and the largest amount is 100. Remember though, that this effects how much kernel memory is used.
