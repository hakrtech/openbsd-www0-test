Voltar para OpenBSD
 

 

Este FAQ é mantido com informações pertencentes a versão 2.7 do OpenBSD. Toda informação apresentada aqui pode não ser exata para antigas versões do OpenBSD. Informações para versões anteriores está disponível. Você deve verificar http://www.openbsd.org/errata.html para atualizações importantes. O FAQ segue as versões lançadas do OpenBSD. Não terá informação para algumas opções -current, tentando somente seguir o que foi lançado no CD. Isto é, assim não há confusão a respeito das versões que estão sendo documentadas aqui.

Este FAQ guiara você nos passos mais críticos para configurar seu próprio sistema OpenBSD. As perguntas variam de usuários novos para usuários avançados. Esperamos que você ache este FAQ útil. As versões para baixar deste FAQ estão disponíveis em texto puro e pdf. Estas versões podem não ser tão atualizadas quanto as versões em HTML disponíveis nesta página.

Todas as perguntas podem ser dirigidas para: faq@openbsd.org. Se você quiser relatar um erro no OpenBSD, leia pôr favor nosso guia para reportar erros detalhadamente.
