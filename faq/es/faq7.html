7.1 - ¿Cómo Puedo Reasignar el Mapa del Teclado? (específico para pcvt/i386)

Usando kcon(1), «kcon - control del teclado y reasignación para el controlador de pcvt». Antes de usarlo debe activar la reasignación del mapa de caracteres con scon.

Ejemplo: 

    bsd# scon -o
bsd# kcon -m es

De este modo cargará el fichero keycap para el teclado español. Tenga en cuenta que si vuelve a ejecutar scon -o, volverá a desactivar la reasignación del mapa de caracteres, lo que probablemente no querrá hacer si está usando kcon para reasignar. Recuerde esto cuando use scon de un modo automatizado, como por ejemplo en guiones de shell ("scripts").

Estas órdenes sólo se pueden ejecutar desde un terminal virtual pcvt.

7.2 - ¿Existe Algo Parecido a gpm en OpenBSD?

Todavía no. Si está considerando crear un porte, contacte con la lista de correo para portes.

7.3 - ¿Cómo Refresco la Consola cada vez que Salga un Usuario?

Para refrescar la consola debe añadir una línea al fichero /etc/gettytab. Cambie la sección actual: 

    P|Pc|Pc console:\
        :np:sp#9600:

Añada la línea ":cl=\E[H\E[2J:" al final, de modo que quede como lo siguiente: 

    P|Pc|Pc console:\
        :np:sp#9600:\
	:cl=\E[H\E[2J:

7.4 - Cómo Acceder a la Memoria Intermedia para Retroceder Páginas en la Consola (específico para pcvt/i386).

En OpenBSD 2.6, la posibilidad de retroceder páginas en consola se incluye sólo en máquinas i386. Esto le permite ver la información que haya pasado ya por su pantalla. Para retroceder y volver a avanzar por las páginas de memoria, use la combinación del teclado "[shift]+[pgup]" para retroceder, y "[shift]+[pgdn]" para volver a avanzar. El número predefinido de páginas de memoria a las que puede retroceder es 8, pero puede cambiarlo mediante la utilidad scon(1). Por ejemplo, para cambiar el número de páginas que puede retroceder a 10, haga lo siguiente: 

    bsd$ scon -b 10

Esta orden sólo se puede ejecutar desde un terminal virtual pcvt.
