7.1 - ¿Cómo puedo reasignar el mapa del teclado? (wscons)

Plataformas que usan el control de consola wscons(4): alpha, i386, y macppc.

Para OpenBSD 3.0 y versiones posteriores:

Con las consolas wscons(4), la mayoría de opciones se pueden controlar usando la utilidad wsconsctl(8). Por ejemplo, para cambiar las asignaciones del teclado con wsconsctl(8) al teclado español, se ejecutaría la siguiente orden: 

    # wsconsctl -w keyboard.encoding=es

En el siguiente ejemplo, reasignaremos la tecla "Bloq Mayús" para que sea "Control L": 

    # wsconsctl -w keyboard.map+="keysym Caps_Lock = Control_L"
Para OpenBSD 2.9:

OpenBSD 2.9 necesitaba unos argumentos para la orden wsconsctl algo distintos; no era necesario anteponer la clave ``keyboard'', sino especificar que la orden se aplicaba al teclado mediante el indicador ``-k''. 

    # wsconsctl -k -w encoding=es

y 

    # wsconsctl -k -w map+="keysym Caps_Lock = Control_L"

De este modo cargará el fichero keycap para el teclado español.

7.2 - ¿Existe algo parecido a gpm en OpenBSD?

Para las plataformas alpha y i386, OpenBSD dispone de wsmoused(8), un porte de moused(8) de FreeBSD. Se puede activar de forma automática para que arranque durante el inicio del sistema editando la línea apropiada en el fichero rc.conf(8).

7.3 - ¿Cómo refresco la consola cada vez que salga un usuario?

Para refrescar la consola debe añadir una línea al fichero /etc/gettytab(5). Cambie la sección actual: 

    P|Pc|Pc console:\
        :np:sp#9600:

añadiendo la línea ":cl=\E[H\E[2J:" al final, de modo que quede como lo siguiente: 

    P|Pc|Pc console:\
        :np:sp#9600:\
	:cl=\E[H\E[2J:

7.4 - Cómo acceder a la memoria intermedia para retroceder páginas en la consola (alpha/macppc/i386).

OpenBSD dispone de una funcionalidad para acceder a la memoria de retroceso de páginas. Esto le permitirá ver información que ya haya pasado por su pantalla. Para poder retroceder y avanzar en la memoria, use las combinaciones del teclado [SHIFT]+[RePág] y [SHIFT]+[AvPág]

La memoria de retroceso predeterminada, o el número de páginas de memoria que puede retroceder para ver, es 8.

7.5 - ¿Cómo cambio entre consolas? (i386)

Los usuarios de sistemas i386 pueden moverse entre cuatro terminales virtuales desde la consola, pulsando [CTRL]+[ALT]+[F1], [CTRL]+[ALT]+[F2], [CTRL]+[ALT]+[F3] y [CTRL]+[ALT]+[F4].

Esto también se puede hacer desde el entorno gráfico X, pasando desde el entorno gráfico de vuelta a un entorno en modo texto. Para volver al entorno gráfico, pulse [CTRL]+[ALT]+[F5].
