3.1 - OpenBSD CD kaufen

Eine OpenBSD CD zu kaufen ist der allgemein beste Weg, mit deiner Unterstützung zu beginnen. Hier ist die Bestellseite: http://www.openbsd.org/orders.html.

Es gibt mehrere gute Gründe, eine OpenBSD CD zu besitzen:

  • CD Verkäufe unterstützen die Weiterentwicklung von OpenBSD.
  • Die Entwicklung eines plattformübergreifenden Betriebssystemes benötigt andauernd Investitionen in Hardware.
  • Deine Unterstützung in Form eines CD Kaufes hat ein realen Einfluß auf zukünftige Entwicklungen.
  • Die CD beinhält Binärdateien (und Quelltexte) für alle unterstüzten Plattformen.
  • Die CD ist bootfähig auf mehreren Plattformen und man kann mittels CD das Betriebssystem direkt installieren.
  • Mit der CD kann man auch die Installation starten, um einen Snapshot zu installieren.
  • Die Installation von CD ist schneller! Und Netzwerkresourcen werden eingespart.
  • Die OpenBSD CD's beinhalten immer sehr nette Aufkleber. Dein System ist nicht komplett, solange du sie nicht hast. Diese Aufkleber kannst du nur durch den Kauf einer CD oder Spenden von Hardware erwerben.

Wenn du eine offizielle Version von OpenBSD installierst, dann solltest du eine CD benutzen. Snapshot Versionen können nur über Netzwerk installiert werden.

3.1.1 - OpenBSD T-Shirts erwerben

Ja, OpenBSD hat nun T-shirts zu deinem Tragevergnügen. Hier kannst du sie sehen: http://www.OpenBSD.org/tshirts.html. Nett, oder? :)

3.1.2 - Wo bekomme ich ein OpenBSD iso image?

Nirgends. Das offizielle OpenBSD CD-ROM Layout unterliegt dem Copyright von Theo de Raadt als Ansporn für die Benutzer, das CD Set zu kaufen. Beachte, daß nur das Layout dem Urheberrechtsschutz unterliegt, OpenBSD selber ist frei. Nichts hindert jemand anderen, OpenBSD runterzuladen und sich seine eigene CD zu kreieren.

3.2 - Downloaden via FTP oder AFS

Es gibt zahlreiche internationale FTP Server, die OpenBSD Versionen und Snapshots führen. AFS Zugang ist auch möglich. Du solltest immer den dir nächsten Server wählen. Bevor du mit dem Runterladen einer Version oder eines Snapshots beginnst, solltest du mittels ping und traceroute feststellen, welcher Server für dich am schnellsten ist. Klarerweise ist das offizielle OpenBSD CD Set immer näher als jeder Server. Zugangsinformationen findest du hier:

http://www.openbsd.org/ftp.html

3.3 - Aktueller Quelltext (CVS)

Die Quelltexte von OpenBSD sind frei erhältlich und verbreitbar. Im Allgemeinen ist der beste Weg, den Verzeichnisbaum mit den aktuellen Quelltexten aufzubauen, die Quelltexte von der letzten CD zu installieren und sie dann mittels AnonCVS regelmäßig zu erneuern. Informationen über AnonCVS findest du hier:

http://www.openbsd.org/anoncvs.html.

sowie auch hier The OpenBSD FAQ.

Sollte deine Netzwerkanbindung für AnonCVS unzureichend sein oder dein Internetzugang besteht via UUCP, kannst du die Quelltexte mittels CTM auf dem laufenden halten. Solltest du dich in dieser Situation befinden, dann ist das Beginnen mit der CD umso wichtiger. Informationen über CTM findest du hier:

http://www.openbsd.org/ctm.html.

Eine weitere Möglichkeit an den Quelltext zu kommen bietet dir das Webinterface "cvsweb" an:

http://www.openbsd.org/cgi-bin/cvsweb/
