Que faut-il faire pour les avoir ?

OpenBSD inclut XFree86 4 avec les "Xft Render Extensions" activées. Pour pouvoir l'utiliser, votre serveur X (pilote de votre carte graphique) doit les supporter. Certains pilotes n'ont pas encore été convertis au code 'fb' et ne peuvent donc pas utiliser les extensions RENDER nécessaires.
Vérifiez comme ceci : 
$ /usr/X11R6/bin/xdpyinfo | grep RENDER
RENDER
Si RENDER n'est pas spécifié dans sa liste des extensions, vous devrez attendre que quelqu'un implémente l'extension pour cette carte.

L'étape suivante est de trouver et d'installer quelques polices "TrueType". L'auteur des "Xft Render Extensions", Keith Packard, a créé un package de polices "TrueType".

J'ai découvert un bel ensemble de polices "TrueType" disponibles gratuitement chez nos amis de Redmond. J'ai créé une archive avec toutes ces polices (elle inclut times, arial, comic, courier, georgia, impact, trebuchet, verdana et webdings) avec les fichiers fonts.scale et XftConfig; vous pouvez les télécharger depuis ...

http://keithp.com/~keithp/truetype.tar.gz

Ou directement (police par police, et sans un fonts.dir depuis) :
http://www.microsoft.com/typography/fontpack/default.htm


Une fois le téléchargement terminé, mettez les polices dans /usr/X11R6/lib/X11/fonts : 
# cd /usr/X11R6/lib/X11/fonts
# tar xzf ~/truetype.tar.gz
Si vous êtes le seul utilisateur sur le système voulant ces polices, créez un fichier $HOME/.xftconfig contenant la ligne suivante : 
	dir "/usr/X11R6/lib/X11/fonts/truetype"
Pour activer les polices pour tout le système, vous devrez éditer /usr/X11R6/lib/X11/XftConfig et ajouter 
	dir "/usr/X11R6/lib/X11/fonts/truetype"
juste après dir "/usr/X11R6/lib/X11/fonts/Type1".

Quelles applications peuvent les utiliser ?

xterm, xditview, ainsi que toutes les applications utilisant QT2 (toutes les applications KDE2 par exemple) reconnaissent Xft.

xterm

Lancez-le comme ceci : blockquote>
$ xterm -fa 'Andale Mono' -fs 14
option:   ressource:   signification
 -fa     *faceName    Nom de la police
 -fs     *faceSize    Taille de la police
Les autres polices "TrueType" (Arial, Georgia, Verdana, ...) ne fonctionnent pas très bien avec xterm. xterm perd un pixel par caractère, car xterm n'est pas disposé à traiter des caractès qui se prolongent au delà de leur cellule.

Les applications QT2/KDE2

Pour lancer une application QT2/KDE2, comme le navigateur konqueror, il n'y a qu'1 (!) seule variable d'environnement devant être définie :
Lancez l'application en définissant la variable QT_XFT comme "true" : 
$ env QT_XFT=true konqueror
ou pour kdbg : 
$ env QT_XFT=true kdbg
et vous avez fini.
