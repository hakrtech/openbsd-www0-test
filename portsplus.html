  • security/gnupg
    GNU privacy guard - a free PGP replacement
    Updated gnupg with an ``idea'' flavor among other changes [reinhard 2001-03-04]

    SGI audiofile library GPLed clone
    Updated libaudiofile to version 0.2.1 [brad 2001-03-03]

    Documentation system for C, C++, IDL and Java
    Updated doc++ to version 3.4.7 [kevlo 2001-03-02]

    Simple, uniform password-checking interface
    Updated checkpassword to version 0.90 [camield 2001-03-02]
  • editors/xemacs21-sumo

    Added xemacs21-sumo-20010204; maintained by Jean-Yves Burlett [naddy 2001-03-01]
  • editors/xemacs21
    Heavily customizable and extensible editor (almost IDE)
    Updated xemacs21 to version 21.1.14 [naddy 2001-03-01]
  • mail/procmail
    Local mail delivery agent
    Updated procmail to version 3.15.1 [brad 2001-03-01]

  • net/bind8
