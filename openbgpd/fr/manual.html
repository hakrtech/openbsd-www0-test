Pages du manuel

Les pages du manuel web sont disponibles sur le site d'OpenBSD. Ces pages du manuel correspondent à la dernière version de développement d'OpenBGPD.

  • bgpd - Service Border Gateway Protocol
  • bgpd.conf - Fichier de configuration du service Border Gateway Protocol
  • bgpctl - Contrôler le service Border Gateway Protocol
Le protocole BGP implémenté dans OpenBGPD est décrit dans la RFC 1771.
