Páginas de Manual

Páginas de Manual estão disponíveis no site do Projeto OpenBSD. Estas páginas de manual refletem a última versão de desenvolvimento do OpenBGPD.

  • bgpd - Border Gateway Protocol (BGP) Daemon
  • bgpd.conf - Border Gateway Protocol Daemon (Arquivo de Configuração)
  • bgpctl - Controle do Daemon do BGP
O protocolo BGP implementado no OpenBGPD está descrito na RFC 1771.
