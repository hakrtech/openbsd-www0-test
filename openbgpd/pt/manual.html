Manual pages

Páginas de Manuais Web estão disponíveis em OpenBSD. Estas páginas de manual refletem a última versão de desenvolvimento do OpenBGPD.

  • bgpd - Border Gateway Protocol daemon
  • bgpd.conf - Border Gateway Protocol daemon arquivo de configuração
  • bgpctl - control the Border Gateway Protocol daemon
O protocolo BGP implementado no OpenBGPD é descrito em RFC 1771.
