Pagine di Manuale

Le pagine di manuale via web sono disponibili da OpenBSD. Corrispondono all'ultima versione di sviluppo di OpenBGPD.

  • bgpd - Border Gateway Protocol daemon
  • bgpd.conf - File di configurazione del Border Gateway Protocol daemon
  • bgpctl - Controlla il Border Gateway Protocol daemon
Il protocollo BGP implementato in OpenBGPD è descritto in RFC 1771.
