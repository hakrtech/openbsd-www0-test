Handleidingen

Web handleidingen zijn beschikbaar via OpenBSD. Deze handleidingen reflecteren de nieuwste ontwikkelingsversie van OpenBGPD.

  • bgpd - Border Gateway Protocol daemon
  • bgpd.conf - Border Gateway Protocol daemon configuratiebestand
  • bgpctl - beheer de Border Gateway Protocol daemon
Het BGP protocol geïmplementeerd in OpenBGPD wordt beschreven in RFC 1771.
