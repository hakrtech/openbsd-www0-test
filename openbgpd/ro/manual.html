Pagini de manual

Paginile manualului Web sunt disponibile de la OpenBSD. Aceste pagini de manual reflectă ultima versiune de dezvoltare a OpenBGPD.

  • bgpd - Demonul Border Gateway Protocol
  • bgpd.conf - Fişierul de configurare a demonului Border Gateway Protocol
  • bgpctl - controleaza demonul Border Gateway Protocol
Implementarea protocolului BGP în OpenBGPD este descrisă în RFC 1771.
