Priručnik / manual stranice

Manual stranice se mogu i preko WWW-a naći, i to kod glavne stranice OpenBSD-projekta. Ove manual stranice se odnose na zadnji status u razvoju OpenBGPD-a.

  • bgpd - Border Gateway Protocol daemon
  • bgpd.conf - Border Gateway Protocol daemon konfiguracijska datoteka
  • bgpctl - upravljanje Border Gateway Protocol daemon-om
Opis BGP-protokola, koji je implementiran u OpenBGPD-u, se nalazi u RFC-u 1771.
