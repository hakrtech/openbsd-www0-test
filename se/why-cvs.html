Varför OpenBSD utvecklare använder CVS

Concurrent Version System (CVS) underhållen av Cyclic Software valdes till att användas av OpenBSD projektgruppen. Med CVS, sjuttiotalet eller liknande antal många utvecklare kan uppdatera källkoden (vilket är över 1349MB, omkring 75 000 filer) som utgör operativsystemet, ports/paket och XFree86. En stor andel av OpenBSD användarna kan ladda ned operativsystem källkoden via anonym CVS, en procedur som är mycket mer pålitlig än Software Upgrade Protocol (sup).

Eftersom CVS samspelar med Secure Shell (ssh), kan det erbjuda pålitlig och säker användar- och värdverifiering. Detta platsar bra in med OpenBSD projektets mål.
