Number 4: The Blowfish shirt
Front

Front of shirt contains a picture of the Blowfish, and a vague reference to how using the blowfish cipher for the OpenBSD password hash makes our password mechanism more secure (blowfish, not babblefish). People won't be able to crack your passwords anymore.. Back of shirt contains the complete source code for our version of the Blowfish algorithm (written in Germany). For more information see our cryptography page...

This shirt is made of white material.
These shirts can be ordered on the web secure order or insecure order.
