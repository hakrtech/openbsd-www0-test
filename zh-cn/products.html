OpenBSD 的商业产品

OpenBSD 鼓励企业以及个人使用OpenBSD或者基于OpenBSD开发新产品。

OpenBSD 的商业支持可以到 我们的支持网页上寻找相关的厂商。

  • 由 OpenBSD 发展而来的产品
  • OpenBSD 支持的硬/软件产品
  • 主机预装 OpenBSD 

    • 基于OpenBSD的产品

  • ePipe by Stallion Technologies
    Stallion Technologies 在他们专为中小型企业设计的 ePipe 拨号网关上使用 OpenBSD 作为嵌入式系统 。 这个网关装置主要为了降低小型商业公司 最后 一公里 使用的VPN技术的高昂费用及 应用程序厂商发 送程序给小型商业公司所需要的昂贵费用。 ePipe 采用 Motorola ColdFire 5307 VL-RISC 处理器，并使用 OpenBSD 内建的加密软件来提供端对端的安全连接。
  • nCircle Network Security
    nCircle Network Security (前身是 Hiverworld)实现了全面的网络安全防护.通过智能化的入侵防护,提供了领先的入侵检测技术。他所采用的 IP360 专利技术结合 全速的网络流量监控，入侵分析等技术,实现持续性非入侵式弱点评估。nCircle 选择 OpenBSD 操作系统作为他的 扫描传感器、入侵感应器及控制台引擎的安全网络平台。
  • RTMX
    RTMX 销售的是增加了支持 POSIX 实时特征 的OpenBSD 版本。他们也很慷慨的提供了这些扩展功能的源码，这些新的功能 将会很快的集成到 OpenBSD 中。
  • syscall Network Solutions
    Syscall Network Solutions 提供基于OpenBSD的syswall安全装置. syswall 安全平台包含了一个状态型包过滤防火墙,一个入侵检测系统和能在动态IP下工作的VPN 功能. 这是我们IT安全专家为了管理安全服务和管理安全监测而特别设计的系统. 因为它支持PPPoE和DHCP,所以也可以将syswall用作 DSL 或 cable 路由器.

    • 用于OpenBSD的硬/软件产品

  • PowerCrypt by GTGI
    Global Technologies Group, Inc. 生产的 PowerCrypt, 是一个智能的， 总线控制 PCI 板，其中整合了 Hifn 7751 加密处理器。系统内核级支持 是由 OpenBSD 所开发， 是由 OpenBSD 社群成员提供的驱动程序以及 IPsec代码。
  • RedHat's Stronghold Secure Web Server
    RedHat 提供 128-bit强加密的 HTTPS 守护进程，主要是基于获得广泛赞誉的 OpenBSD 版 Apache Web服务器，并配合使用 赢得许多奖项的 Stronghold 3 安全web服务器。
  • Open Sound System by 4Front (只支持 i386 平台)
    4Front Technologies 提供一个更好的商业 版本的 OpenBSD/i386 的声卡驱动程序。这些驱动程序与 OpenBSD/i386 自带的声卡驱动兼容，但它支持更多其它声卡，并且可以有效利用声卡的 各种功能。 兼容OpenBSD/i386 3.x 的软件包可以从这里获得
  • Core Systems
    Core Systems 出售的 InSite, 是一个OpenBSD/i386下的使用简单的服务器端web统计工具.InSite 是与 WebTrends 相似的产品，但它可以配置为使用web接口，快速产生报表，并只占据极少的 CPU 时间。(如果有特别需求的话，核心团队成员可以 提供在 i386 平台以外的 InSite 版本)。
  • Software2Go Motif (只支持 i386 及 SPARC 平台)
    Software2Go, LLC 有 供 OpenBSD 使用的 Motif 2.1.20 Development 及 Runtime toolkits.
    这个研发中的版本，可供开发图形化用户界面应用程序的软件开发者，开发出 Motif 模式的图形界面外观及使用方式。 其中 runtime 版本在二进制可执行程序中，是必备的。
    联络: Software2Go, LLC 在线订购， 817-431-8775 (电话/传真)
  • Zeus Web Server (只支持 i386 平台)
    Zeus Technology 研发的 Zeus Web Server, 是世界上可扩充性最强的高效能web服务器软件，目前支持 OpenBSD/i386 2.8 及 2.9 版本。其中一小部分的优异功能是: 非常好的扩充性，即时的web统计数据，单个服务器的带宽控制，安全的 CGI 执行环境，支持集群，以及对 FastCGI 的支持。
