OpenBSD 的商业产品

OpenBSD 鼓励公司以及个别发展者将 OpenBSD 做为他们的产品或是使用 OpenBSD 来 创作出新的产品。

OpenBSD 的商业支援可以到 我们的支援网页上寻找相关的厂商。

  • 由 OpenBSD 发展而来的产品
  • OpenBSD 支援的硬/软体产品
  • 预先安装 OpenBSD 的电脑主机

    • 由 OpenBSD 发展而来的产品

  • ePipe by Stallion Technologies
    Stallion Technologies 在一个中小企业拨接网路闸道 ePipe 上搭载 OpenBSD 的嵌入式作业系统 。 这个闸道装置主要是要取代小型商业公司高价的 VPNs 最末端网段及 应用程式厂商提供给小型商业公司的服务程式。 ePipe 采用 Motorola ColdFire 5307 VL-RISC 处理器，并使用 OpenBSD 内建的加密 软体提供点对点的安全连线。

  • nCircle Network Security
    nCircle Network Security (前身是 Hiverworld) 超越了入侵侦测技术，进而提供智慧型入侵防范达到全面的网路 安全防护。他所采用的 IP360 专利技术提供了持续、无侵略性的弱点稽核，并结合了 全速的网路流量监控，入侵分析等技术。nCircle 以 OpenBSD 作业系统做为他的 扫描侦测、入侵侦测及控制端安全网路平台。

  • RTMX
    RTMX 贩售的是有支援 POSIX 即时作业系统 的 OpenBSD 版本。他们也很善意的提供了相关支援的原始程式码，这些新的功能 将会很快的整合到 OpenBSD 中。

  • syscall Network Solutions
    Syscall Network Solutions 使用OpenBSD制作syswall安全保护平台. syswall 安全平台包含了强大的小包过滤防火墙, 入侵侦测系统和VPN 功能(即便是动态IP). 这是我们IT安全专家特定的设计, 用于管理安全服务和管理安全监测系统. 也可以用syswall作为 DSL 和 cable 路游器, 因为它提供PPPoE和DHCP.

    • OpenBSD 支援的硬/软体产品

  • PowerCrypt by GTGI
    Global Technologies Group, Inc. 研发的 PowerCrypt, 是一个智慧型， bus-mastering PCI 板，其中整合了 Hifn 7751 加密处理器。系统核心层的支援 是由 OpenBSD 所研发， 由 OpenBSD 社群成员撰写的驱动程式及 IPsec 原始程式码所撰写而成。

  • RedHat's Stronghold Secure Web Server
    RedHat 提供 128-bit 的加强密码安全 HTTPS 伺服器程式，主要是基于被广泛使用的 OpenBSD 版 Apache 网页伺服器，并配合使用 赢得许多奖项的 Stronghold 3 安全网页伺服器。

  • Open Sound System by 4Front (只支援 i386 平台)
    4Front Technologies 提供一个更好的商业 版本板 OpenBSD/i386 平台的音效卡驱动程式。这些驱动程式相容於 OpenBSD/i386 平台 所搭载的音效卡驱动程式，并且支援许多其他的音效卡，并且较能够善用个别音效卡的 各种功能。 这个软体相容於 OpenBSD/i386 3.0 版

  • Core Systems
    Core Systems 贩售的 InSite, 是一个容易使用，伺服器端的网页统计工具，支援 OpenBSD/i386 平台。InSite 是与 WebTrends 相似的产品，但他也提供了可藉由网页修改调校的介面，可以飞快 地产生报表，并只占据极少的 CPU 时间。(如果有特别需求的话，核心团队成员可以 提供在 i386 平台以外的 InSite 版本)。

  • Software2Go Motif (只支援 i386 及 SPARC 平台)
    Software2Go, LLC 有 供 OpenBSD 使用的 Motif 2.1.20 Development 及 Runtime toolkits.
    这个研发中的版本，可供建构图形使用者介面应用程式的软体发展者，发展出 Motif 模式的图形介面观感及使用方式。 其中 runtime 版本在只有可执行档的应用程式中，是必备的。
    联络: Software2Go, LLC 线上订购， 817-431-8775 (电话/传真)

  • Zeus Web Server (只支援 i386 平台)
    Zeus Technology 研发的 Zeus Web Server, 是世界上可扩充性最强的高效能网页伺服器软体，目前支援 OpenBSD/i386 2.8 及 2.9 版本。其中一小部分的优异功能是: 非常好的扩充性，即时的网页介面统计资料，个别 伺服器的频宽调节，安全的 CGI 执行环境，支援丛集伺服器群，以及 FastCGI 的支援。
