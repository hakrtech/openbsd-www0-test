OpenBSD/iris

History and Status:

There is currently no iris port. There is however lots of R3000/R4000 code, so it should be fairly easy to do. As well, there is a Linux port in progress, which will likely later provide information for people doing a port.

Contact Theo de Raadt if you are interested in doing this and need more information.
