Listes de diffusion

Liste privée des développeurs

Les problèmes ayant trait à la sécurité peuvent être rapportés sur la liste privée security@opensmtpd.org qui est lue uniquement par les développeurs d'OpenSMTPD. Elle n'est pas appropriée pour les questions d'ordre général.

Les autres problèmes peuvent être rapportés à la liste privée bugs@opensmtpd.org qui est lue, elle aussi, uniquement par les développeurs d'OpenSMTPD. Elle n'est pas appropriée pour les questions d'ordre général.

Liste de discussions générales à propos d'OpenSMTPD

Les discussions d'ordre général, les problèmes et suggestions d'amélioration peuvent être discutés sur la liste misc@opensmtpd.org. Cette liste n'est pas modérée, mais l'inscription est requise.

Pour vous inscrire, envoyez simplement un mail ayant pour sujet [misc] register à misc@opensmtpd.org.
Pour vous désinscrire, envoyez un mail ayant pour sujet [misc] unregister.

Pour l'instant, la liste a un faible trafic, n'hésitez pas à vous présenter ;-)
