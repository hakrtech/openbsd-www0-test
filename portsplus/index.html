  • x11/gentoo
    GTK+ Graphical filemanager
    Updated gentoo to version 0.11.33 [pvalchev 2002-09-03]

    Create barcode image with GD
    Added p5-GD-Barcode-1.13; maintained by Jim Geovedi [brad 2002-09-03]

    Generate a cross-referenced HTML mail archive
    Updated hypermail to version 2.1.5 [brad 2002-09-03]

    Updated fluxbox to version 0.1.11 [brad 2002-09-03]

    Simple tcp re-engineering tool
    Added tcpreen-1.0.2; maintained by Margarida Sequeira [lebel 2002-09-03]

    SDL C++ GUI widget library
    Updated guilib to version 1.1.1 [mark 2002-09-02]

    XMMS input plugin for playing SID files
    Updated xmms-sid to version 0.7.4 [naddy 2002-09-02]
  • mail/p5-Mail-SpamAssassin
    Mailfilter to identify and mark spam
    Updated p5-Mail-SpamAssassin to version 2.40 [lebel 2002-09-02]
  • security/keychain
    Front-end to ssh-agent
    Updated keychain to version 2.0.2 [naddy 2002-09-01]
  • print/foiltex

  • security/p5-IO-Socket-SSL

  • x11/waimea

    Added waimea-0.3.3; maintained by Damien Couderc [couderc 2002-09-01]
