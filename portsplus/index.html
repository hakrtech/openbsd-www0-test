Changes made to the ports collection in 2004

This page lists changes to the OpenBSD ports collection made in 2004.

For ports changes in 2003, click here.

January 2004

  • audio/id3lib
    Library for manipulating ID3v1 and ID3v2 tags
    Updated id3lib to version 3.8.3 [espie 2004-01-05]
