August 2002

  • misc/p5-Locale-Codes
    Module to provide access to ISO3166 and ISO639 codes
    Updated p5-Locale-Codes to version 2.06 [kevlo 2002-08-05]
  • math/sc
    Curses spreadsheet-calculator program
    Updated sc to version 7.15 [pvalchev 2002-08-05]
  • graphics/gwenview
    Image viewer for KDE
    Added gwenview-0.14.4; maintained by Brad Smith [brad 2002-08-04]
  • textproc/expat
    XML 1.0 parser written in C
    Updated expat to version 1.95.4 [brad 2002-08-04]
  • audio/libsidplay
    C64 music player and SID chip emulator library
    Updated libsidplay to version 1.36.55 [naddy 2002-08-04]
  • devel/p5-Expect
    Talk to other applications
    Added p5-Expect-1.15; maintained by Michael Lestinsky [naddy 2002-08-04]
  • devel/p5-IO-Tty
    Provide an interface to create pseudo ttys
    Added p5-IO-Tty-1.02; maintained by Michael Lestinsky [naddy 2002-08-04]
  • net/dhcping
    Dhcp daemon ping program
    Added dhcping-1.2; maintained by Margarida Sequeira [pvalchev 2002-08-04]
  • print/acroread
    View, distribute and print PDF documents
    SECURITY FIX: Updated acroread to version 5.06 [naddy 2002-08-03]
  • net/gtk-gnutella
    Gtk-based GUI client for the Gnutella Network
    Updated gtk-gnutella to version 0.90 [naddy 2002-08-03]
  • net/openh323
    H.323 Video Conferencing library
    Marked openh323 broken [naddy 2002-08-02]
  • www/p5-HTTPD-Log-Filter
    Httpd log filter library
    Added p5-HTTPD-Log-Filter-1.07; unmaintained [jakob 2002-08-02]
  • archivers/p5-IO-Zlib
    IO:: style interface to Compress::Zlib
    Added p5-IO-Zlib-1.01; maintained by Jakob Schlyter [jakob 2002-08-02]
  • editors/beav
    Binary editor and viewer
    Updated beav to version 1.40.15 [kevlo 2002-08-01]
  • math/calc
    C-style arbitrary precision calculator
    Updated calc to version 2.11.5.8 [wcobb 2002-08-01]
