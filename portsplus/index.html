December 2001

  • misc/xnc
    Filemanager for X Window
    Updated xnc to version 4.4.6 [reinhard 2001-12-06]
  • education/keduca
    Tool to allow the creation and revision of form based test
    Added keduca-0.4; maintained by Kevin Lo [kevlo 2001-12-06]
  • net/fping
    Quickly ping N hosts w/o flooding the network
    Updated fping to version 2.2b1 [form 2001-12-05]
  • net/vicq
    Text-based ICQ implementation
    Added vicq-0.3; maintained by Kevin Lo [kevlo 2001-12-05]
  • www/linkchecker
    Checks HTML documents for broken links
    Added linkchecker-1.3.11; maintained by Jakob Schlyter [jakob 2001-12-05]
  • net/curl
    Get files from FTP, Gopher, HTTP or HTTPS servers
    Updated curl to version 7.9.2 [brad 2001-12-05]
  • mail/mboxgrep
    Scan mailboxes for messages matching a regular expression
    Added mboxgrep-0.7.4; maintained by Kevin Lo [kevlo 2001-12-05]
  • net/p5-Net-ICQ2000
    Module for ICQv7 protocol realisation
    Added p5-Net-ICQ2000-0.2.1; maintained by Kevin Lo [kevlo 2001-12-05]
  • games/taxipilot
    Silly little game based on Spacetaxi for the C64
    Added taxipilot-0.6.0; maintained by Kevin Lo [kevlo 2001-12-05]
  • textproc/p5-Text-Unidecode
    US-ASCII transliterations of Unicode text
    Added p5-Text-Unidecode-0.04; maintained by Kevin Lo [kevlo 2001-12-05]
  • security/scanssh
    SSH remote version scanner
    Updated scanssh to version 1.6b [brad 2001-12-04]
  • math/calcoo
    Gtk-based scientific calculator
    Updated calcoo to version 1.3.2 [naddy 2001-12-04]
  • security/logcheck
    Logfile auditing tool
    Added logcheck-1.1.1; maintained by Srebrenko Sehic [jsyn 2001-12-03]
  • graphics/libmng
    Multiple-image Network Graphics (MNG) reference library
    Updated libmng to version 1.0.3 [brad 2001-12-03]
  • graphics/lcms
    Color management library
    Added lcms-1.08; maintained by Brad Smith [brad 2001-12-03]
  • net/centericq
    Curses-based icq client implementation
    Updated centericq to version 4.2.1 [pvalchev 2001-12-02]
  • math/grace
    GRaphing, Advanced Computation and Exploration of data
    Updated grace to version 5.1.5 [pvalchev 2001-12-02]
  • mail/mailman
    Mailing list manager with web interface
    SECURITY FIX: Updated mailman to version 2.0.8 [pvalchev 2001-12-02]
  • games/slash-em
    Dungeon explorin', hackin', game. Hard
    Updated slash-em to version 6e4f6 [espie 2001-12-01]
  • mail/pine
    Program for Internet E-mail and News, with pico
    Updated pine to version 4.43 [brad 2001-12-01]
  • mail/c-client
    University of Washington's c-client mail access routines
    Updated c-client to version 4.43 [brad 2001-12-01]
  • education/ktouch
    Interactive typing tutor
    Added ktouch-1.0; maintained by Kevin Lo [kevlo 2001-12-01]
  • textproc/p5-Text-PDF
    Perl modules for creating and manipulating PDF
    Updated p5-Text-PDF to version 0.18 [shel 2001-12-01]
  • education/klettres
    Interactive learning the French alphabet
    Added klettres-1.0; maintained by Kevin Lo [kevlo 2001-12-01]
