July 2001

  • net/dlint
    DNS zone verification tool
    Added dlint-1.4.0; maintained by Jason Peel [lebel 2001-07-02]
  • audio/nap
    GNU curses-based napster client
    Updated nap to version 1.4.5 [pvalchev 2001-07-01]
  • math/fftw
    C routines for computing the Discrete Fourier Transform
    Added fftw-2.1.3; maintained by Nikolay Sturm [lebel 2001-07-01]
  • editors/nano
    Small, easy to use editor
    Updated nano to version 1.0.3 [naddy 2001-07-01]
  • shells/zsh/stable
    The Z shell, Bourne shell-compatible, release flavor
    Updated zsh-stable to version 4.0.2 [lebel 2001-07-01]
  • x11/p5-Gtk
    Perl modules for Gtk libraries
    Added p5-Gtk-0.7008; maintained by Shell Hung [shell 2001-07-01]
  • textproc/py-Chart
    Python library for creating postscript or PDF charts
    Added py-Chart-1.10; maintained by Shell Hung [shell 2001-07-01]
  • textproc/p5-XML-LibXML
    Perl interface to the libxml2 library
    Updated p5-XML-LibXML to version 0.97 [shell 2001-07-01]
  • textproc/libxslt
    The XSLT C Library for GNOME
    Updated libxslt to version 0.13.0 [shell 2001-07-01]
