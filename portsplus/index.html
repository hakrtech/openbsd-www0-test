  • NOTE: The ports tree is currently frozen (for the most part) pending the 3.0 release.
  • net/xchat
    X-Chat is an X11 IRC client
    Updated xchat to version 1.8.4 [shell 2001-10-07]
  • net/ethereal
    Network protocol analyzer
    Updated ethereal with a ``no_x11' flavor [jakob 2001-10-07]
  • devel/cscope
    Code browsing program
    Updated cscope to version 15.3 [naddy 2001-10-07]
  • net/bind8
    Berkeley Internet Name Daemon
    Updated bind8 to version 8.2.5 [ho 2001-10-06]

    Gtk-based AOL Instant Messenger (AIM) client
    Updated gaim to version 0.45 [jcs 2001-10-06]

    Added p5-Chatbot-Eliza-0.97; maintained by Kevin Lo [kevlo 2001-10-05]

    Updated zangband to version 2.6.2 [lebel 2001-10-05]

    Updated p5-Graphics-ColorNames to version 0.30 [shell 2001-10-05]

    Updated lclint to version 2.5r [espie 2001-10-04]
