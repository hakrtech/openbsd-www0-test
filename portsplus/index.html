  • www/nscache
    Netscape cache browser
    Updated nscache to version 0.6 [dugsong 2001-06-02]
  • databases/mysql
    Multithreaded SQL database (client)
    Updated mysql to version 3.23.38; added a client sub-package [brad 2001-06-02]
  • net/nmap
    Port scanning large networks
    Updated nmap to version 2.54BETA24 [bmc 2001-06-02]
  • converters/rpm2cpio
    A simple perl converter
    Updated rpm2cpio to version 1.1 [espie 2001-06-02]
  • graphics/dia
    Visio-like technical drawing
    Updated dia to version 0.88.1 [obecian 2001-06-02]
  • net/nemesis
    Command line arbitrary packet builder
    Updated nemesis to version 1.3 [obecian 2001-06-01]
  • net/mrtg
    Multi-router traffic grapher
    Updated mrtg to version 2.9.12a [brad 2001-06-01]
  • net/vnc
    Cross-platform remote desktop access
    Updated vnc to include a ``viewer'' sub-package [matt 2001-06-01]
  • net/arpwatch
    Monitor arp & rarp requests
    Updated arpwatch to version 2.1a11 [jakob 2001-06-01]
  • devel/rats
    C source code auditing tool
    Added rats-0.9; maintained by Jason Peel [camield 2001-06-01]
  • www/august
    HTML editor designed for the experienced web designer
    Updated august to version 0.63b [kevlo 2001-06-01]
  • devel/flawfinder
    C/C++ source code auditing tool
    Added flawfinder-0.12; maintained by Jason Peel [shell 2001-06-01]
  • databases/p5-DBI
    Unified perl interface for database access
    Updated p5-DBI to version 1.16 [shell 2001-06-01]
  • textproc/p5-XML-LibXSLT
    Fast XSLT module for perl
    Updated p5-XML-LibXSLT to version 1.03 [shell 2001-06-01]
