November 2001

  • graphics/libungif
    Tools and library routines for working with GIF images
    Added libungif-4.1.0b1; maintained by Brad Smith [brad 2001-11-02]
  • x11/ogle
    DVD player
    Updated ogle to version 3.1 [espie 2001-11-02]
  • converters/libdvdcss
    Descramble DVDs
    Added libdvdcss-0.0.3.ogle3; unmaintained [espie 2001-11-02]
  • lang/jikes
    Compile Java source into .class files (quickly!)
    Updated jikes to version 1.15 [kevlo 2001-11-02]
  • www/zope
    Object-oriented web application server
    Updated zope to version 2.4.2 [matt 2001-11-01]
  • net/gaim
    Gtk-based AOL Instant Messenger (AIM) client
    Updated gaim to version 0.47 [jcs 2001-11-01]
  • editors/vim/stable
    vi clone, many additional features
    Updated vim-stable to version 6.0.64 [lebel 2001-11-01]
  • x11/rxvt
    Color, low memory usage, xterm replacement
    Reverted rxvt to version 2.6.3 [jasoni 2001-11-01]
  • x11/gnomemm
    C++ interface for the GNOME libraries
    Added gnomemm-1.2.1; maintained by Peter Stromberg [wilfried 2001-11-01]
  • x11/gtkmm
    C++ interface for gtk+
    Updated gtkmm to version 1.2.7 [wilfried 2001-11-01]
  • textproc/libxslt
    XSLT C Library for GNOME
    Updated libxslt to version 1.0.6 [shell 2001-11-01]
