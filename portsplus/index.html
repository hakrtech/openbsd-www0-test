Changes made to the ports collection in 2001

This page lists changes to the OpenBSD ports collection made in 2001. It is maintained by Joshua Stein.

April 2001

  • net/trafshow
    Full screen visualization of network traffic
    Updated trafshow to include a KAME IPv6 patch [brad 2001-04-02]
  • editors/vim/snapshot
    vi clone, many additional features
    Added vim-6.0z; maintained by David Lebel [espie 2001-04-02]
  • databases/p5-DBI
    Unified perl interface for database access
    Updated p5-DBI to version 1.15 [avsm 2001-04-02]
  • editors/vim
    Removed vim port; split into vim/stable and vim/snapshot [espie 2001-04-02]
  • editors/vim/stable
    vi clone, many additional features
    Added vim-5.7.25; maintained by Jason Ish [espie 2001-04-02]
  • databases/mysqlgui
    Graphical sql client for MySQL
    Added mysqlgui-1.7.4; maintained by Peter Stromberg [wilfried 2001-04-02]
  • devel/mysql++
    C++ API for MySQL
    Added mysql++-1.7.8; maintained by Peter Stromberg [wilfried 2001-04-02]
  • x11/fleditor
    Text Editor for FLTK
    Added fleditor-0.4; maintained by Peter Stromberg [wilfried 2001-04-02]
  • x11/flvw
    FLTK Virtual Styles, List, Table, and Complex Table Widgets
    Added flvw-1.0; maintained by Peter Stromberg [wilfried 2001-04-02]
  • graphics/netpbm
    Toolkit for converting images between different formats
    Updated netpbm to version 2.37.5 [brad 2001-04-01]
  • x11/wmmenu
    wm-dockapp; A popup menu of icons that launch programs
    Added wmmenu-0.5; maintained by Peter Stromberg [wilfried 2001-04-01]
