Changes made to the ports collection in 2006

This page lists changes to the OpenBSD ports collection made in 2006.

For ports changes in May 2006, click here.

June 2006

  • x11/kde
    K Desktop Environment
    Updated KDE to version 3.5.3 [espie 2006-06-03]
  • lang/python
    Interpreted object-oriented programming language
    Switched default Python version to 2.4, and bumped the PKGNAMES of all the depending ports (with steven) [alek 2006-06-03]
  • www/trac
    Enhanced wiki and bug tracking system for software projects
    Added NO_REGRESS [alek 2006-06-03]
  • devel/p5-IO-KQueue
    Perl interface to the BSD kqueue system call
    Updated p5-IO-KQueue to version 0.30 [steven 2006-06-03]
  • mail/p5-MIME-tools
    Modules for parsing (and creating) MIME entities
    Updated p5-MIME-tools to version 5.420 [steven 2006-06-03]
  • devel/p5-Graph
    Graph data structures in perl
    Updated p5-Graph to version 0.74 [steven 2006-06-03]
  • net/gajim
    Jabber client written in pygtk
    Removed outdated patches [jolan 2006-06-02]
  • databases/mysql
    Multithreaded SQL database (client)
    SECURITY FIX: Updated mysql to version 5.0.22 [bernd 2006-06-02]
  • graphics/win32-codecs
    Huge compilation of Win32 binary codecs
    Updated win32-codecs to version 20060501 [bernd 2006-06-02]
  • mail/mini_sendmail
    Accept email on behalf of real sendmail
    Updated mini_sendmail to version 1.3.6 [aanriot 2006-06-01]
  • comms/gnokii
    Tools to talk to GSM cellular phones
    Added the manual pages and MESSAGE and disabled debug code [robert 2006-06-01]
  • lang/ucblogo
    Berkeley's implementation of the logo programming language
    Corrected distfile [jolan 2006-06-01]
  • net/freeradius
    RADIUS server implementation
    Fixed ownership of log directory [steven 2006-06-01]
  • textproc/intltool
    Internationalization tool
    Updated intltool to version 0.35.0 [steven 2006-06-01]
  • archivers/libmspack
    Library for handling microsoft compression formats
    Updated libmspack to version a [jolan 2006-06-01]
  • games/openttd
    Open source clone of the game Transport Tycoon Deluxe
    Added openttd version 0.4.7; maintained by Michael Knudsen [jolan 2006-06-01]
  • net/p5-Net-IP
    Perl module for IPv4 address parsing
    Updated p5-Net-IP to version 1.25 [steven 2006-06-01]
