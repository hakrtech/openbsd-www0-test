OpenBSD/hp300 FAQ

  • How do I change the hp400 boot ROM mode from Domain/OS to HP-UX?
    Remove the Domain keyboard, replacing it with a HIL model. Press the service switch on the front panel; the LED will light, then press the reset switch on the back of the machine. The system should then boot into MD mode, and display a > prompt. If the prompt does not display, try pressing return a few times; when the prompt appears, type hp and press return. Your system should then reset and display the HP-UX style ROM messages.

  • How do I set make the serial port be the console on a 300-series machine?
    For anything before the hp380, you need to set the remote bit on the motherboard or serial card; settings are different for every model and beyond the scope of this document. For the hp380 and above (including the 400 series), simply press the space bar during the ROM configuration so that the boot menu appears; at the prompt type C and press return. The configuration menu will appear, from where you can put the serial port into remote mode.
