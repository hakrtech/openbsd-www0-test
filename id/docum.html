Pohon Info GNU

  • Info - Sistem jelajah dokumentasi.
  • Bison - Yet Another Compiler.
  • CPP - Preprosesor C GNU.
  • Emacs - Penyunting teks dokumentasi-otomatis yang bisa.
  • Forms mode - Penyunting dengan mode berbasis formulir.
  • G++ & GCC - Kompiler C++ dan C GNU.
  • G77 - Kompiler Fortran GNU.
  • GDB - Debugger tataran-kode C.
  • Gawk - Implementasi GNU dari AWK.
  • Gnus - Pembaca/Pengirim/Caraka news GNUS.
  • Gprof - Pemrofile GNU.
  • libstdc++ - Pustaka C++ GNU standar.
  • Make - Implementasi GNU pada Make.
  • rx & regex - Pustaka ekspresi reguler.
  • Tar - Tar GNU.
  • Termcap - Pustaka termcap, yang membuat aplikasi bisa mengelola semua tipe terminal bertampilan karakter.
  • Texinfo - Dengan satu berkas sumber, bisa membuat sebuah manual tercetak (dengan TeX) ataupun sebuah berkas Info (dengan texinfo).
  • VIP - Modus emulasi Vi untuk Emacs.
  • Viper - Emulator terbaru Vi untuk Emacs.
  • VM - Sebuah antarmuka mail yang baru (Elm dan fodler mailx-selaras).
  • Calc - Calc Emacs GNU -- sebuah kalkulator simbolis dan numerik berbasis RPN.
  • Elisp - Bagaimana memperluas penggunaan Emacs.
  • Babyl - Bentuk (BABYL) dari ~/RMAIL, digunakan oleh rmail emacs .
  • Perl - Dokumentasi Perl.
  • GNATS - Sistem manajemen pelaporan masalah GNU.
  • groff - Implementasi GNU untuk troff.
  • screen - Pengelola terminal maya.

Sistem operasi sepadan Unix lainnya

  • NetBSD - SO 4.4BSD multiplatform
  • FreeBSD - SO 4.4BSD mutakhir untuk i386 dan ke atas
  • Lites - server berbasis 4.4BSD dan pustaka emulasi pada sistem berbasis Mach
  • Sistem Operasi Exokernel MIT - Menggunakan utilitas dan pustaka standar OpenBSD, sebagaimana implementasi sebuah antarmuka pada system calls OpenBSD.
  • Flux - Proyek Mach baru di Utah
  • Linux - SO Unix-like Linus Torvald untuk i386 dan ke atas
  • GNU - SO Free Software Foundation berbasis pada kernel Mach/Hurd
  • Minix - Kloning Unix V7 Andy Tanenbaum
  • Amoeba - SO terdistribusi Andy Tanenbaum
  • Xinu - Sistem Operasi Xinu
  • Sprite - Sistem Operasi Sprite
