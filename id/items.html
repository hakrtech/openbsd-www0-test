OpenBSD 3.4: 1 November 2003


Untuk memesan CDROM 3.4, klik di sini.
  • Tiga CDROM dalam sebuah tempat eksklusif.
  • Komponen instalasi lengkap untuk LIMA arsitektur: i386, macppc, vax, sparc, sparc64.
  • Arsitektur berikut hanya tersedia lewat download FTP: alpha, hppa, hp300, mvme68k, mac68k.
  • CD bisa di boot pada i386, macppc, sparc, dan sparc64.
  • Sebuah artistik kertas tersertakan dalam CD yang bergaya dan penuh kejutan yang berisi petunjuk istalasi. Keterangan pada selembar kertas membuat OpenBSD terasa lebih mudah diinstall daripada jika anda melakukan instalasi lewat FTP.
  • Struktur kode sumber lengkap (siap untuk penggunaan AnonCVS).
  • Binari XFree86 terakhir paling reliable untuk semua arsitektur.
  • Kode sumber XFree86 terakhir dengan modifikasi kecil oleh kami untuk membuatnya lebih manis dan lebih aman.
  • Struktur ports milik kami sendiri yang telah meningkat dalam jumlah yang besar sejak OpenBSD 3.3. Hampir semua paket bisa berjalan pada hampir semua arsitektur.
  • Beberapa paket binari pre-built untuk arsitektur yang paling umum dipakai, yang sangat mudah diinstall secara langsung dari CDROM.
  • Seperti biasa, termasuk juga stiker!
  • Dan masih banyak lagi...

OpenBSD 3.3: 1 Mei 2003


Untuk memesan sebuah CDROM 3.3, klik di sini.
  • Tiga CDROM dalam sebuah tempat eksklusif.
  • Komponen instalasi lengkap untuk LIMA arsitektur: i386, macppc, vax, sparc, sparc64.
  • The following architectures only available via FTP download: alpha, hppa, hp300, mvme68k, mac68k.
  • CD bisa dipakai untuk boot pada i386, macppc, sparc, and sparc64.
  • A funky and surprisingly artistic CD insert sheet which contains installation instructions. The information on this piece of paper makes OpenBSD somewhat easier to install than if you do an FTP install.
  • Sebuah struktur kode sumber lengkap (siap untuk penggunaan AnonCVS ).
  • The latest reliable XFree86 binaries for all architectures
  • The latest XFree86 source code with small modifications by us to make it prettier.
  • Our own ports tree which has improved an insane amount since OpenBSD 3.2. Almost all packages work on almost all architectures.
  • Several pre-built binary packages for the most common architectures, which are very easy to install directly off the CDROM.
  • As always, stickers included!
  • Dan lebih banyak lagi...

OpenBSD 2.3: 19 Mei 1998


Untuk memesan sebuah CDROM 2.3, klik di sini.
  • Dua CDROM dalam sebuah tempat eksklusif.
  • Komponen instalasi lengkap untuk SEPULUH arsitektur: alpha, amiga, arc, hp300, i386, mac68k, mvme68k, pmax, powerpc, sparc.
  • CD bisa dipakai untuk boot pada i386, sparc, alpha, powerpc, and arc.
  • A funky CD insert sheet which contains much better installation instructions than OpenBSD 2.2 shipped with. The information on this piece of paper makes OpenBSD much easier to install than if you do an FTP install.
  • Sebuah struktur kode sumber lengkap (siap untuk penggunaan AnonCVS ).
  • The latest XFree86 binaries for most of the architectures (all except powerpc). Unlike OpenBSD 2.2, XFree86 is now significantly more easy to install.
  • The latest XFree86 source code with small modifications by us to make it prettier.
  • Our own little ports tree which has improved a lot since OpenBSD 2.2.
  • A very large archive of distfiles which are used by the above-mentioned ports tree to build binaries easier.
  • The following pre-built binary packages for most architectures, which are very easy to install directly off the CDROM:
    bash-2.01.1, emacs-20.2, gmake-3.76.1, id-utils-3.2, ircii-2.8.2-epic3.004, pine-3.96, screen-3.7.4, tcsh-6.07.02, unzip-5.3.1.
  • Two NEW OpenBSD 2.3 stickers included!
  • Dan lebih banyak lagi...
