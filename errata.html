  • All architectures

    • RELIABILITY FIX: Nov 8, 1999
      A race condition in newsyslog(8) can cause errors in log file rotation. A source code patch exists which remedies this problem.

    • SECURITY FIX: Nov 9, 1999
      Any user can change interface media configurations. A source code patch exists which remedies this problem.

    • FUNCTIONALITY FIX: Nov 10, 1999
      m4 is quite broken in the 2.6 release. A source code patch exists which remedies this problem.

    • FUNCTIONALITY ADDITION: Nov 10, 1999
      OpenSSH in 2.6 did not support user-owned ~/.ssh/known_hosts files. A source code patch exists which adds this functionalty.
