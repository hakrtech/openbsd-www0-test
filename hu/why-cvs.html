Miért használják az OpenBSD fejlesztők a CVS-t

Az OpenBSD projekt a Concurrent Version System (CVS) programját használja. A CVS segítségével körülbelül hetven fejlesztő dolgozik az operációs rendszer, a portok/csomagok, és az X Window System forráskódján (ami több mint 1349 MB és körülbelül 75000 fájlból áll). A felhasználók így hozzáférhetnek a rendszer forráskódjához az anonymous CVS-en át, ami sokkal megbízhatóbb mint a Software Upgrade Protocol ( sup(1)).

A CVS lehetővé teszi az ágak könnyű létrehozását és karbantartását, ezáltal egy kiadás + patch forrásfát és egy naprakész fejlesztői fát is használhatunk párhuzamos fejlesztéssel. Mindez elvégezhetõ a  CVSweb segítségével is, ami egy kényelmes és navigálható, weben keresztül elérhetõ felület.

Mivel a CVS képes a Secure Shell (ssh(1)) használatára, megbízható felhasználó- és gép azonosítást tesz lehetővé, ami tökéletesen illeszkedik az OpenBSD projekt biztonsági céljaihoz.
