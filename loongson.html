OpenBSD/loongson

OpenBSD/loongson is a port intended to run on systems based upon the Loongson (also known as Godson) 2E and 2F MIPS-compatible processors, using the PMON firmware and boot loader.

The current port maintainer is Miod Vallat (miod@openbsd.org).

History:

The idea of an OpenBSD/loongson started when the EMTEC Gdium Liberty netbook was announced. Unfortunately, the general availability of these machines kept being delayed. Because of this, focus switched to the Lemote Yeeloong netbook instead. After procrastinating for months, serious work started in august 2009, and the kernel work was completed by the end of november 2009.

Unfortunately, most of the Loongson 2F-based hardware available at that time suffers from serious problems in the processor's branch prediction logic, causing the system to freeze, for which no errata has been published except for an e-mail on a toolchain mailing-list.

After a rush of adding silicon problems workarounds, the port to the Lemote Yeeloong was completed and self hosting by the end of january 2010.

Supported hardware:

The OpenBSD/loongson kernel has generic support for Loongson 2E and Loongson 2F processors, as well as the 2F integrated PCI and memory controller. Specific hardware support is then written on a design-by-design basis.

At the moment, the following machines are supported:

  • Lemote Yeeloong
    Both the 8.9" and 10" models are supported. All on-board devices but the audio controller are supported.

Current status:

Development is currently pretty hectic at the moment, although it will slow down in preparation for the first OpenBSD/loongson release.

All supported systems run multiuser, and a currently unaccelerated X server is available.

Projects (in no particular order):

  • Better workarounds for the 2F problems.
  • Write a PMON-based bootloader able to load the kernel from a BSD filesystem, instead of having to put the kernel file on a Linux filesystem.
  • Write support for the onboard audio controller on Yeeloong.
  • Port to other Loongson-based machines, such as the Gdium Liberty netbook and the Lemote Fuloong mini-PC.

Getting and installing OpenBSD/loongson

There is no OpenBSD/loongson formal release yet.

Snapshots will be made available shortly, in this location as well as in a few mirrors.
