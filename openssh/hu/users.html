OpenSSH-t használó operációs rendszerek

A következõ operációs rendszerek és termékek tartalmazzák alaprendszerük részeként az OpenSSH-t.

A lista idõrendi, tehát azok a rendszerek, amik korábban integrálták elõbb szerepelnek.

  • OpenBSD
  • Debian Linux
  • FreeBSD
  • Suse Linux
  • Redhat Linux
  • Mandrake Linux
  • BSDi BSD/OS
  • NetBSD
  • Computone
  • Conectiva Linux
  • Slackware Linux
  • Stallion
  • Cygwin
  • e-smith server és gateway

Amennyiben a fenti listából hiányozna egy olyan rendszer, ami az alaprendszerben tartalmazza az OpenSSH-t kérem írja meg nekünk. Az OpenSSH alaprendszerbe való integráltsága alatt azt értjük, hogy az OpenSSH a rendszer szerves része, nincs szükség arra, hogy a felhasználó annak rendszerhez illesztését önmaga végezze el.
