OpenSSH OpenBSD-re

Mivel az OpenSSH-t az OpenBSD csapat fejleszti, az OpenSSH része az alap operációs rendszernek az OpenBSD 2.6-os kiadása óta. Így nem kell semmit telepítened vagy hozzáadnod a rendszeredhez. Azonban az újabb szolgáltatások és hibajavítások miatt szükség lehet a frissítére.

A jelenlegi OpenSSH kiadás letölthető ezekről az ftp tükrökről.

Akövetkező OpenSSH verziók elérhetők OpenBSD-re:

  • OpenSSH 3.7.1: 2003. Szeptember 16.

    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openssh-3.7.1.tgz

    Ha OpenSSH 3.7.1-et telepítessz OpenBSD 3.3 vagy régebbi rendszerre, akkor a következő foltra van szükséged:
    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openbsd3x_3.7.1.patch.

    # cd /usr/src/usr.bin
    # tar xvfz .../openssh-3.7.1.tgz
# cd ssh
# patch -p0 < .../openbsd3x_3.7.1.patch
# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc/ssh
# mkdir /var/empty
    Az sshd(8) "privilege separation" szolgáltatásának támogatásához néhány régebbi OpenBSD kiadásban, egy új sshd nevű felhasználó és csoport lett hozzáadva a rendszerhez. A felhasználó hozzáadása vipw(8) segítségével: 
    sshd:*:27:27::0:0:sshd privsep:/var/empty:/sbin/nologin

    Add a következőket az /etc/group fájlba:

    sshd:*:27:
_sshagnt:*:34:
    Megjegyzés: A fenti forrást nem lehetett lefordítani korábbi OpenBSD rendszereken, mivel a libc-ben lévő IPV6 támogatás hibás volt.
