Java

A következõ "ingyenes" ügyfélprogramok ajánlottak ahhoz, hogy az OpenSSH-t használhassuk Java ügyfelekbõl:

  • MindTerm 1 egy SSH1 megvalósítás. GPL Licenc alatt.

    "A MindTerm egy teljesen INGYENES(*) SSH protokoll 1 ügyfélprogram amelyet teljesen tiszta Java-ban írtak. A forráskóddal együtt van terjesztve, mivel mi hiszünk abban, hogy egy biztonsággal kapcsolatos szoftverben másként nem lehetne komolyan megbízni. A MindTerm-et magában is lehet futtatni vagy appletként egy weblapon. Lehet futtatni GUI-val (magában) és anélkül. Van két fontos beállítási tulajdonsága ami eltér más kliensektõl, beépített scp fájl átvitel és egy különleges ftp-csatorna amely közönséges ftpd-vel dolgozik az sshd mögött.

    MEGJEGYZÉS: Mindterm 2 nem ingyenes.

  • SSH Plugin egy SSH1 megvalósítás, amit Matthias L. Jugel és Marcus Meißner készített. GPL Licenc alatt.

    "Ez az SSH megvalósítás IDEA és RSA/PKCS#1 kódolást biztosít és eredetileg 1998-ban Cédric Gourio írta. Felhasználta a régi Java Telnet Applet egy részét hogy megírja a saját Java SSH ügyfelét ezért a kód hasonló volt a mi régi appletünkhöz."

  • Ganymed-SSH2. BSD szerû licenc.

    "A Java programkönyvtárhoz készült Ganymed-SSH2-t (mely ismeri az SSH-2 protokollt) használja a Ganymed és még néhány más projekt a zürichi ETH-nál. Ezzel lehetõvé válik SSH kiszolgálókhoz kapcsolódni Java programokból."
