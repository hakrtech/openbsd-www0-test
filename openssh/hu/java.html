Java

A következő "ingyenes" ügyfélprogramok ajánlottak ahhoz, hogy az OpenSSH-t használhassuk Java ügyfelekből:

  • MindTerm 1 egy SSH1 megvalósítás. GPL Licensz alatt.

    "A MindTerm egy teljesen INGYENES(*) SSH protokoll 1 ügyfélprogram amelyet teljesen tiszta Java-ban írtak. A forráskóddal együtt van terjesztve, mivel mi hiszünk abban, hogy egy biztonsággal kapcsolatos szoftverben másként nem lehetne komolyan megbízni. A MindTerm-et magában is lehet futtatni vagy appletként egy weblapon. Lehet futtatni GUI-val (magában) és anélkül. Van két fontos beállítási tulajdonsága ami eltér más kliensektől, beépített scp fájl átvitel és egy különleges ftp-csatorna amely közönséges ftpd-vel dolgozik az sshd mögött.

    MEGJEGYZÉS: Mindterm 2 nem ingyenes.

  • Java-SSH egy SSH1 megvalósítás, amit Cédric Gourio készített.  GPL és nem kereskedelmi licenszek zavaros keveréke alatt adta ki.

    "Ez az applet használja az Ssh 1.5-öt. A Terminal Emuláció majdnem VT100 megfelelő. Biztonságos kommunikációt biztosít nem biztonságos hálózaton keresztül. Minden kommunikáció magától láthatatlanul kódolódik. Ez biztosítja az adatok sértetlenségét is."

  • SSH Plugin egy SSH1 megvalósítás, amit Matthias L. Jugel és Marcus Meißner készített. GPL Licensz alatt.

    "Ez az SSH megvalósítás IDEA és RSA/PKCS#1 kódolást biztosít és eredetileg 1998-ban Cédric Gourio írta. Felhasználta a régi Java Telnet Applet egy részét hogy megírja a saját Java SSH ügyfelét ezért a kód hasonló volt a mi régi appletünkhöz."

  • SSH Toolkits for Java. LGPL/GPL Licensz alatt.

    "Jelenleg három LGPL/GPL alapú projektünk van: J2SSH, egy SSH2 könyvtár, SSHTerm, egy terminál kliens alkalmazás, és egy Windows SSH kiszolgáló."
