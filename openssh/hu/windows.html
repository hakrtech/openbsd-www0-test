Windows

A következõ "szabad" SSH megvalósítások ajánlottak az OpenSSH-val való együttmûködésre Windows gépekrõl:

  • PuTTY SSH1+SSH2 megvalósítás. A PSCP, amely egy scp-szerû program a Windowshoz szintén elérhetõ.

    A PuTTY a MIT licenc alatt érhetõ el (BSD-szerû).

    "A PuTTY egy ingyenes Telnet és SSH megvalósítás Win32 platformokra. Az Angliában élõ Simon Tatham írta és fejleszti."

  • TTSSH (SSH1) SSH1 megvalósítás, Robert O'Callahan-tõl.

    "A TTSSH egy szabad Windowsos SSH kliens. A Teraterm Pro kiegészítõ DDL-jeként használható. A Teraterm Pro egy nagyszerû ingyenes terminál emulátor/telnet kliens Windowsra, forráskódja is elérhetõ. TTSSH a Teraterm meglévõ funkcióinak feláldozása nélkül teszi képessé azt az SSH használatára. A TTSH ugyanígy szabadon letölthetõ és használható a forráskódjával együtt. A TTSSH-t teljesen Ausztráliában fejlesztették [...]."

  • Cygwin (POSIX szoftver Windowson)

    Az OpenSSH (SSH1 és SSH2 protokoll) hordozható verziója, képes a Cygwin segítségével a Windowson futni.

  • MSSH

    A Metropolitan State College of Denver MSSH-ja a Windows 95 és Windows 98-at és az SSH1 protokollt támogatja.

  • Secure iXplorer GPL

    A Secure iXplorer egy grafikus felület a PuTTY pscp.exe-jéhez.

  • WinSCP

    A WinSCP egy scp(1) program a Windowshoz, PuTTY alapra építve.

  • FileZilla

    FileZilla egy kiválló FTP-kliens, Windows 9x, ME, NT4, 2000 és XP alá. Könnyû használatra és a lehetõ legtöbb szolgáltatás támogatására tervezték úgy, hogy mégis gyors és megbízható maradjon. Támogatja az SFTP-t.

  • SSHWindows

    OpenSSH for Windows egy szabad csomag, amely egy minimális OpenSSH kiszolgálót és kliens segédprogramokat telepít a Cygwin csomagba anélkül, hogy szükségég lenne a teljes Cygwin telepítésre. Teljes SSH/SCP/SFTP támogatást biztosít. Az SSH terminál egy megszokott Windows Command prompt-ot ad, viszont megtartja a Unix/Cygwin-stílusú elérési utakat az SCP és az SFTP részére.
