Windows

A következő "ingyenes" kliensek ajánlottak az OpenSSH-val való együttműködésre Windows gépekről:

  • PuTTY SSH1+SSH2 megvalósítás. A PSCP, amely egy scp-szerű program a Windowshoz szintén elérhető.

    A PuTTY a MIT licenc alatt érhető el (BSD-szerű).

    "A PuTTY egy ingyenes Telnet és SSH megvalósítás Win32 platformokra. Az Angliában élő Simon Tatham írta és fejleszti."

  • TTSSH (SSH1) SSH1 megvalósítás, Robert O'Callahan-től.

    "A TTSSH egy szabad Windowsos SSH kliens. A Teraterm Pro kiegészítő DDL-jeként használható. A Teraterm Pro egy nagyszerű ingyenes terminál emulátor/telnet kliens Windowsra, forráskódja is elérhető. TTSSH a Teraterm meglévő funkcióinak feláldozása nélkül teszi képessé azt az SSH használatára. A TTSH ugyanígy szabadon letölthető és használható a forráskódjával együtt. A TTSSH-t teljesen Ausztráliában fejlesztették [...]."

  • Cygwin (POSIX szoftver Windowson)

    Az OpenSSH (SSH1 és SSH2 protokoll) hordozható verziója, képes a Cygwin segítségével a Windowson futni.

  • MSSH

    A Metropolitan State College of Denver MSSH-ja a Windows 95 és Windows 98-at és az SSH1 protokollt támogatja.

  • Secure iXplorer GPL

    A Secure iXplorer egy grafikus felület a PuTTY pscp.exe-jéhez.

  • WinSCP

    A WinSCP egy scp(1) program a Windowshoz, PuTTY alapra építve.

  • FileZilla

    FileZilla egy kiválló FTP-kliens, Windows 9x, ME, NT4, 2000 és XP alá. Könnyű használatra és a lehető legtöbb szolgáltatás támogatására tervezték úgy, hogy mégis gyors és megbízható maradjon. Támogatja az SFTP-t.
