Windows és Macintosh

A következo "szabad" kliensek ajánlottak az OpenSSH-val való együttmûködésre Windows gépekrõl:

  • PuTTY SSH1+SSH2 implementáció. A PSCP, amely egy scp-szerû program a Windowshoz szintén elérhetõ.
    A PuTTY a MIT licensz alatt érhetõ el (BSD-szerû). A Putty SSH2 része nem támogatja a DSA kulcsokkal történõ autentikálást.

    "A PuTTY egy ingyenes telnet és SSH felületet biztosít a Win32 platformokra. Az Angliában élõ Simon Tatham írja és fejleszti."

  • TTSSH (SSH1) SSH1 implementáció, Robert O'Callahan-tõl.

    "A TTSSH egy szabad Windowsos SSH kliens. A Teraterm Pro kiegészítõ DDL-jeként használható. A Teraterm Pro egy ingyenes telnet kliens Windowsra, forráskódja is elérhetõ. TTSSH a Teraterm meglévõ funkcióinak feláldozása nélkül teszi képessé azt az SSH használatára. A TTSH ugyanígy szabadon letölthetõ a forráskódjával együtt. A TTSSH-t Ausztráliában fejlesztették [...]."

  • Telneat (SSH1) egy telnet/SSH kliens Windows platformra Kosta Scheglofftól.

    "a következõ tulajdonságokkal rendelkezik:

    • teljes telnet támogatás
    • teljes SSH támogatás (IDEA, DES, TripleDes, ArcFour, BlowFish, tömörítés)
    • teljes termcap támogatás
    • támogatja a Win95, OSR2, Win98, WinNT, Win2000 platformokat
    • külsõ SSH kódolási pluginek támogatása
    • maximum 10 konzol
    • egérrel használható cut&paste
    • visszafelé görgetési lehetõség
    • automatikus belépés támogatása
    • külön beállítások minden szerverhez"

A következo "szabad" kliensek ajánlottak az OpenSSH-val való együttmûködésre Macintosh gépekrol:

  • NiftyTelnet 1.1 SSH SSH1 implementáció ami egy scp-szerû programmal érkezik. Jonas Wallden készítette.

    "A NiftyTelnet 1.1 SSH r3 a Chris Newman által írt NiftyTelnet 1.1 kibõvített változata, ami lehetõvé teszi, hogy kódolt csatornákat használjunk az SSH protokoll használatával. A program terjesztése elõtt kérem olvassa el a csatolt Readme fájlt!"

  • MacSSH egy kizárólag SSH2-t támogató implementáció.

    "A MacSSH a BetterTelnet módosított változata SSH2 támogatással. [...] Az egyetlen általam talált MacOS-en futó SSH2 kliens kereskedelmi termék, amelynek az ára több mint 100 amerikai dollár és a kapcsolat bezárásakor lefagyasztja a gépet. Ha azt akarod, hogy valami mûködjön, csináld meg magad... Ennek az eredménye a MacSSH"
