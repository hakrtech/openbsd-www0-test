Windows és Macintosh

A következő "ingyenes" kliensek ajánlottak az OpenSSH-val való együttmûködésre Windows gépekrõl:

  • PuTTY SSH1+SSH2 megvalósítás. A PSCP, amely egy scp-szerû program a Windowshoz szintén elérhetõ.

    A PuTTY a MIT licensz alatt érhetõ el (BSD-szerû).

    "A PuTTY egy ingyenes Telnet és SSH megvalósítás Win32 platformokra. Az Angliában élõ Simon Tatham írta és fejleszti."

  • TTSSH (SSH1) SSH1 megvalósítás, Robert O'Callahan-tõl.

    "A TTSSH egy szabad Windowsos SSH kliens. A Teraterm Pro kiegészítõ DDL-jeként használható. A Teraterm Pro egy nagyszerű ingyenes terminál emulátor/telnet kliens Windowsra, forráskódja is elérhetõ. TTSSH a Teraterm meglévõ funkcióinak feláldozása nélkül teszi képessé azt az SSH használatára. A TTSH ugyanígy szabadon letölthetõ és használható a forráskódjával együtt. A TTSSH-t teljesen Ausztráliában fejlesztették [...]."

  • Cygwin (POSIX szoftver Windowson)

    Az OpenSSH (SSH1 és SSH2 protokoll) hordozható verziója, képes a Cygwin segítségével a Windowson futni.

  • MSSH

    A Metropolitan State College of Denver MSSH-ja a Windows 95 és Windows 98-at és az SSH1 protokollt támogatja.

  • Secure iXplorer GPL

    A Secure iXplorer egy grafikus felület a PuTTY pscp.exe-jéhez.

  • WinSCP

    A WinSCP egy scp(1) program a Windowshoz, PuTTY alapra építve.

A Mac OS X alapból tartalmazza az OpenSSH-t. A következő "ingyenes" kliensek ajánlottak az OpenSSH-val való együttmûködésre Mac OS 9 gépekre:

  • NiftyTelnet 1.1 SSH SSH1 implementáció ami egy scp-szerû programmal érkezik. Jonas Wallden készítette.

    "A NiftyTelnet 1.1 SSH r3 a Chris Newman által írt NiftyTelnet 1.1 kibõvített változata, ami lehetõvé teszi, hogy kódolt csatornákat használjunk az SSH protokoll használatával. A program terjesztése elõtt kérlek olvasd el a csatolt Readme fájlt!"

  • MacSSH egy kizárólag SSH2-t támogató implementáció.

    "A MacSSH a BetterTelnet módosított változata SSH2 támogatással. [...] Az egyetlen általam talált MacOS-en futó SSH2 kliens kereskedelmi termék, amelynek az ára több mint 100 amerikai dollár és a kapcsolat bezárásakor lefagyasztja a gépet. Ha azt akarod, hogy valami mûködjön, csináld meg magad... Ennek az eredménye a MacSSH"

  • Fugu SFTP és SCP megvalósítás.

    "Fugu SSH egy grafikus felület Mac OS X alá az OpenSSH Secure File Transfer (biztonságos fájlátvitel, SFTP) alkalmazáshoz. Fuguval kihasználhatod az SFTP biztonságát anélkül, hogy fel kellene adnod a grafikus felület könnyű használhatóságát. Fugu ezen kívül támogatja az SCP fájlátvitelt és képes biztonságos csatornát létrehozni SSH-n keresztül."
