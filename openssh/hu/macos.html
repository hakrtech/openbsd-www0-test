Mac OS

A Mac OS X alapból tartalmazza az OpenSSH-t. A következő "ingyenes" kliensek ajánlottak az OpenSSH-val való együttműködésre Mac OS 9 gépekre:

  • NiftyTelnet 1.1 SSH SSH1 implementáció ami egy scp-szerű programmal érkezik. Jonas Wallden készítette.

    "A NiftyTelnet 1.1 SSH r3 a Chris Newman által írt NiftyTelnet 1.1 kibõvített változata, ami lehetõvé teszi, hogy kódolt csatornákat használjunk az SSH protokoll használatával. A program terjesztése elõtt kérlek olvasd el a csatolt Readme fájlt!"

  • MacSSH egy kizárólag SSH2-t támogató implementáció.

    "A MacSSH a BetterTelnet módosított változata SSH2 támogatással. [...] Az egyetlen általam talált Mac OS-en futó SSH2 kliens, egy kereskedelmi termék, amelynek az ára több mint 100 amerikai dollár és a kapcsolat bezárásakor lefagyasztja a gépet. Ha azt akarod, hogy valami működjön, csináld meg magad... Ennek az eredménye a MacSSH"

  • Fugu SFTP és SCP megvalósítás.

    "Fugu SSH egy grafikus felület Mac OS X alá az OpenSSH Secure File Transfer (biztonságos fájlátvitel, SFTP) alkalmazáshoz. Fuguval kihasználhatod az SFTP biztonságát anélkül, hogy fel kellene adnod a grafikus felület könnyű használhatóságát. Fugu ezen kívül támogatja az SCP fájlátvitelt és képes biztonságos csatornát létrehozni SSH-n keresztül."

  • Cyberduck egy SFTP és FTP böngésző.

    "Cyberduck egy nyílt forrású SFTP (SSH Secure File Transfer) és FTP böngésző, GPL licenc alatt. Kezdetektől fogva a használhatóságot figyelemben tartava készül, ugyanazt a felhasználói felületet adva mind az SFTP mind az FTP böngészőnek. Egyszerre több kapcsolat is támogatott. [...] Rendszer szintű technológiák mint a Keychain vagy a Rendezvous támogatottak."
