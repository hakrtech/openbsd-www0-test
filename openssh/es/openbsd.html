OpenSSH para OpenBSD

Ya que el desarrollo de OpenSSH lo lleva a cabo el grupo de OpenBSD, OpenSSH va incluido en el sistema operativo básico, desde la versión 2.6 de OpenBSD. Así que no hay nada que deba ser instalado o añadido a su sistema.

Si embargo se han desarrollado nuevas revisiones y versiones de OpenSSH desde:

  • OpenSSH 1.2.3 (lanzado el 6 de marzo de 2.000)

    Se puede obtener el archivo comprimido en ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openssh-1.2.3.tgz

    Si lo está instalando en OpenBSD 2.6, también necesitará el siguiente parche: http://www.openssh.com/openbsd26_1.2.3.patch.

    Para extraerlo e instalarlo en su sistema OpenBSD:

    # cd /usr/src/usr.bin
# tar xvfz .../openssh-1.2.3.tgz
# cd ssh
# patch -p1 < openbsd26_1.2.3.patch
# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc
Si está interesado en saber si existen otros parches para OpenSSH, mire en las páginas de erratas de OpenBSD.
