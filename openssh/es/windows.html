Windows y Mac

Se recomiendan los siguientes clientes «libres» para interoperar con OpenSSH desde sistemas Windows:

  • PuTTY es una implementación sólo para el protocolo SSH1. También existe PSCP, un programa para Windows al estilo scp.
    PuTTY se encuentra disponible bajo la licencencia MIT (del estilo BSD).

    «PuTTY es una implementación libre de Telnet y SSH para plataformas Win32, diseñado y mantenido principalmente por Simon Tatham desde Gran Bretaña.»

  • TTSSH (SSH1) es una implementación sólo para el protocolo SSH1 de Rober O'Callahan.

    «TTSSH es un cliente libre para Windows. Está implementado como una extensión DLL para Teraterm Pro. Teraterm Pro es un fantáastico cliente emulador de terminal y cliente de telnet libre para Windows, y su código fuente se encuentra disponible. TTSSH añade funcionalidades de SSH a Teraterm Pro sin sacrificar ninguna de las ya existente funcionalidades de Teraterm. TTSSH está disponible bajo una licencia abierta. Es más, TTSSH ha sido desarrollado completamente en Australia [...].»

Se recomiendan los siguientes clientes para interoperar con OpenSSH desde sistemas Mac:

  • NiftyTelnet 1.1 SSH es una implementación sólo para el protocolo SSH1 que viene con un programa del estilo de scp. Está escrito por Jonas Wallden.

    «NiftyTelnet 1.1 SSH r3 es una versión mejorada de la aplicación NiftyTelnet 1.1 de Chris Newman, con soporte para sesiones de terminal cifradas usando el protocolo SSH ("Secure Shell"). Por favor, lea el fichero Readme que incluye antes de distribuir esta versión.»
