Mac OS

Se recomiendan los siguientes clientes para interoperar con OpenSSH desde máquinas Mac OS. Note que Mac OS X incluye OpenSSH por omisión.

  • NiftyTelnet 1.1 SSH sólo implementa SSH1 y viene con un programa tipo scp. Escrito por Jonas Wallden.

    "NiftyTelnet 1.1 SSH r3 es una versión mejorada de la aplicación NiftyTelnet 1.1 de Chris Newman, con soporte para sesiones de terminal cifradas usando el protocolo SSH (Secure Shell). Por favor, lea el archivo Readme que incluye antes de distribuir esta versión."

  • MacSSH sólo implementa SSH2.

    "MacSSH es una versión modificada de BetterTelnet con soporte para SSH2. [...] El único cliente de SSH2 para MacOS que he podido encontrar es un producto comercial que cuesta más de $100, y que cuelga mi Mac al cerrar la sesión... Ya que lo mejor es hacer las cosas por uno mismo, aquí está MacSSH."

  • Fugu es una implementación de SFTP y SCP para Mac OS X.

    "Fugu SSH es una interfaz gráfica en Mac OS X de la aplicación SFTP de OpenSSH. Fugu permite aprovechar la seguridad de SFTP sin sacrificar la facilidad de uso de una interfaz gráfica de usuario. Fugu también incluye soporte para transferencias de archivos mediante SCP, y la posibilidad de crear túneles seguros a través de SSH."

  • Cyberduck es un examinador de SFTP y FTP para Mac OS X.

    "Cyberduck is un examinador open source de SFTP (SSH Secure File Transfer) y FTP licenciado bajo la GPL. Ha sido construido desde la base con la facilidad de uso en mente, teniendo la misma interfaz gráfico de usuario consistente tanto para examinar SFTP como para FTP. Soporta múltiples conexiones. [...] Tecnologías de sistema importantes tales como Keychain y Rendezvous son soportadas."
