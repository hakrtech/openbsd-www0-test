Sistemas que usan OpenSSH

Los siguientes sistemas operativos y productos tienen OpenSSH integrado en el sistema base.

Esta lista está en orden cronológico, empezando por los sistemas que primero lo integraron.

  • OpenBSD
  • Debian Linux
  • FreeBSD
  • Suse Linux
  • Redhat Linux
  • Mandrake Linux
  • BSDi BSD/OS
  • NetBSD
  • Apple Mac OS-X / Darwin
  • Computone
  • Conectiva Linux
  • Slackware Linux
  • Stallion

Si hemos omitido mencionar algún sistema que incorpore OpenSSH en el sistema base, por favor háganoslo saber (OpenSSH debe estar incluido como opción predeterminada, por ejemplo como parte integral del sistema o en el CD de instalación, sin que sea necesario ningún proceso de integración por parte del usuario).
