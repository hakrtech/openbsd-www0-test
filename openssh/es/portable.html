OpenSSH Portable

El equipo de desarrollo de OpenSSH produce una versión para el proyecto OpenBSD muy pequeña, segura, y fácil de mantener. El Equipo de Portabilidad de OpenSSH coge esa versión pura y le añade código de portabilidad, para que OpenSSH pueda funcionar en otros sistemas operativos (desafortunadamente, y debido a que OpenSSH usa autenticación, tiene muchas diferencias en los distintos sistemas operativos Unix).

OpenSSH está diseñado para funcionar en los siguientes sistemas operativos Unix:

  • Linux
  • Solaris
  • HP/UX
  • Irix
  • AIX
  • SCO
  • NeXT
  • ... y más que van siendo añadidos continuamente.

La versión portable de OpenSSH sigue el desarrollo de la versión oficial, pero las versiones no están sincronizadas. Las versiones portables están marcadas con una 'p' (v.g. 2.1.0p2). El código fuente oficial para OpenBSD nunca se usará el sufijo 'p', sino que se incrementará el número de la versión cuando llegue a «puntos estables» en el desarrollo.
