OpenSSH FTP

Código fuente base de OpenSSH 2.5.1: liberado el 16 de febrero de 2.001

El código fuente base ha sido diseñado específicamente para OpenBSD, para hacerlo simple, limpio, mínimo, y auditable. Un equipo aparte convierte OpenSSH en una versión portable (mucho más grande) que funciona en todos los sitemas operativos.
