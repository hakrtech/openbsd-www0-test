OpenSSH FTP

Código fuente base de OpenSSH 3.3: liberado el 21 de junio de 2.002

El código fuente base ha sido diseñado específicamente para OpenBSD, para hacerlo simple, limpio, mínimo, y auditable. Un equipo aparte convierte OpenSSH en una versión portable que funciona en todos los sistemas operativos.

Insistimos: ¡Para otros sistemas operativos vea los ficheros de la versión portable!
