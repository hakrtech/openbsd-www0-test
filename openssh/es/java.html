Java

Los siguientes clientes «libres» se recomiendan para la interoperabilidad con OpenSSH desde clientes Java:

  • MindTerm 1 es una implementación sólo para SSH1 bajo la licencia GPL.

    «Mindterm es un programa cliente del protocolo 1 de SSH completamente LIBRE(*), escrito en puro Java al 100% (no certificado). Se distribuye con el código fuente, ya que creemos que el software relacionado con la seguridad no sería serio de otro modo. MindTerm puede funcionar como un programa por sí solo, o como una aplicación de java ("applet") en una página web. Se puede usar con o sin interfaz gráfica de usuario (GUI). Dispone de dos funcionalidades características que lo distingue de otros clientes, transferencia de ficheros scp integrada y un túnel ftp especial que funciona con ftpds ordinarios "detrás" de sshd.»

    Nota: MindTerm 2 no es libre.

  • Java-SSH es una implementación sólo para SSH1, de Cédric Gourio. Viene con una confusa mezcla de licencia no comercial y GPL.

    «Esta aplicación de java implementa el protocolo de SSH 1.5. La emulación del terminal es cuasi-conforme con VT100. Provee comunicaciones seguras por redes inseguras. Todas las comunicaciones se cifran de forma transparente y automática. También se usa el cifrado para proteger la integridad.»

  • SSH Plugin es una implementación sólo para SSH1, de Matthias L. Jugel y Marcus Meißner. Licencia GPL.

    «Esta implementación de SSH provee cifrado con IDEA y RSA/PKCS#1, y fue escrita originalmente en 1.998 por Cédric Gourio. Usó parte del viejo "Java Telnet Applet" para escribir su cliente Java SSH, por lo que el código era muy parecido a nuestro viejo applet.»
