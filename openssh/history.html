Project History and Credits

OpenSSH is a derivative of the original free ssh 1.2.12 release from Tatu Ylönen.

Rapidly after the 1.2.12 release, newer versions bore successively more restrictive licenses.

Earlier this year Björn Grönvall re-discovered this particular release and started fixing bugs. His version of ssh is called OSSH.

OpenBSD project members became aware of Björn's work less than two months before the OpenBSD 2.6 release. We wanted to include ssh in the 2.6 release, but we had to make sure that it was perfect. Therefore, we decided to immediately fork from the OSSH release, and pursue rapid development in the same way as the original OpenBSD security auditing process had happened. Theo de Raadt started by removing non-portabilities which made the code harder to read -- the goal being simpler source code, so that security holes and other issues could be spotted easier. Niels Provos quickly removed the remaining cryptographic and GPL'd components, so that we could end up with a completely freely reusable source code base. Markus Friedl jumped in and very quickly managed to replace the SSH protocol code, revision 1.3 in the 1.2.12 release, with revision 1.5 code compatible with the modern ssh 1.2.27 series (this change was needed to operate with a lot of SSH-compatible Windows clients).

Therefore, OpenSSH is based on these older versions of ssh, but with more bugs removed and newer features re-added:

  • has all components of a restrictive nature (i.e. patents, see ssl) directly removed from the source code; any licensed or patented components are chosen from external libraries (e.g. OpenSSL).
  • has been updated to support ssh protocol 1.5.
  • contains added support for kerberos authentication and ticket passing.
  • supports one-time password authentication with skey
  • And hundreds of other changes that people can spot in the new man pages, and in the source code changes.

OpenSSH has been created by Aaron Campbell, Bob Beck, Markus Friedl, Niels Provos, Theo de Raadt, and Dug Song. We thank the many people who have found bugs and reported them.
