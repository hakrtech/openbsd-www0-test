Windows & Mac

Klien "gratis" berikut direkomendasikan untuk interoperasi dengan OpenSSH dari mesin Windows:

  • PuTTY adalah sebuah implementasi SSH1+SSH2. PSCP, sebuah program bergaya scp utnuk Windows, juga tersedia.

    PuTTY tersedia dibawah lisensi MIT (BSD-like).

    "PuTTY adalah sebuah implementasi gratis dari Telnet dan SSh untuk platform Win32, ditulis dan dikelola utamanya oleh Simon Tatham, yang tinggal di Inggris Raya."

  • TTSSH (SSH1) adalah sebuah implementasi SSh1-saja, oleh Robert O'Callahan.

    "TTSSH adalah sebuah klien SSH gratis untuk Windows. Diimplementasikan sebagai perluasan DLL untuk Teraterm Pro. Teraterm Pro adalah sebuah emulator terminal/klien telnet gratis yang hebat untuk Windows, dan kode sumbernya tersedia. TTSSH menambahkan kemampuan SSH ke Teraterm Pro tanpa mengorbankan satupun fungsi yang ada pada Teraterm. TTSSH juga bisa dimuatdapat secara gratis dan digunakan dan kode sumbernya juga disediakan, dengan sebuah lisensi kode terbuka. Lebih jauh, TTSSH telah dikembangkan seluruhnya di Australia [...]."

  • Cygwin (peranti lunak POSIX di atas Windows)

    OpenSSH (protokol SSH1 dan SSH2) dengan Cygwin dapat berjalan pada Windows menggunakan versi portabel OpenSSH.

  • MSSH

    MSSH dari Metropolitan State College of Denver mendukung Windows 95 dan Windows 98, mendukung protokol SSH1.

  • Secure iXplorer GPL

    Secure iXplorer GPL adalah antarmuka grafis untuk pscp.exe PuTTY's.

  • WinSCP

    WinSCP adalah sebuah program scp(1) untuk Windows, dengan PuTTY terintegeasi didalamnya.

Mac OS X menyertakan OpenSSH secara langsung. Klien berikut ini direkomendasikan untuk interoperasi dengan OpenSSH dari mesin Mac OS 9:

  • NiftyTelnet 1.1 SSH adalah sebuah implementasi SSH1-saja yang hadir dengan sebuah program bergaya scp. Ditulis oleh Jonas Wallden.

    "NiftyTelnet 1.1 SSH r3 adalah sebuah versi yang lebih baik dari aplikasi Chris Newman's NiftyTelnet 1.1 yang menambahkan dukungan untuk sesi terminal terenkripsi menggunakan protokol SSH (Secure Shell). MOhon baca berkas Readme yang disertakan sebelum mendistribusikan versi ini."

  • MacSSH adalah sebuah implementasi SSH2-saja.

    "MacSSH adalah sebuah versi termodifikasi dari BetterTelnet dengan dukungan SSH2. [...] Klien SSH2 saja untuk MacOS yang bisa saya temukan adalah sebuah produk komersil yang berharga lebih dari $100, dan membuat Mac saya kacau ketika menutup sebuah sesi... Karena merupakan hal terbaik untuk melakukan suatu hal dengan satu saja, maka inilah MacSSH."
