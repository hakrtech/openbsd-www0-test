メーリングリスト

アナウンスのためのメーリングリスト

OpenSSH の Linux/Unix への移植版には、読む専用のメーリングリスト openssh-unix-announce@mindrot.org があります。 このメーリングリストでは、アップデートされたバージョンや関連する ソフトウエアのアナウンスが行われます。

参加のための情報とアカウント管理については こちらをご覧ください。

開発者のためのメーリングリスト

このメーリングリスト openssh-unix-dev@mindrot.org は他プラットフォームへの移植や拡張を考えている開発者のためのものです。 OpenSSH 一般に関する議論をするところではありませんが、バグの報告や 議論をする場所としてはよいでしょう。

このメーリングリストは参加者以外にも公開されています。 HTML メールはここでは 許されません。小さな添付ファイル (diff ファイルなど) は常識の範囲内で許可されています。 このリストは MARC によってアーカイブされています。

参加のための情報とアカウント管理については こちらをご覧ください。

一般的な SSH ユーザのためのメーリングリスト

一般的な (とくに相互運用の問題に関する) サポートを、 Secure Shell メーリングリスト secureshell@securityfocus.com で受けることができます。 あるいは、Usenet ニュースグループ comp.security.ssh に記事を 投稿するという手もあります。

このメーリングリストに参加するには、 secureshell-subscribe@securityfocus.com に空のメッセージを送ってください。

メーリングリストからぬけるには、 secureshell-unsubscribe@securityfocus.com に空のメッセージを送ってください。

このメーリングリストは、 MARC によってアーカイブされています。
