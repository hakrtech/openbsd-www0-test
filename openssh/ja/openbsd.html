OpenBSD 専用の OpenSSH

OpenSSH は OpenBSD グループによって開発されているため、 OpenBSD リリース 2.6 から OpenSSH は基本のオペレーティングシステムに 含まれています。ですから、お使いのシステムに追加したりインストールしたり する必要はありません。けれども新しい機能やその他のパッチのために アップグレードしたいこともあるでしょう。

OpenSSH の最新リリースは ftp ミラーサイト から ダウンロードできます。

以下の OpenSSH のバージョンを、お使いの OpenBSD マシン上で 動かすことができます:

  • OpenSSH 3.3: 2002年 6月 21日

    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openssh-3.3.tgz

    OpenSSH 3.3 は OpenBSD 3.1 システムで、 修正せずそのままコンパイルできます。 

    # cd /usr/src/usr.bin
# tar xvfz .../openssh-3.3.tgz
# cd ssh
# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc/ssh
# mkdir /var/empty
    OpenBSD 3.0 上でインストールするには、以下のパッチが必要です: To support sshd(8) の root権限分離機能をサポートするために、新しいユーザとグループ sshd が システムに追加されています。 vipw(8) を使って、以下のユーザエントリを追加してください: 

    sshd:*:27:27::0:0:sshd privsep:/var/empty:/sbin/nologin

    Add the following to /etc/group:

    sshd:*:27:
    もし新しいユーザおよびグループが追加できない場合は、以下のパッチが必要になります:
    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openbsd31_3.3.patch

    # cd /usr/src/usr.bin
# tar xvfz .../openssh-3.3.tgz
# cd ssh
# patch -p0 < .../openbsd31_3.3.patch
# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc
# mkdir /var/empty

    OpenBSD 3.0 上でインストールするには、以下のパッチが必要です:

    # /usr/bin/ssh-keygen -d -f /etc/ssh_host_dsa_key -N ''
