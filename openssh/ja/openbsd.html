OpenBSD 専用の OpenSSH

OpenSSH は OpenBSD グループによって開発されているため、 OpenBSD リリース 2.6 から OpenSSH は基本オペレーティングシステムに 含まれています。ですから、お使いのシステムに追加したりインストールしたり する必要はありません。けれども新しい機能やその他のパッチのために アップグレードしたいこともあるでしょう。

OpenSSH の最新リリースは ftp ミラーサイト から ダウンロードできます。

以下の OpenSSH のバージョンを、お使いの OpenBSD マシン上で 動かすことができます:

  • OpenSSH 3.7.1: 2003年 9月 16日

    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openssh-3.7.1.tgz

    OpenBSD 3.3 あるいはそれ以前のシステムに OpenSSH 3.7.1 をインストールする場合は、 以下のパッチが必要です: ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openbsd3x_3.7.1.patch

    # cd /usr/src/usr.bin
# tar xvfz .../openssh-3.7.1.tgz
# cd ssh
# patch -p0 < .../openbsd3x_3.7.1.patch
# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc/ssh
# mkdir /var/empty
    OpenBSD のいくつかの古いリリースでは、 sshd(8) の root 権限分離機能をサポートするために、新しいユーザとグループ sshd が システムに追加されています。 vipw(8) を使って、以下のユーザエントリを追加してください: 
    sshd:*:27:27::0:0:sshd privsep:/var/empty:/sbin/nologin

    以下を /etc/group に追加してください: 

    sshd:*:27:
_sshagnt:*:34:
    注意: 上のソースでは旧バージョンの OpenBSD ではビルドに失敗します。 これは libc の IPV6 サポートが完全ではなかったためです。
