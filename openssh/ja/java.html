Java

以下の「フリー」クライアントが Java クライアントから OpenSSH と 相互運用できるものとしておすすめです:

  • MindTerm 1 は SSH1 プロトコルのみの実装です。GPL ライセンスです。

    「MindTerm は完全にフリー(*) の SSH プロトコル 1 クライアントプログラムで、 100% pure Java で書かれています (保証はされていませんが)。 これはソースコードとともに配布することができます。なぜならセキュリティ 関係のソフトウエアはまじめに信頼する以外にないからです。MindTerm は スタンドアロンのプログラムとしても web ページのアプレットとしても 動きます。GUI つき (スタンドアロン) でも GUI なしでも OK です。 これには他のいくつかのクライアントとは違う 2つの機能があって、 scp を組み込みで持っていることと、『通常の』ftpd が sshd の『後で』 動く特別な ftp トンネルの機能があります」

    注意: Mindterm 2 はフリーではありません。

  • Java-SSH は SSH1 プロトコルのみの実装で、Cédric Gourio によるものです。 非商用と GPL の、混乱しやすいミックスライセンスです。

    「このアプレットは Ssh プロトコル 1.5 を実装したものです。 端末エミュレーションはほとんど VT100 に対応しています。これは 安全でないネットワーク上で安全な通信を提供します。すべての通信は 自動的かつ透過的に暗号化され、この暗号化はまたデータの清潔さを 守るためにも使われます」

  • SSH Plugin は Matthias L. Jugel と Marcus Meißner による、 SSH1 プロトコルのみの実装で、GPL ライセンスです。

    「この SSH の実装は IDEA と RSA/PKCS#1 の暗号化を提供するもので、 これはもともと Cédric Gourio によって 1998年に書かれました。 彼は古い Java 用 Telnet アプレットの一部をつかって自分の Java 用 SSH クライアントを書いたのです。そのためコードは私たちの古い アプレットに似ています」

  • SSH Toolkits for Java は LGPL/GPL ライセンスです。

    「現在、私たちは LGPL/GPL ベースのプロジェクトを 3つ進めています: SSH2 ライブラリ の J2SSH、クライアント端末アプリケーションの SSHTerm、そして Windows 用の SSH サーバです」
