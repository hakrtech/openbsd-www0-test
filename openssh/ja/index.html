OpenSSH 3.8 がリリースされました。2004年 2月 24日。
移植版 OpenSSH 3.8.1p1 がリリースされました。2004年 4月 19日。
SSH1 および SSH2 プロトコル両方をサポートしています。

OpenSSH はネットワーク接続ツールである SSH プロトコルスイートを、 フリーで 実装したものです。OpenSSH は インターネット上の多くの人々にますます信頼されるようになってきています。 Telnet, rlogin, ftp などのプログラムを使っている多くのユーザは、 自分のパスワードがネット上を暗号化されないままで流れていることに 気づいていないかもしれません。でも、実際にはそうなんです。 OpenSSH は (パスワードを含む) すべての通信を暗号化し、 盗聴や接続ハイジャック、およびその他の ネットワークレベルの攻撃を効率よく排除します。 さらに OpenSSH は数多くの安全なトンネリング機能を備えています。

OpenSSH は、rlogin と telnet の代わりになる ssh と、rcp の代わりになる scp 、それに加えて ftp の代わりになる sftp のセット一式を含んでいます。また、サーバ側のパッケージ sshd や、その他の基本ツール ssh-add、 ssh-agent、 ssh-keysign、 ssh-keyscan、 ssh-keygen、および sftp-server も含んでいます。 OpenSSH はバージョン 1.3, 1.5, および 2.0 のプロトコルをサポートしています。

OpenSSH は OpenBSD プロジェクト によって 最初に開発され、オペレーティングシステム OpenBSD 2.6 に初めて塔載されました。 このソフトウエアはアメリカ合衆国の外で開発されており、約10ヶ国の人々が 開発したコードを使っています。そのため、これは BSD ライセンスのもとで 誰でも自由に利用および再利用することができます。

OpenSSH のディストリビューション管理は 2つのチームに分けられています。 ひとつは厳密に OpenBSD ベースの開発をするチームで、できるだけ清潔かつ 簡潔で安全なコードを提供することを目標としています。 移植のための余計な皮がついていない単純なコードによって、 質の管理とレビューの容易さが実現できています。 もうひとつのチームはこの清潔なバージョンをとってきて、 これがほかの多くのオペレーティングシステム上でも走るように移植のための 「皮付け」をします (これらは p リリースとして知られており、 "OpenSSH 3.8.1p1" などのような名前になっています)。 お使いのオペレーティングシステムへのリンクをクリックしてください。

OpenBSD プロジェクトでは OpenSSH Tシャツ と ポスター を販売しています。これらの商品の売り上げは、 開発を支援するための資金として利用されています。

SSH は簡単に使えます。さらに進んだユーザには Daniel Barrett と Richard Silverman による優れた書籍である、 オライリーの SSH book をおすすめします。
