OpenSSH 3.0.2 がリリースされました。2001年 12月 3日。
SSH1 および SSH2 プロトコル両方のサポートを含んでいます。

OpenSSH は、ネットワーク接続ツールである SSH プロトコルを フリーで 実装したものです。 インターネット上の多くの人々が、これに信頼を寄せるようになってきています。 Telnet, rlogin, ftp などのプログラムを使っている多くのユーザは、 自分のパスワードが実はインターネット上を暗号化されないままで 流れていることを認識していないかもしれません。でも実際にはそうなんです。 OpenSSH は (パスワードを含む) すべての通信を暗号化し、 盗聴や接続ハイジャック、およびその他の ネットワークレベルの攻撃を効率的に排除します。 さらに OpenSSH は数多くの安全なトンネリング機能を備えています。

OpenSSH は、rlogin と telnet の代わりとなる ssh と、rcp の代わりとなる scp 、それに加えて ftp の代わりとなる sftp のセット一式を含んでいます。また、サーバ側のパッケージ sshd や、その他の基本ツール ssh-add、 ssh-agent、 ssh-keygen、および sftp-server も含まれています。 OpenSSH はバージョン 1.3, 1.5, および 2.0 のプロトコルをサポートしています。

OpenSSH は最初 OpenBSD プロジェクト によって 開発され、オペレーティングシステム OpenBSD 2.6 に初めて塔載されました。 このソフトウエアはアメリカ合衆国の外で開発されており、約10ヶ国の人々が 開発したコードを使っています。そのため、これは BSD ライセンスのもとで 誰でも自由に利用および再利用することができます。

OpenSSH のディストリビューション管理は 2つのチームに分けられています。 ひとつは厳密に OpenBSD ベースの開発をするチームで、できるだけ清潔かつ 簡潔で安全なコードを提供することを目標としています。 もうひとつは清潔なバージョンを移植可能にするチームです。 このため、 OpenSSH は数多くのオペレーティングシステム上で 動きます (これらは p リリースとして知られており、 "OpenSSH 2.1.1p4" などのような名前になっています)。 お使いのオペレーティングシステムへのリンクをクリックしてください。

OpenBSD プロジェクトでは OpenSSH Tシャツ と ポスター を販売しています。これらの商品の売り上げは、 開発を支援するための資金として利用されています。

SSH は簡単に使えます。さらに進んだユーザには Daniel Barrett と Richard Silverman による優れた書籍である、 オライリーの SSH book をおすすめします。
