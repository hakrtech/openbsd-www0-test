OpenSSH 4.7/4.7p1 リリース (2007年 9月 4日)

OpenSSH は、SSH プロトコルを使用するネットワーク接続ツールの フリーな 実装です。 インターネット上の技術者は、これらのツールに依存しています。 Telnet, rlogin, ftp などのプログラムを使っているユーザは、 自分のパスワードがネット上を暗号化されないまま流れていることを ご存じないかもしれませんが、実際にはこれは危険です。 OpenSSH はパスワードを含むすべての通信を暗号化し、盗聴やネットワーク接続の乗っ取り、 その他の攻撃を効率的に排除します。 また、OpenSSH は安全なトンネリング機能や多様な認証方式を備えており、 すべてのバージョンの SSH プロトコルをサポートしています。

OpenSSH は、rlogin と telnet の代わりになる ssh と、rcp の代わりになる scp 、それに加えて ftp の代わりになる sftp のセット一式を含んでいます。また、(サーバ側のパッケージである) sshd や、 ssh-add、 ssh-agent、 ssh-keysign、 ssh-keyscan、 ssh-keygen、および sftp-server のようなツールも含んでいます。

OpenSSH は OpenBSD プロジェクト によって開発されています。 このソフトウエアは暗号の輸出を認める国々で開発されており、BSD ライセンスのもとで 誰でも自由に利用および再利用することができます。 しかしながら開発には費用がかかっているため、もし OpenSSH がお役に立っている場合 (とくに商用システムの一部として OpenSSH を配布している場合) には、 どうか当プロジェクトへの寄付をご検討ください。

OpenSSH は 2つのチームによって開発されています。 ひとつは厳密に OpenBSD ベースの開発をするチームで、できるだけ清潔かつ 簡潔で安全なコードを提供することを目標としています。 私たちは移植のための余計な皮がついていない単純なコードによって、 質の管理とレビューの容易さが実現できると考えています。 もうひとつのチームは、この清潔なバージョンをとってきて、 これがほかの多くのオペレーティングシステム上でも走るように移植のための 「皮付け」をします。これらは -p リリースとして知られており、 "OpenSSH 4.7p1" などの名前になっています。
