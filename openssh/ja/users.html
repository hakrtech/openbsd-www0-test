OpenSSH を使っているシステム

OpenSSH を基本システムに組み込んでいるものとしては、 以下のオペレーティングシステムや製品が知られています。

このリストは年代順です。 早く組み込んだシステムが最初にくるようにしてあります。

  • OpenBSD
  • Debian Linux
  • FreeBSD
  • Suse Linux
  • Redhat Linux
  • Mandrake Linux
  • BSDi BSD/OS
  • NetBSD
  • Computone
  • Conectiva Linux
  • Slackware Linux
  • Caldera OpenLinux
  • Stallion
  • Rock Linux
  • Cygwin
  • e-smith server and gateway
  • Engarde Linux
  • Mac OS X Version 10.1 以降
  • HP Procurve Switch 4108GL and 2524/2512
  • IBM AIX
  • Gentoo Linux
  • Gwynux/Toadware Linux
  • Sun Solaris 9 以降 (SunSSH という名前です)
  • SmoothWall
  • IPCop
  • SGI Irix
  • ThinLinc
  • Nokia IPSO
  • Cisco CSS11500 series content services switches
  • Cisco SN 5400 series storage routers
  • TopLayer IDS balancers
  • NTI SSH Serial Port Switch
  • Bluecoat (formerly Cacheflow) Proxy SG
  • Novell NetWare
  • Digi CM Console Servers
  • Fedora Linux
  • Alcatel OmniSwitch

OpenSSH を基本システムで使っていて、この一覧に含まれていないものが あれば知らせてください。(ここで言っているのは OpenSSH がデフォルトで 含まれているもの、たとえば CD やシステムに内蔵されていて、ユーザが 自分でインストールする必要がないもののことです。)
