Mac OS

以下のクライアントが Mac OS マシンから OpenSSH と相互運用できるものとしておすすめです。 なお、Mac OS X はデフォルトで OpenSSH を塔載しています。
  • NiftyTelnet 1.1 SSH は SSH1 プロトコルのみの実装で、 scp 風のプログラムもついてきます。 Jonas Wallden によって書かれました。

    「NiftyTelnet 1.1 SSH r3 は Chris Newman の NiftyTelnet 1.1 アプリケーションに SSH (Secure Shell) プロトコルを使った 暗号化された端末セッションのサポートを追加したものです。 このバージョンを配布する前に、これに含まれている Readme ファイルを 読んでください」

  • MacSSH は SSH2 プロトコルのみの 実装です。

    「MacSSH は SSH2 サポートつきの、BetterTelnet の修正バージョンです。 (中略) 私が見つけた唯一の Mac OS 用の SSH2 クライアントは 商用の製品でしたが、 $100 以上もして、おまけに私のマックでは セッションを閉じるときにクラッシュしてしまいました ... 結局自分でやるのが 一番ということになり、MacSSH ができあがったわけです」

  • Fugu は Mac OS X 用の SFTP と SCP の実装です。

    「Fugu SSH は Mac OS X 用の OpenSSH SFTP (Secure File Transfer application) のための グラフィカル・フロントエンドです。Fugu は GUI の簡単な操作性をそこねることなく SFTP のセキュリティを提供します。Fugu はまた SCP のファイル転送もサポートしており、 SSH を介してセキュアなトンネルを作成することも可能です」

  • Cyberduck は Mac OS X 用の SFTP および FTP ブラウザです。

    「Cyberduck は GPL でライセンスされた、オープンソースの SFTP (SSH による安全なファイル転送) および FTP ブラウザです。これは使いやすさを 考えてゼロから作られており、SFTP と FTP のどちらのブラウズでも一貫した グラフィカルインターフェイスをもっています。複数の接続も可能です。 (中略) Keychain や Rendezvous のような、Core システムの技術もサポートされています」
