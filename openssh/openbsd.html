OpenSSH for OpenBSD

Since OpenSSH is developed by the OpenBSD group, OpenSSH is included in the base operating system, starting at OpenBSD release 2.6.

So there's nothing to install or add to your system.

If you are interested in seeing if there are any OpenSSH patches, see the OpenBSD errata pages.
