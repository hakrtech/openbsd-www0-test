OpenSSH for OpenBSD

Since OpenSSH is developed by the OpenBSD group, OpenSSH is included in the base operating system, starting at OpenBSD release 2.6. So there's nothing to install or add to your system.

However, newer releases of OpenSSH have been made since:

  • OpenSSH 1.2.2: released Jan 23, 2000

    A tar ball is available at ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openssh-1.2.2.tgz

    To extract and install it on your OpenBSD 2.6 system:

    # cd /usr/src/usr.bin
# tar xvfz .../openssh-1.2.2.tgz
# cd ssh
# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc
If you are interested in seeing if there are any OpenSSH patches, see the OpenBSD errata pages.
