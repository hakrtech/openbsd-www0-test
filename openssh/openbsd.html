OpenSSH for OpenBSD

Since OpenSSH is developed by the OpenBSD group, OpenSSH is included in the base operating system, starting at OpenBSD release 2.6. So there's nothing to install or add to your system. However, due to newer features or other patches, you may wish to upgrade.

The following versions of OpenSSH are available for putting on your OpenBSD machines:

  • OpenSSH 2.1: May 8, 2000

    A tar ball is available at ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openssh-2.1.tgz

    It will be difficult to port OpenSSH 2.1 to OpenBSD 2.6, until someone writes a patch similar to the one shown below for OpenSSH 1.2.3.

  • OpenSSH 1.2.3: released March 6, 2000

    A tar ball is available at ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openssh-1.2.3.tgz

    If you are installing on OpenBSD 2.6, you also need the following patch: http://www.openssh.com/openbsd26_1.2.3.patch.

    To extract and install it on your OpenBSD system:

    # cd /usr/src/usr.bin
# tar xvfz .../openssh-1.2.3.tgz
# cd ssh
# patch -p1 < openbsd26_1.2.3.patch
# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc
If you are interested in seeing if there are any OpenSSH patches, see the OpenBSD errata pages.
