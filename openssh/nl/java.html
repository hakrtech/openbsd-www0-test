Java

De volgende "vrije" clients worden aangeraden voor het gebruik van OpenSSH met Java clients:

  • MindTerm 1 is een SSH1-implementatie. GPL Licentie.

    "MindTerm in een volledig VRIJ(*) SSH protocol 1 clientprogramma geschreven in 100% pure Java (niet gecertificeerd). Het wordt inclusief broncode uitgebracht omdat we geloven dat anders op veiligheidsgerelateerde software niet serieus vertrouwd kan worden. MindTerm kan stand-alone draaien of als een applet in een webpagina. Het kan met of zonder GUI gedraaid worden (stand-alone). Het heeft twee noemenswaardige functionaliteiten die het onderscheiden van andere clients: ingebouwde scp bestandsoverdracht en een speciale ftp-tunnel die met "gewone" ftpd's "achter" sshd werkt."

    N.B.: Mindterm 2 is niet vrij.

  • SSH Plugin is een SSH1-implementatie door Matthias L. Jugel en Marcus Meißner. GPL Licentie.

    "Deze implementatie van SSH biedt IDEA- en RSA/PKCS#1-encryptie en was oorspronkelijk in 1998 geschreven door Cédric Gourio. Hij gebruikte een deel van de oude Java Telnet Applet om zijn Java SSH-client te schrijven en hierdoor leek de code op die van onze oude applet.

  • Ganymed SSH-2 for Java. BSD-stijl licentie.

    "Ganymed SSH-2 voor Java is een vrij beschikbare open-source library die het SSH-2 protocol implementeert in pure Java. Het ondersteunt SSH sessies (remote uitvoering van commando's en shell toegang), lokale en remote port forwarding, lokale stream forwarding, X11 forwarding, SCP en SFTP. Het hangt niet af van een JCE provider, aangezien alle crypto functionaliteit inbegrepen is."
