Java

De volgende "vrije" clients worden aangeraden voor het gebruik van OpenSSH met Java clients:

  • MindTerm 1 is een SSH1-implementatie. GPL Licentie.

    "MindTerm in een volledig VRIJ(*) SSH protocol 1 clientprogramma geschreven in 100% pure Java (niet gecertificeerd). Het wordt inclusief broncode uitgebracht omdat we geloven dat anders op veiligheidsgerelateerde software niet serieus vertrouwd kan worden. MindTerm kan stand-alone draaien of als een applet in een webpagina. Het kan met of zonder GUI gedraaid worden (stand-alone). Het heeft twee noemenswaardige functionaliteiten die het onderscheid van andere clients, ingebouwde scp bestandsoverdracht en een speciale ftp-tunnel die met "gewone" ftpd's "achter" sshd werkt."

    N.B.: Mindterm 2 is niet gratis.

  • Java-SSH is een SSH1-implementatie door Cédric Gourio. Bevat een verwarrende mix van niet-commerciële licenties en GPL-licenties.

    "Deze applet implementeert het Ssh Protocol 1.5. De Terminal Emulation is bijna volledig VT100. Het biedt veilige communicatie over onveilige netwerken. Alle communicatie wordt automatisch en transparant versleuteld. Encryptie wordt ook gebruikt om de integriteit te beschermen."

  • SSH Plugin is een SSH1-implementatie door Matthias L. Jugel en Marcus Meißner. GPL Licentie.

    "Deze implementatie van SSH biedt IDEA- en RSA/PKCS#1-encryptie en was oorspronkelijk in 1998 geschreven door Cédric Gourio. Hij gebruikte een deel van de oude Java Telnet Applet om zijn Java SSH-client te schrijven en hierdoor leek de code op die van onze oude applet.
