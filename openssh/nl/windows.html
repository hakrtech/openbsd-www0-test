Windows

De volgende "vrije" SSH-implementaties worden aangeraden voor gebruik met OpenSSH vanaf Windows machines:

  • PuTTY is een SSH1+SSH2-implementatie. PSCP, een scp-achtig programma voor Windows is ook beschikbaar.

    PuTTY is beschikbaar onder de MIT-licentie. (BSD-achtig).

    "PuTTY is een vrije implementatie van Telnet en SSH voor Win32-platformen, geschreven en onderhouden voornamenlijk door Simon Tatham, die in Groot-Britannië leeft."

  • TTSSH (SSH1) is een SSH1-implementaie, door Robert O'Callahan.

    "TTSSH is een vrije SSH-cliënt voor Windows. Het is geimplementeerd als een extensie-DLL voor Teraterm Pro. Teraterm Pro is een geweldige vrije terminal emulator/telnetcliënt voor Windows en de broncode is beschikbaar. TTSSH voegt SSH-functionaliteit toe aan Teraterm Pro zonder in te doen aan de bestaande functionaliteit van Teraterm. TTSSH is gratis te downloaden en te gebruiken en de broncode is ook beschikbaar onder een open-sourcelicentie. Verder is TTSSH in z'n geheel ontwikkeld in Australië [...]."

  • Cygwin (POSIX software bovenop Windows)

    OpenSSH (SSH1- en SSH2-protocol) met Cygwin kan op Windows draaien met de portable versie van OpenSSH.

  • MSSH

    MSSH van het Metropolitan State College van Denver ondersteunt Windows 95 en Windows 98, hierbij het SSH1-protocol ondersteunend.

  • Secure iXplorer GPL

    Secure iXplorer GPL is een grafische schil voor PuTTY's pscp.exe.

  • WinSCP

    WinSCP is een scp(1)-programma voor Windows, met PuTTY geïntegreerd.

  • FileZilla

    FileZilla is een veelzijdige FTP-client voor Windows 9x, ME, NT4, 2000 en XP. Het is ontworpen om makkelijk in het gebruik te zijn en zoveel mogelijk features te ondersteunen en hiernaast snel en betrouwbaar te zijn. Het ondersteunt SFTP.

  • SSHWindows

    • OpenSSH voor Windows is een vrij pakket dat een minimale OpenSSH server en clientprogramma's in het Cygwin pakket installeert zonder daarvoor de volledige Cygwin installatie nodig te hebben. Het biedt volledige ondersteuning voor SSH/SCP/SFTP, ondersteuning voor een SSH terminal heeft een bekende Windows Command prompt terwijl er gebruikt wordt gemaakt van Unix/Cygwin-achtige padnamen voor SCP en SFTP.
