Windows

De volgende "vrije" SSH-implementaties worden aangeraden voor gebruik met OpenSSH vanaf Windows machines:

  • PuTTY is een SSH1+SSH2-implementatie. PSCP, een scp-achtig programma voor Windows is ook beschikbaar.

    PuTTY is beschikbaar onder de MIT-licentie. (BSD-achtig).

    "PuTTY is een vrije implementatie van Telnet en SSH voor Win32-platformen, geschreven en onderhouden voornamelijk door Simon Tatham, die in Groot-Britannië leeft."

  • TTSSH (SSH1) is een SSH1-implementaie, door Robert O'Callahan.

    "TTSSH is een vrije SSH-client voor Windows. Het is geïmplementeerd als een extensie-DLL voor Teraterm Pro. Teraterm Pro is een geweldige vrije terminal emulator/telnetclient voor Windows en de broncode is beschikbaar. TTSSH voegt SSH-functionaliteit toe aan Teraterm Pro zonder in te boeten aan de bestaande functionaliteit van Teraterm. TTSSH is gratis te downloaden en te gebruiken en de broncode is ook beschikbaar onder een open-sourcelicentie. Verder is TTSSH in z'n geheel ontwikkeld in Australië [...]."

    Er is ook een variant UTF-8 TeraTerm Pro met TTSSH2 versie beschikbaar.

  • Cygwin (POSIX software bovenop Windows)

    OpenSSH (SSH1- en SSH2-protocol) met Cygwin kan op Windows draaien met de portable versie van OpenSSH, die ofwel vanaf broncode kan gebouwd worden of geïnstalleerd als een Cygwin native package. Er bestaan verscheidene andere packages van OpenSSH op Cygwin (bv. sshwindows (niet onderhounden), COPSSH).

  • WinSCP

    WinSCP is een GUI sftp(1) en scp(1) client programma voor Windows. De basis voor het SSH-protocol is gebaseerd op PuTTY.

  • FileZilla

    FileZilla is een veelzijdige FTP-client voor Windows 9x, ME, NT4, 2000 en XP. Het is ontworpen om makkelijk in het gebruik te zijn en zoveel mogelijk functionaliteiten te ondersteunen en hiernaast snel en betrouwbaar te zijn. Het ondersteunt SFTP.
