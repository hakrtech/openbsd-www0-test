Windows & Mac

De volgende "vrije" cliënten voor Windows machines worden aangeraden voor gebruik met OpenSSH:

  • PuTTY is een SSH1+SSH2-implementatie. PSCP, een scp-achtig programma voor Windows is ook beschikbaar.

    PuTTY is beschikbaar onder de MIT-licentie. (BSD-achtig).

    "PuTTY is een vrije implementatie van Telnet en SSH voor Win32-platformen, geschreven en onderhouden voornamenlijk door Simon Tatham, die in Groot-Britannië leeft."

  • TTSSH (SSH1) is een SSH1-implementaie, door Robert O'Callahan.

    "TTSSH is een vrije SSH-cliënt voor Windows. Het is geimplementeerd als een extensie-DLL voor Teraterm Pro. Teraterm Pro is een geweldige vrije terminal emulator/telnetcliënt voor Windows en de broncode is beschikbaar. TTSSH voegt SSH-functionaliteit toe aan Teraterm Pro zonder in te doen aan de bestaande functionaliteit van Teraterm. TTSSH is gratis te downloaden en te gebruiken en de broncode is ook beschikbaar onder een open-sourcelicentie. Verder is TTSSH in z'n geheel ontwikkeld in Australië [...]."

  • Cygwin (POSIX software bovenop Windows)

    OpenSSH (SSH1- en SSH2-protocol) met Cygwin kan op Windows draaien met de portable versie van OpenSSH.

  • MSSH

    MSSH van het Metropolitan State College van Denver ondersteunt Windows 95 en Windows 98, hierbij het SSH1-protocol ondersteunend.

  • Secure iXplorer GPL

    Secure iXplorer GPL is een grafische schil voor PuTTY's pscp.exe.

  • WinSCP

    WinSCP is een scp(1)-programma voor Windows, met PuTTY geïntegreerd.

OpenSSH is standaard bij Mac OS X inbegrepen. De volgende clients worden aangeraden om vanaf Mac OS 9 machines te gebruiken met OpenSSH:

  • NiftyTelnet 1.1 SSH is een SSH1-implementatie met een scp-achtig programma. Geschreven door Jonas Wallden.

    "NiftyTelnet 1.1 SSH r3 is een verbeterde versie van Chris Newman's NiftyTelnet 1.1 applicatie die ondersteuning voor versleutelde terminalsessies toevoegt met behulp van het SSH (Secure Shell) protocol. Lees alstublieft het Readme-bestand voordat u deze versie distribueert."

  • MacSSH is een SSH2-implementatie.

    "MacSSH is een aangepaste versie van BetterTelnet met ondersteuning voor SSH2. [...] De enige SSH2-cliënt voor MacOS die ik kon vinden was een commercieel produkt die meer dan $100 kost en mijn Mac laat crashen wanneer een sessie afgesloten wordt... Aangezien je het beste alles zelf kan doen is hier MacSSH."

  • Fugu is een implementatie van SFTP en SCP.

    "Fugu SSH is een Mac OS X grafische front-end voor de Secure File Transfer applicatie (SFTP) van OpenSSH. Met Fugu kun je gebruik maken de veiligheid van SFTP zonder het gemak van een GUI op te hoeven geven. Fugu heeft ook ondersteuning voor bestandsoverdracht via SCP en de mogelijkheid om veilige tunnels te maken via SSH."
