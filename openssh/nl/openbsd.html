OpenSSH voor OpenBSD

Aangezien OpenSSH ontwikkeld wordt door de OpenBSD groep is OpenSSH standaard bij het besturingssysteem inbegrepen sinds OpenBSD versie 2.6. Er hoeft dus niets geinstalleerd of toegevoegd te worden aan uw systeem. Maar u wilt wellicht upgraden voor nieuwe functionaliteiten of andere patches.

U kunt de huidige OpenSSH versie downloaden van deze ftp mirrors.

De volgende versies van OpenSSH zijn beschikbaar voor u om op uw OpenBSD machines te zetten:

  • OpenSSH 3.4: 26 Juni, 2002

    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openssh-3.4.tgz

    OpenSSH 3.4 compileert goed zonder aanpassingen op uw OpenBSD 3.1 systeem. 

    # cd /usr/src/usr.bin
# tar xvfz .../openssh-3.4.tgz
# cd ssh
# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc/ssh
# mkdir /var/empty
    Voor de privilege seperation optie van sshd(8) zijn een nieuwe gebruiker en groep genaamd sshd aan het systeem toegevoegd. Voeg de volgende gebruiker toe met vipw(8): 

    sshd:*:27:27::0:0:sshd privsep:/var/empty:/sbin/nologin

    Voeg het volgende toe aan /etc/group:

    sshd:*:27:
    Als u de nieuwe gebruiker en groep niet kunt toevoegen heeft u ook deze patch nodig:
    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openbsd31_3.4.patch.

    # cd /usr/src/usr.bin
# tar xvfz .../openssh-3.4.tgz
# cd ssh
# patch -p0 < .../openbsd31_3.4.patch
# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc
# mkdir /var/empty

    Als u het installeert op OpenBSD 3.0 heeft u ook de volgende patch nodig:
    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openbsd30_3.4.patch.
