Mac OS

De volgende clients worden aanbevolen voor het samenwerken met OpenSSH vanaf Mac OS machines. Merk op dat Mac OS X standaard OpenSSH bevat.

  • NiftyTelnet 1.1 SSH is een enkel-SSH1 implementatie die een scp-stijl programma bundelt. Geschreven door Jonas Wallden.

    "NiftyTelnet 1.1 SSH r3 is een verbeterde versie van Chris Newman's NiftyTelnet 1.1 toepassing die ondersteuning voor vercijferde terminal sessies met het SSH (Secure Shell) protocol toevoegt. Lees alstublieft het Readme bestand dat er bij zit alvorens deze versie te verspreiden."

  • MacSSH is een enkel-SSH2 implementatie.

    "MacSSH is een gewijzigde versie van BetterTelnet met SSH2 ondersteuning. [...] De enige SSH2 client voor Mac OS die ik kon vinden is een commercieel product dat meer dan $100 kost, en het laat mijn Mac crashen bij het sluiten van een sessie... Aangezien het het best is om de dingen zelf te doen, is hier MacSSH."

  • Fugu is een implementatie van SFTP en SCP voor Mac OS X.

    "Fugu SSH is een Mac OS X grafische front-end voor OpenSSH's Secure File Transfer toepassing (SFTP). Fugu laat u toe voordeel te halen uit SFTP's beveiliging zonder het gebruiksgemak van een GUI te moeten opgeven. Fugu bevat ook ondersteuning voor SCP bestandtransfers, en de mogelijkheid om beveiligde tunnels te creëren via SSH."

  • Cyberduck is een SFTP en FTP browser voor Mac OS X.

    "Cyberduck is een open bron SFTP (SSH Secure File Transfer) en FTP browser onder de GPL licentie. Het werd van de grond af gebouwd met bruikbaarheid in het achterhoofd, en heeft dezelfde consistente grafische gebruikersinterface voor zowel SFTP als FTP browsen. Meerdere verbindingen worden ondersteund. [...] Kernsysteemtechnologieën zoals Keychain en Rendezvous worden ondertsteund."
