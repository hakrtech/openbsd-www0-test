Perfil del uso de SSH

Esta máquina es parte de un proyecto para obtener un de los servidores de Secure Shell (SSH) que se utilizan en Internet. Este proyecto es muy parecido al Netcraft Web Server Survey. Recogemos y examinamos muestras aleatorias de anfitriones en internet que provean el servicio SSH. A partir de ahía conectamos a cada uno de ellos para determinar la versión y el protocolo desde el diálogo de conexión. Los resultados se utilizarán para publicar datos sobre el uso del protocolo SSH en Internet. Nuestra intención es la de publicar nuestros resultados en Internet y en la próxima conferencia de Usenix.

En cualquier caso nos limitaremos a dar una rápida mirada a la cadena de conexión de su máquina. El explorador (escáner) no intentará meterse en su máquina. En ningún caso se repetirán las conexiones a una sola máquina, ya que el explorador genera solamente direcciones aleatorias únicas en Internet.

No se dará ninguna información sobre una dirección o dominio específico a nadie que no pertenezca al proyecto, sólo las conclusiones. No publicaremos ningún detalle sobre qué anfitriones estén usando qué versiones.

Las estadísticas sobre nuestros resultados de exploración actuales se pueden ver aquí. Las exploraciones se han llevado a cabo con el programa scanssh.
