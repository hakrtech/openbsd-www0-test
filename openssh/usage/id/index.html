Implementasi SSH di Internet.

Profil penggunaan SSH

Mesin ini digunakan dalam sebuah proyek untuk membuat profil server Secure Shell (SSH) dalam penggunaannya di Internet. Proyek ini sangat mirip dengan Survei Server Web Netcraft. Kami mengumpulkan dan memilah sample acak host di Internet yang menyediakan layanan SSH. Kami lalu melakukan koneksi ke masing-masing host untuk menetukan versi server dan protokol dari dialog koneksi. Hasilnya akan digunakan untuk mengumumkan data penggunaan protokol SSH secara luas di Internet. Kami mempublikasikan hasil kami pada web maupun pada sebuah konferensi Usenix berikutnya.

Kami tidak akan melakukan lebih dari sebuah pemeriksaan kecil pada string koneksi mesin anda. pemindai tidak akan mencoba untuk masuk ke mesin anda. Tidak ada pengulangan koneksi ke mesin anda karena pemindai membangkitkan hanya benar-benar alamat acak di Internet yang unik

Tidak ada keterangan khusus suatu alamat atau domain akan disediakan untuk siapa saja di luar proyek, sebagai pengecualian. Kami tidak akan mengumumkan setiap rincian tentang versi yang dijalankan setiap hosts.

Statistik dari hasil pemindaian terakhir kami dapat ditemukan di sini. Pemindaian dilakukan dengan peranti lunak scanssh.
