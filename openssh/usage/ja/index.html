SSH 利用統計

このマシンは、インターネット上における Secure Shell (SSH) のバージョンをプロファイリングするプロジェクトで 利用されています。このプロジェクトは Netcraft Web Server Survey によく似ています。 私たちはインターネット上で SSH のサービスを起動しているホストを ランダムに抽出しました。つぎにこれらのホストに接続し、 その接続内容からサーバとプロトコルのバージョンを特定しています。 この結果は、インターネット全体の SSH プロトコルの使用状況に関するデータを 公表するために利用されます。この結果は web サイトおよび 来たる Usenix カンファレンスで発表される予定です。

私たちは、皆様のマシンに接続できた時の文字列を読む以上のことは いたしません。スキャンプログラムはお使いのマシンに ログインしようとはしません。生成されるインターネット上のアドレスは ランダムなもので、接続が 2度以上行われることはありません。

プロジェクト外の人間に、特定のアドレスやドメインに関する情報が 漏れることはありません。どのホストでどのバージョンが走っているかという 詳細を公表するつもりはありません。

現在のスキャン結果による統計は こちら で見ることができます。 スキャンは scanssh ソフトウエア によって行われています。
