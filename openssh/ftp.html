OpenSSH FTP

OpenSSH 3.0.1 base source code: released November 15, 2001

The base source code is designed specifically for OpenBSD, to make it simple, clean, minimal, and auditable. A separate team converts OpenSSH to a (much larger) portable release which runs on all operating systems.

We repeat: For other systems, see the portable release files!
