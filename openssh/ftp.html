OpenSSH FTP

OpenSSH 2.1.0 base source code: released May 8, 2000

The base source code is designed specifically for OpenBSD, to make it simple, clean, minimal, and auditable. A seperate team converts OpenSSH to a (much larger) portable release which runs on all operating systems.
