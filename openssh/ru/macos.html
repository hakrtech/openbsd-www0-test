Mac OS

Следующие клиенты для Mac OS рекомендуются для работы с OpenSSH. Не забывайте, что Mac OS X включает в себя OpenSSH по умолчанию.

  • NiftyTelnet 1.1 SSH представляет из себя реализацию в стиле scp, поддерживающую только SSH1. Автор: Йонас Wallden.

    "NiftyTelnet 1,1 SSH r3 является усовершенствованной версией NiftyTelnet 1.1 Криса Ньюмана (Chris Newman), которая включает в себя поддержку шифрования сеансов связи, используя протокол SSH (Secure Shell). Если Вы захотите распространять эту версию, пожалуйста, прочитайте сначала Readme файл".

  • MacSSH является реализацией SSH2.

    "MacSSH представляет модифицированную версию BetterTelnet с поддержкой SSH2. [...] Единственный SSH2-клиент для Mac OS, который я смог найти - это коммерческий продукт Thats, который стоит больше 100 долларов, и после закрытии сессии которого зависает мой Mac... Так что я решил, что будет лучше написать клиент самому. Вот MacSSH".

  • Fugu является реализацией SFTP и SCP для Mac OS X.

    "Fugu SSH представляет из себя Mac OS X GUI для OpenSSH's Secure File Transfer application (SFTP). Он позволяет пользоваться преимуществами безопасности SFTP без необходимости жертвовать простотой использования, находясь в GUI. Fugu SSH также включает в себя поддержку SCP, а также возможность создавать защищенные туннели через SSH".

  • Cyberduck это SFTP и FTP-браузер для Mac OS X.

    "Cyberduck является open source SFTP (SSH Secure File Transfer) и FTP-браузер (под лицензией GPL). Он был написан с нуля. При разработке акцент делался главным образом на функциональность, а также на создание простого GUI для работы SFTP и FTP. Поддерживается работа нескольких соединений [...] Поддерживаются такие системные технологии как Keychain и Rendezvous.
