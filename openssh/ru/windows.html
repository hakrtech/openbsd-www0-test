Windows

Следующие "свободые" реализации SSH рекомендуются для работы с OpenSSH из под Windows-машин:

  • PuTTY является SSH1+SSH2 реализацией. PSCP, программа для Windows в scp-стиле, также доступна.

    PuTTY распространяется по лицензии MIT (BSD-подобной).

    "PuTTY является свободная реализацей Telnet и SSH для Win32 платформ. Написана и поддерживается в первую очередь Simon Tatham, который живет в Великобритании."

  • TTSSH (SSH1) поддерживает только первую версию SSH. Разрабатывается Robert O'Callahan.

    "TTSSH - свободный SSH-клиент для Windows. Он реализован как дополнительное расширение DLL для Teraterm Pro. Teraterm Pro является превосходным свободным эмулятором терминала/telnet-клиент для Windows, и ее исходники доступны. TTSSH добавляет возможности SSH к Teraterm Pro, при этом функциональность самой Teraterm Pro от этого не стадает. TTSSH свободно распространяется под открытой лицензией. Кроме того, TTSSH был разработан в Австралии[...]."

    Также существует альтернативный вариант UTF-8 TeraTerm Pro with TTSSH2.

  • Cygwin (POSIX-Software для Windows)

    OpenSSH (SSH1 и SSH2) с Cygwin может работать на Windows с помощью портированной версии OpenSSH, которая может быть собрана из исходников или установленна как пакет Cygwin.

  • WinSCP

    WinSCP представляет из себя GUI для sftp(1) и scp(1) клиентские программы для Windows. Ядро её SSH-протокола основано на PuTTY.

  • FileZilla

    FileZilla это мощный FTP-клиент для Windows 9x, ME, NT4, 2000 и XP. Он был разработан главным образом как простой в использовании клиент с поддержкой множества возможностей, возможно, еще и будучи быстрым и надежным. Он поддерживает SFTP.
