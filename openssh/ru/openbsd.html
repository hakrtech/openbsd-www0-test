OpenSSH для OpenBSD

OpenSSH разрабатывается командой OpenBSD и включён в базовую поставку операционной системы, начиная с OpenBSD 2.6. Поэтому нет необходимости дополнительно устанавливать или добавлять его в систему. Однако, в связи с появлением новых возможностей или исправлений, вам, вероятно, захочется обновить его.

Вы можете переписать текущий релиз OpenSSH с ftp зеркал.

Доступные версии OpenSSH для установки на OpenBSD:

  • OpenSSH 3.1: 7 Марта, 2002

    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openssh-3.1.tgz

    OpenSSH 3.0.2 без проблем скомпилируется на OpenBSD 3.0. Начиная с версии OpenSSH 3.1, конфигурационные файлы расположены в /etc/ssh/. Если вы хотите, чтобы конфигурационные файлы были расположены в /etc/, вам необходим следующий патч:
    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openbsd30_3.1.patch.

    # cd /usr/src/usr.bin
# tar xvfz .../openssh-3.1.tgz
# cd ssh
# patch -p0 < .../openbsd30_3.1.patch
# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc
    Убедитесь, что в sshd_config есть строка 'HostKey /etc/ssh_host_dsa_key'.

    Если вы устанавливаете на OpenBSD 2.9, вам необходим следующий патч:
    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openbsd29_3.1.patch.

    Для установки на OpenBSD 2.9:

    # /usr/bin/ssh-keygen -d -f /etc/ssh_host_dsa_key -N ''
