SSH Implementationen im Internet.

Profil der SSH Verbreitung

Diese Maschine ist Teil eines Projektes, dass die Verbreitung der verschiedenen Secure Shell (SSH) Servern im Internet misst. Dieses Projekt ist dem Netcraft Web Server Survey sehr ähnlich. Wir sammeln zufällige Daten von Hosts im Internet, die den SSH Dienst anbieten und werten sie aus. Wir verbinden uns dann mit jedem, um die Server und Protokoll-Version aus dem Verbindungsdialog herauszubekommen. Die Resultate stellen dann die Daten über die Benutzung des SSH Protokolls im Internet bereit . Wir wollen die die Daten sowohl im Web, als auch auf einer kommenden Usenix Konferenz veröffentlichen.

Wir werden keinen genauen Blick auf den Connection String deiner Maschine werfen. Der Scanner wird nicht versuchen, sich in deine Maschine einzuloggen. Es wird keine mehrmaligen Verbindungen geben, da der Scanner zufällige Adressen aus dem Internet auswählt.

Es werden keinerlei Addressen oder Domain spezifische Informationen an irgendjemanden ausserhalb des Projektes gegeben, einzig die Schlussfolgerungen. Wir werden keine Details darüber preisgeben, welcher Host welche Version benutzt.
