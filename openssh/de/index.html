         *NEW* OpenSSH 1.2.3 released March 6, 2000.

OpenSSH ist eine freie Version der Remote-Login Software SSH auf die sich immer mehr Leute im Internet verlassen. Vielen Benutzer von telnet, rlogin, ftp und ähnlichen Programmen ist noch nicht aufgefallen, dass ihr Passwort dabei unverschlüsselt über das Internet übertragen wird. So ist es jedoch! OpenSSH verschlüsselt die Passwörter und die gesamte Verbindung um Mithören (Eavesdropping), das "Entführen von Verbindungen" (Connection Hijacking) und ähnliche Netzwerkangriffe zu unterbinden.

OpenSSH beinhaltet das Programm ssh, das rlogin und telnet, und scp, das rcp und ftp ersetzt. Ausserdem sind ist der Server sshd und weitere Programme ssh-add, ssh-agent, und ssh-keygen enthalten. OpenSSH unterstuetzt die Protokollversionen 1.3 und 1.5 (an Unterstützung für Version 2.0 wird gearbeitet).

OpenSSH wird vom OpenBSD Projekt entwickelt und OpenBSD 2.6 ist das erste Betriebssystem das OpenSSH integriert hat (CDs können bestellt werden). Die Software wird ausserhalb der USA entwickelt. Dabei wird Programmcode aus ungefähr 10 Ländern genutzt. OpenSSH steht unter der BSD Lizenz und ist somit für jeden frei nutz- und wiederverwertbar.

Das OpenBSD Projekt verkauft ein OpenSSH Tshirt.
