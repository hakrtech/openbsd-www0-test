         *NEW* OpenSSH 1.2.3 released March 6, 2000.

OpenSSH ist eine FREIE Version der SSH-Suite von Netzwerk-Verbindungs-Tools, auf die sich eine steigende Anzahl von Leuten im Internet verlassen müssen. Viele Benutzer von telnet,rlogin,ftp und anderer solcher Programme mögen noch nicht realisiert haben, dass ihr Passwort dabei unverschlüsselt über das Internet übertragen wird, aber so ist es ! OpenSSH verschlüsselt den ganzen Verkehr (inklusive der Passwörter) um Mithören (Eavesdropping), das "Entführen von Verbindungen" (Connection Hijacking) und andere derartige Netzwerk-Angriffe effektiv zu unterbinden.

Die OpenSSH Suite beinhaltet das ssh Programm, das rlogin und telnet ersetzt, und scp , was rcp und ftp ersetzt. Genauso sind sshd ,das ist die Server-Seite von OpenSSH, und die anderen Basis-Werkzeuge wie ssh-add, ssh-agent, und ssh-keygen enthalten. OpenSSH unterstuetzt die Protokoll Versionen 1.3 und 1.5 (an Unterstützung für Protokoll-Version 2.0 wird gearbeitet).

OpenSSH wird entwickelt vom OpenBSD Projekt, und seine erste Implementation in ein Betriebsystem geschah in OpenBSD 2.6 (CDs können bestellt werden). Die Software wird ausserhalb der USA entwickelt, wobei Code aus ungefähr 10 Ländern genutzt wird, und ist für jeden frei nutzbar und wiederverwertbar-useable by everyone unter der BSD Lizenz.

Das OpenBSD Project verkauft ein OpenSSH tshirt.
