OpenSSH 2.5.2 vom 19. März 2001.
Es enthält Unterstützung für die SSH1 und SSH2 Protokolle.

OpenSSH ist eine FREIE Version der SSH Protokoll Suite von Netzwerk-Tools auf die sich eine steigende Anzahl von Leuten im Internet verlassen. Vielen Benutzer von telnet, rlogin, ftp und ähnlichen Programmen ist noch nicht aufgefallen, dass ihr Passwort dabei unverschlüsselt über das Internet übertragen wird. So ist es jedoch! OpenSSH verschlüsselt die Passwörter und die gesamte Verbindung um Mithören (Eavesdropping), das "Entführen von Verbindungen" (Connection Hijacking) und ähnliche Netzwerkangriffe zu unterbinden. Zusätzlich bietet OpenSSH eine Unzahl sicherer "tunneling" Möglichkeiten, genauso wie eine Unmenge von Authenfikationsmöglichkeiten.

OpenSSH beinhaltet das Programm ssh, das rlogin und telnet ersetz ersetztt, scp, das rcp ersetzt und sftp, das ftp ersetzt. Ausserdem sind der Server sshd und die Programme ssh-add, ssh-agent, ssh-keygen und sftp-server enthalten. OpenSSH unterstützt die SSH Protokollversionen 1.3, 1.5 und 2.0 .

OpenSSH wird hauptsächlich vom OpenBSD Projekt entwickelt und OpenBSD 2.6 ist das erste Betriebssystem, das OpenSSH integriert hat. Die Software wird ausserhalb der USA entwickelt. Dabei wird Programmcode aus ungefähr 10 Ländern genutzt. OpenSSH steht unter der BSD Lizenz und ist somit für jeden frei nutz- und wiederverwertbar.

Die Verwaltung von OpenSSH ist in zwei Teams aufgeteilt. Ein Team macht strikt OpenBSD-basierte Entwicklung, um Code zu entwickeln, der so sauber, einfach und sicher wie möglich ist. Das andere Team nimmt diese saubere Version und macht sie portabel, so dass sie auf vielen Betriebssystemen läuft.(diese ist als die p Version bekannt, und heisst z.B."OpenSSH 2.1.1p4"). Bitte klicke auf den für dein Betriebssystem passenden Link.

Das OpenBSD Projekt verkauft ein OpenSSH Tshirt und Poster. Die Einnahmen dieser Dinge helfen das Projekt zu unterstützen.

Wir empfehlen das O'Reilly SSH Buch Buch von Daniel Barrett und Richard Silverman als eine exzellente Referenz.
