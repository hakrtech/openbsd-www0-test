Systeme, die OpenSSH benutzen

Die folgenden Betriebssysteme und Produkte integrieren bekanntermaßen OpenSSH in ihre Basissysteme.

Die Liste ist in chronologischer Reihenfolge sortiert. Diejenigen, die es früher integriert haben, werden auch früher genannt.

  • OpenBSD
  • FreeBSD
  • BSDi BSD/OS
  • NetBSD
  • Computone
  • Stallion
  • Cygwin
  • e-smith Server und Gateway
  • Mac OS X in Version 10.1 und höher
  • HP Procurve Switch 4108GL und 2524/2512
  • IBM AIX
  • Sun Solaris 9 und später (SunSSH genannt)
  • SmoothWall
  • IPCop
  • SGI Irix
  • ThinLinc
  • Nokia IPSO
  • Content Services Switch 11500 Serie
  • Cisco SN 5400 Storage-Router-Serie
  • TopLayer IDS balancers
  • NTI SSH Serial Port Switch
  • Bluecoat (vormals Cacheflow) Proxy SG
  • Novell NetWare
  • Digi CM Console Server
  • Alcatel OmniSwitch
  • Dell PowerConnect L2- und L3-Switches
  • HP-UX (bekannt als HP-UX Secure Shell)
  • Packeteer PacketShaper 6.0 und höher.
  • Juniper Networks JUNOS
  • alle Linux-Systeme

Wenn wir ein System übersehen haben, das OpenSSH in sein Basissystem integriert, lass es uns bitte wissen (gemeint ist, dass OpenSSH standardmäßig auf CD oder bereits im System integriert ist, so dass der Anwender es nicht extra installieren muss).
