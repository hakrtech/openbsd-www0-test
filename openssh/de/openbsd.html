OpenSSH für OpenBSD

Da OpenSSH von der OpenBSD Gruppe entwickelt wird, ist OpenSSH Teil der Basis des Betriebssystems, beginnend bei OpenBSD Version 2.6. Es gibt also nichts, was man dem System hinzufügen oder ändern müsste. Nichtsdestotrotz möchtest du vielleicht aufgrund neuer Features ein Upgrade machen.

Du kanst die folgenden Versionen von OpenSSH auf deinen OpenBSD-Maschinen installieren:

  • OpenSSH 2.1: 8. Mai 2000

    Einen 'tar ball' gibt es unter ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openssh-2.1.tgz

    Es wird schwierig sein OpenSSH 2.1 auf OpenBSD 2.6 zu portieren, solange nicht jemand einen Patch schreibt, wie der der unten für OpenSSH 1.2.3.

  • OpenSSH 1.2.3: veröffentlicht am 6. März 2000

    Einen 'tar ball' gibt es unter ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openssh-1.2.3.tgz

    Wenn du auf OpenBSD 2.6 installierst, brauchst du auch diesen patch: http://www.openssh.com/openbsd26_1.2.3.patch.

    Um das ganze zu entpacken und auf deinem OpenBSD System zu installieren:

    # cd /usr/src/usr.bin
# tar xvfz .../openssh-1.2.3.tgz
# cd ssh
# patch -p1 < openbsd26_1.2.3.patch
# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc
Wenn du dich für möglicherweise vorhandene OpenSSH patches interessierst, sieh unter den OpenBSD errata pages nach.
