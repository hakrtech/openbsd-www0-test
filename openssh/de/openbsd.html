OpenSSH für OpenBSD

Da OpenSSH von der OpenBSD Gruppe entwickelt wird, ist OpenSSH Teil der Basis des Betriebssystems, beginnend ab OpenBSD Version 2.6 aufwärts. Es gibt also nichts, was man dem System hinzufügen oder ändern müsste. Nichtsdestotrotz möchtest du vielleicht aufgrund neuer Features ein Upgrade machen.

Du kannst die aktuellste OpenSSH Version von diesen ftp mirrors herunterladen.

Du kanst die folgenden Versionen von OpenSSH auf deinen OpenBSD-Maschinen installieren:

  • OpenSSH 3.0.1: 15. November 2001

    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openssh-3.0.1.tgz

    OpenSSH 3.0.1 kompiliert ohne Veränderungen auf OpenBSD 3.0 Systemen: 

    # cd /usr/src/usr.bin
# tar xvfz .../openssh-3.0.1.tgz
# cd ssh
# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc
    Dazu brauchst du aber einen 'HostKey /etc/ssh_host_dsa_key' in sshd_config

    Wenn du auf OpenBSD 2.9 installieren willst, brauchst du zusätzlich diesen Patch:
    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openbsd29_3.0.1.patch.

    Um ihn auszupacken und auf deinem OpenBSD 2.9 System zu installieren:

    # cd /usr/src/usr.bin
# tar xvfz .../openssh-3.0.1.tgz
# cd ssh
# patch -p0 < .../openbsd29_3.0.1.patch
# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc
    Wenn du auf OpenBSD 2.8 installieren willst, brauchst du zusätzlich diesen Patch:
    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openbsd28_3.0.1.patch.

    Um ihn auszupacken und auf deinem OpenBSD 2.8 System zu installieren:

    # cd /usr/src/usr.bin
# tar xvfz .../openssh-3.0.1.tgz
# cd ssh
# patch -p0 < .../openbsd28_3.0.1.patch# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc
    Wenn du auf OpenBSD 2.7 installieren willst, brauchst du zusätzlich diesen Patch:
    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openbsd27_3.0.1.patch.

    Um ihn auszupacken und auf deinem OpenBSD 2.7 System zu installieren: 

    # cd /usr/src/usr.bin
# tar xvfz .../openssh-3.0.1.tgz
# cd ssh
# patch -p0 < .../openbsd27_3.0.1.patch
# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc
    Hinweis: Der obige Quellcode kann nicht auf älteren Versionen von OpenBSD übersetzt werden, weil die IPv6 Unterstützung in der libc nicht komplett war.
