OpenSSH für OpenBSD

Da OpenSSH von der OpenBSD Gruppe entwickelt wird, ist OpenSSH Teil der Basis des Betriebssystems, beginnend bei OpenBSD Version 2.6. Es gibt also nichts, was man dem System hinzufügen oder ändern müsste.

Nichtsdestotrotz wurdem seitdem neuere Versionen von OpenSSH herausgebracht:

  • OpenSSH 1.2.3: veröffentlicht am 6. März 2000

    Einen tar ball bekommt man unter ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openssh-1.2.3.tgz

    Wenn du auf OpenBSD 2.6 installierst, brauchst du auch diesen patch: http://www.openssh.com/openbsd26_1.2.3.patch.

    Um das ganze zu entpacken und auf deinem OpenBSD System zu installieren:

    # cd /usr/src/usr.bin
# tar xvfz .../openssh-1.2.3.tgz
# cd ssh
# patch -p1 < openbsd26_1.2.3.patch
# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc
Wenn du dich für möglicherweise vorhandene OpenSSH patches interessierst, sieh unter den OpenBSD errata pages nach.
