Erscheinungen in den Medien

July, 2000

  • OpenSSH's Cinderella story, Mandrake User, 14. Juli 2000
    Tom Berger schreibt ein ausführliches OpenSSH Tutorium für neue User.

  • SSH 1.2.30 released, new restrictive license, Linux Weekly News, 13. Juli 2000
    Die letzte Version des SSH.COM's SSH1 Angebotes bekommt einen warmen Empfang. "If you haven't already moved over to using OpenSSH, available under a fully free license, this is a good time consider it". Es gibt weitere Kommentare in diesem Artikel in den wöchentlichen Sicherheitsseiten.

  • OpenSSH's Cinderella story, ZDNet News, 10. Juli 2000
    Veteranen-Autor Stephan Somogyi erzählt die Geschichte einer Gruppe von "less-than-gruntled" Entwicklern, die die einzige unbelastete OpenSource Implementation von SSH1 und SSH2 entwickelt haben.

  • Why do vendors ship us junk they wouldn't use?, Security Portal, 5. Juli 2000
    Kurt Seifried nimmt die Linux Anbieter aufs Korn und fragt warum -- neben anderen Sünden -- Telnet bei Lieferung immer standardmässig eingeschaltet ist und SSH nicht.

    Juni 2000

  • OpenSSH 2.1.1 released, Linux Weekly News, 15. Juni 2000
    Als Teil der Veröffentlichungserklärung interviewt der Chefredakteur Liz Coolbaugh Theo de Raadt über OpenSSH, das nun etwas wie ein "Erfolg des Untergrundes" ist.

  • Vince's guide to OpenSSH, LAN Systems Gazette, 10. Juni 2000
    Vince Hillier gibt eine Anleitung wie man OpenSSH auf verschiedenen Linux Distributionen erzeugt und installiert.

  • Exorcise FTP, Telnet and other evil daemons, ZDNet Help & How-To; How-To, Juni 2000
    Henry Kingman gibt einige grundlegende inetd Sicherheitstipps und sagt man solle Telnet und FTP mit modernen Tools wie ssh und scp ersetzen. Aufgrund der "seltsamen" Lizenz von SSH sollten kommmerzielle User sich OpenSSH oder lsh ansehen, schreibt er.

    Mai 2000

  • SSH: From Secure Administration to Virtual Private Networking, ISP Planet, 16. Mai 2000
    "Wenn du immernoch *NIX Server über das Internet mit Hilfe von rsh oder Telnet administrierst, hör' jetzt damit auf", sagt die Autorin Lisa Phifer im Vorwort ihres Artikels. Sie fährt dann damit fort, dass es keine grosse Sache ist einen OpenSSH 2.1 (SSH2) Server zu installieren und testet dann die Kompatibilität mit drei kommerziellen Windows Clients. Sie betrachtet auch einfache VPN Konfigurationen.

  • sufficiently advanced technology: the gathering, NTK Now, 12. Mai 2000
    Dieses kompromisslose Online-Magazine ehrt die "Knights of OpenSSH" ("Ritter von OpenSSH") für die Möglichkeit, von unsicheren cyber-cafe Terminals zu Servern zu connecten. Der Draht ist sicher ,aber was ist mit den verflixten 'keystroke-loggers' ? Kein Problem, nutzte doch einfach die S/KEY-Einmal-Passwörter mit ssh.

  • OpenSSH unterstützt jetzt das SSH 2 Protokoll, Linux Weekly News, 4. Mai 2000
    Einmal mehr schlägt 'Linux Weekly News' den Rest der Medien und merkt an, dass OpenSSH jetzt dass SSH 2 Protokoll unterstützt. Wir rühmen sie dafür, dass sie die Geschichte von SSH 2 korrekt wiedergeben...

    März 2000

  • OpenSSH - sicherer und weniger restriktiv, Das FreeBSD Diary, 19. März 2000
    Eine weitere Diskussion über die Benutzung von OpenSSH, diesesmal speziell unter FreeBSD.

    Februar 2000

  • All About SSH - Part II: OpenSSH, Security Portal, 28. Februar, 2000
    Seán Boran schliesst seinen Überblick über SSH mit einem Artikel über OpenSSH auf OpenBSD, Linux und Solaris. Er ist sehr angetan vom OpenSSH-Projekt und seinen Zielen.

  • All About SSH - Part 1, Security Portal, 14. Februar, 2000
    Ein zweiteiliger Artikel über SSH vom Security Consultant Seán Boran. Teil eins erklärt was SSH ist, warum man es benutzen sollte und wie man es auf seinem Linux- oder BSD-System installiert. Der Artikel gibt auch einen Überblick über die verschiedenen kommerziellen und freien Implementationen inklusive OpenSSH. Teil zwei, der später erscheinen wird, soll sich auf OpenSSH konzentrieren.

  • Open Source Security Tools, Information Security, Februar 2000
    OpenSSH wird als eines der vielen Open Source Security Tools aufgeführt.

  • OpenBSD 2.6 - new features, Security Portal, 9. Februar, 2000
    OpenSSH wird als Teil von Kurt Seifried's Überblick über OpenBSD 2.6 erwähnt.

    Januar 2000

  • Opening up, government style, ZDNet, January 24, 2000
    Der Linux Kolumnist Evan Leibovitch diskutiert die möglichen Verzweigungen von Open Source Software nach den jüngsten Lockerungen der US-Krypto-Export Gesetze. Wenn auch Linux-Distributionen in der Lage sind, starke Kryptografie zu exportieren, macht es das RSA-Patent nach wie vor schwer OpenSSH und die OpenSSL-Bibliothek in den USA zu benutzen.

    Dezember 1999

  • How SSH was freed, Daemonnews, 1. Dezember, 1999
    Daemonnews berichtet über die Entstehung von OpenSSH.

    November 1999

  • Turning on the Zedz, ZDNet, 2. November, 1999
    Der Linux Kolumnist Evan Leibovitch taucht in das byzantinische Reich der US-Krypto-Export-Regulierungen ein und erwähnt OpenSSH als eins der frei erhätlichen Pakete, die zwar in die USA importiert, aber nicht von dort aus exportiert werden dürfen.

    Auf Freshmeat erscheint die offizielle OpenSSH Pressemitteilung.

    Linux Weekly News war die erste nicht-BSD News-Agentur, die von der Existenz von OpenSSH berichtet, das mit OpenBSD 2.6 ausgeliefert wird.

  • The OpenBSD project's announcement of OpenSSH, 25. Oktober, 1999
    Die offizielle Ankündigung , die das OpenBSD Projekt gemacht hat.
