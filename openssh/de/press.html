    Januar 2001

  • Top Ten Secure Shell FAQs, O'Reilly Network, 25. Januar 2001
    Die Top-Ten-Fragen und Benutzungshinweise von Richard Silverman, Ko-Autor von SSH, The Secure Shell: The Definitive Guide, auch bekannt als das Schnecken Buch.

  • OpenSSH 2.3 And SSH Secure Shell 2.4, Planet IT, January 9, 2001
    OpenSSH 2.3 vs. SSH Communications 2.4 unter Linux, ein Überblick von Pete Loshin.

    Dezember 2000

  • Red Hat 7.0 Review, FreeOS.com, 24. Dezember 2000
    Gerade noch rechtzeitig für den Einkaufstrip gibt's jetzt Red Hat Linux inklusive OpenSSH.

  • The 101 Uses of OpenSSH: Part I, Linux Journal, 14. Dezember 2000
    Der Kolumnist Mick Bauer vom 'Paranoid Penguin' gibt eine kurze Beschreibung der Geschichte von OpenSSH und schreibt über die coolen Sachen, die man damit machen kann.

  • Securing Linux: Part 2, FreeOS.com, December 13, 2000
    In einer fortlaufenden Serie von HowTo-Artikeln listet Trevor Warren einige proaktive Sicherheitsmassnahmen für Linux auf, darunter auch OpenSSH.

  • Interview with Kurt Seifried of SecurityPortal.com, Linux Security Brasil, 4. Dezember 2000
    In einem Interview von Renato Murilo Langona listet der Sicherheits-Kolumnist und Forscher Kurt Seifried OpenSSH unter seinen unverzichtbaren Sicherheits-Werkzeugen auf.

  • Protect Your Data with OpenSSH, Linux Magazine, Dezember 2000
    Eine Einführung in sichere Kommunikation mittels OpenSSH von Joe "Zonker" Brockmeier (zur Zeit nur auf Papier).

    November 2000

  • Keep Tux Safe, Sm@rt Partner, ZD Net, 30. November 2000
    "Alle "r" Dienste (rsh, rlogin and rcp) sollten bei Bedarf komplett ersetzt werden, und zwar durch OpenSSH. Wenn möglich sollten auch telnet und ftp ersetzt werden" , schreibt David Raikow.

  • Replacing Telnet; OpenSSH, a secure alternative , FreeOS, 21. November 2000
    Mayank Sarup gibt eine Schritt-für-Schritt-Anleitung zum Installieren von OpenSSH und seiner zugehörigen Unterstützung Zlib und OpenSSL.

  • Building OpenSSH 2.3.0 on Mac OS X Public Beta, stepwise.com, 18. November 2000
    Noch eine Plattform für OpenSSH! Scott Anguish merkt an, dass das System mit einer nicht-für-den-kommerziellen-Gebrauch-Version von SSH ausgeliefert wird, also führt er den Leser durch die notwendigen Schritte zur Installation und Konfiguration von OpenSSH.

    Oktober 2000

  • Installing and Configuring OpenSSH, SysAdmin Magazine, Oktober 2000
    Matt Lesko gibt eine knappe, aber gründliche Einführung in OpenSSH, seine Anfänge, Installation und Abhängigkeiten und seine Benutzung. Ausserdem werden Zusätze und einige Windows-Clients abgedeckt.

    September 2000

  • Kostenlos geschützt, FreeX Magazin, 4. Quartal 2000 (in German)
    Dr. Bernhard Röhrig schreibt einen kompletten technischen Artikel über OpenSSH, inklusive detaillierter Anweisungen, wie man es zusammen mit den Abhägigkeiten unter Linux installiert, wie es funktioniert und wie man es benutzt. Er deckt dabei auch scp und port tunnelling ab.
    [FreeX's Homepage druckt leider die Artikel nicht online ab..]

  • RSA Algorithm Released: Update, Security Portal, 7. September 2000
    Kurt Seifried leistet beachtliche Arbeit beim Sortieren all der Verwirrung rund um das RSA Patent, und um RSA Inc.'s Ankündigung, dass sie RSA zu Public Domain machen. Er nennt auch OpenSSH als eines von vielen Programmen, die von der zusätzlichen Freiheit profitieren werden.

  • Complete Reference Guide to Creating a Remote Log Server, Linux Security, 22. August 2000
    OpenSSH ist eine Hauptkomponente in einem sicheren 'remote log server', hier in allen Details von Eric Hines und Fyodor Yarochkini beschrieben.

  • SSH Communications changes ssh licence , Linux Weekly News, 17. August 2000
    SSH hat seine Lizenz erneut geändert! Die neue "free software people can use our software freely" Lizenz ist der neuen Motif Lizenz sehr ähnlich. Vielleicht haben sie realisiert, dass du den Leuten, die die Software sowieso nicht kaufen, auch so geben kannst. In der Zwischenzeit entscheiden sich sowieso die meisten Leute für OpenSSH.

  • Red Hat boosts Linux security, Network World Fusion, 14. August 2000
    OpenSSH ist Teil von Red Hat's Plänen die Sicherheit ihrer kommenden version 7.0  pinstripe  Linux zu verbessern.

  • The Truth About Security, The OutRider Computing Journal, August 2000
    Jason Fink schreibt eine minimalistische Anleitung in Bezug auf Sicherheit für alle jene, die zu wenig Zeit für die Feinheiten haben. Eine seiner Empfehlungen ist sichere Protokolle für Internet Kommunaktionen zu benutzen, d.h. OpenSSH.

    July, 2000

  • Interview: Theo de Raadt, Sécurité.org, 26. Juli 2000
    Nicolas Fischbach hat Theo de Raadt auf der CanSecWest in Vancouver vor einer Weile getroffen, und das daraus resultierende Interview diskutiert "Secure by Default" und die Schöpfung von OpenSSH.

  • SSH, Mandrake User, 14. Juli 2000
    Tom Berger schreibt ein ausführliches OpenSSH Tutorium für neue User.

  • SSH 1.2.30 released, new restrictive license, Linux Weekly News, 13. Juli 2000
    Die letzte Version des SSH.COM's SSH1 Angebotes bekommt einen warmen Empfang. "If you haven't already moved over to using OpenSSH, available under a fully free license, this is a good time consider it". Es gibt weitere Kommentare in diesem Artikel in den wöchentlichen Sicherheitsseiten.

  • OpenSSH's Cinderella story, ZDNet News, 10. Juli 2000
    Veteranen-Autor Stephan Somogyi erzählt die Geschichte einer Gruppe von "less-than-gruntled" Entwicklern, die die einzige unbelastete OpenSource Implementation von SSH1 und SSH2 entwickelt haben.

  • Why do vendors ship us junk they wouldn't use?, Security Portal, 5. Juli 2000
    Kurt Seifried nimmt die Linux Anbieter aufs Korn und fragt warum -- neben anderen Sünden -- Telnet bei Lieferung immer standardmässig eingeschaltet ist und SSH nicht.

  • Linux in the home - everywhere in the home, OS Opinion, July 2000
    Peter Colijn, ein High School Student in Calgary, AB, erklärt wie Linux eine preiswerte Lernplattform für das Lernen zu Hause darstellen kann. "It's also running OpenSSH so I can login from my school to check my e-mail (yes, our high school has about 10 Linux boxes, also running OpenSSH)", schreibt er.

    Juni 2000

  • OpenSSH 2.1.1 released, Linux Weekly News, 15. Juni 2000
    Als Teil der Veröffentlichungserklärung interviewt der Chefredakteur Liz Coolbaugh Theo de Raadt über OpenSSH, das nun etwas wie ein "Erfolg des Untergrundes" ist.

  • Vince's guide to OpenSSH, LAN Systems Gazette, 10. Juni 2000
    Vince Hillier gibt eine Anleitung wie man OpenSSH auf verschiedenen Linux Distributionen erzeugt und installiert.

  • Exorcise FTP, Telnet and other evil daemons, ZDNet Help & How-To; How-To, Juni 2000
    Henry Kingman gibt einige grundlegende inetd Sicherheitstipps und sagt man solle Telnet und FTP mit modernen Tools wie ssh und scp ersetzen. Aufgrund der "seltsamen" Lizenz von SSH sollten kommmerzielle User sich OpenSSH oder lsh ansehen, schreibt er.

  • Building a Secure Gateway System, Linux Gazette, Juni 2000
    Chris Stoddard beschreibt detailliert die Installation eines sicheren Gateway-Systems inklusive OpenSSH.

    Mai 2000

  • SSH: From Secure Administration to Virtual Private Networking, ISP Planet, 16. Mai 2000
    "Wenn du immernoch *NIX Server über das Internet mit Hilfe von rsh oder Telnet administrierst, hör' jetzt damit auf", sagt die Autorin Lisa Phifer im Vorwort ihres Artikels. Sie fährt dann damit fort, dass es keine grosse Sache ist einen OpenSSH 2.1 (SSH2) Server zu installieren und testet dann die Kompatibilität mit drei kommerziellen Windows Clients. Sie betrachtet auch einfache VPN Konfigurationen.

  • sufficiently advanced technology: the gathering, NTK Now, 12. Mai 2000
    Dieses kompromisslose Online-Magazine ehrt die "Knights of OpenSSH" ("Ritter von OpenSSH") für die Möglichkeit, von unsicheren cyber-cafe Terminals zu Servern zu connecten. Der Draht ist sicher ,aber was ist mit den verflixten 'keystroke-loggers' ? Kein Problem, nutzte doch einfach die S/KEY-Einmal-Passwörter mit ssh.

  • OpenSSH unterstützt jetzt das SSH 2 Protokoll, Linux Weekly News, 4. Mai 2000
    Einmal mehr schlägt 'Linux Weekly News' den Rest der Medien und merkt an, dass OpenSSH jetzt dass SSH 2 Protokoll unterstützt. Wir rühmen sie dafür, dass sie die Geschichte von SSH 2 korrekt wiedergeben...

    Januar 2000

  • Opening up, government style, ZDNet, January 24, 2000
    Der Linux Kolumnist Evan Leibovitch diskutiert die möglichen Verzweigungen von Open Source Software nach den jüngsten Lockerungen der US-Krypto-Export Gesetze. Wenn auch Linux-Distributionen in der Lage sind, starke Kryptografie zu exportieren, macht es das RSA-Patent nach wie vor schwer OpenSSH und die OpenSSL-Bibliothek in den USA zu benutzen.
