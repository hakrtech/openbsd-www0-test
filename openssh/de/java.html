Java

Die folgenden ,freien' Clients werden für die Zusammenarbeit mit OpenSSH von Java-Clients aus empfohlen:

  • MindTerm 1 ist eine Nur-SSH1-Implementierung. GPL-Lizenz.

    "MindTerm ist ein vollständig freies(*) SSH- (Protokoll 1) Clientprogramm, das in 100% purem Java geschrieben wurde (nicht-zertifiziert). Es wird mit Sourcecode ausgeliefert, da wir glauben, dass sicherheitsrelevanter Software ansonsten nicht getraut werden kann. Mindterm kann als standalone-Programm oder als Applet in einer Webseite benutzt werden. Es kann mit oder ohne GUI (Graphical User Interface) benutzt werden (stand-alone). Es hat zwei besondere Funktionalitäten, die andere Clients nicht haben: eingebaute scp-Dateiübertragung und einen speziellen ftp-Tunnel, der mit ,ordinären' ftpds hinter sshd arbeiten kann."

    HINWEIS: neuere Versionen sind nicht frei.

  • SSH Plugin ist eine Nur-SSH1-Implementierung von Matthias L. Jugel und Marcus Meißner. GPL-Lizenz.

    "Diese Implementierung von SSH bietet IDEA- und RSA/PKCS#1-Verschlüsselung und wurde in seiner Originalform 1998 von Cédric Gourio geschrieben. Er nutzte einen Teil des alten Java Telnet Applets, um seinen eigenen Java-SSH-Client zu schreiben, und daher war der Code unserem alten Applet sehr ähnlich."

  • Ganymed-SSH2. BSD-artige Lizenz.

    "Der Ganymed-SSH2 für Java-Bibliothek (implementiert das SSH-2-Protokoll) wird in Ganymed und einigen anderen Projekten an der ETH Zürich genutzt. Es ermöglicht dir, dich von deinen Java-Programmen aus an den SSH-Servern anzumelden."
