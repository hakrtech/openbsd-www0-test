FTP OpenSSH

Code source OpenSSH 3.6.1 de base : disponible depuis le 1er Avril 2003

Le code source de base est conçu spécifiquement pour OpenBSD, afin de le rendre simple, propre, minimal et auditable. Une équipe séparée convertit OpenSSH en version portable fonctionnant sur tous les systèmes d'exploitation.

Nous le répétons : Pour d'autres systèmes, consultez le fichier de la version portable !
