Palm OS

Les clients "libres" suivants sont recommandés pour interagir avec OpenSSH sous Palm OS:

  • Top Gun ssh en utilisant pilotSSLeay

    "Top Gun ssh est un client ssh (version 1) pour les Palms utilisant Palm OS et ayant une pile TCP/IP. Ceci inclut le Palm Pilot Pro, Palm V, et les modèles Workpad et Visor. Cela n'inclut pas le Palm VII; Apparemment cette machine ne semble pas supporter TCP/IP."

  • pssh

    "pssh est un client SSH 2 libre pour Palm OS. Il fonctionne sur les assistants personnels dotés de Palm OS 5, un processeur ARM, et un écran haute résolution 320x320 ou plus."

    Supporte SSHv2. Le code source est disponible. En partie basé sur OpenSSH.

  • TuSSH

    "TuSSH est un client SSH pour les machines Palm OS. Il devrait en principe fonctionner avec toute machine équipée de Palm OS 4 ou supérieur. Il pourrait même fonctionner avec les machines sous OS 3.5 mais aucun test ni support n'est prévu pour ces machines."

    Supporte SSHv1 et supporte partiellement SSHv2. Utilisation libre mais les sources ne sont pas disponibles.
