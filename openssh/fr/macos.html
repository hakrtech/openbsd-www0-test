Mac OS

On recommande les clients suivants pour interagir avec OpenSSH à partir de machines Mac OS. Notez que Mac OS X inclut OpenSSH par défaut.

  • NiftyTelnet 1.1 SSH implémente SSH1 uniquement avec un programme type scp. Ecrit par Jonas Wallden.

    "NiftyTelnet 1.1 SSH r3 est une version améliorée de l'application NiftyTelnet de Chris Newman qui ajoute le support du chiffrage des sessions en utilisant le protocole SSH (Secure Shell). Lisez s'il vous plaît le fichier Readme inclut avant de distribuer cette version."

  • MacSSH implémente SSH2 uniquement.

    "MacSSH est une version modifié de BetterTelnet avec le support SSH2.[...] Le seul client SSH2 pour MacOS que j'ai pu trouver est un produit commercial qui coute plus de 100$, et celui-ci plante mon Mac quand je ferme une session... Puisque le mieux est de faire les choses soi-même, voici MacSSH."

  • Fugu est une implémentation de SFTP et de SCP pour Mac OS X.

    "Fugu SSH est une interface graphique pour l'application Secure File Transfer (SFTP) de OpenSSH. Il vous permet de bénéficier de la sécurité de SFTP sans devoir sacrifier la facilité d'utilisation offerte par une interface utilisateur graphique. Fugu contient aussi le support des transferts de fichiers SCP, et la possibilité de créer des tunnels sécurisés à travers SSH".

  • Cyberduck est un navigateur SFTP et FTP pour Mac OS X.

    "Cyberduck est un navigateur libre SFTP (SSH Secure File Transf) et FTP sous licence GPL. Il a été conçu dès le départ pour privilégier l'ergonomie en ayany une interface graphique utilisateur similaire pour la navigation SFTP et FTP. Plusieurs connexions sont supportées. [...] Des technologies système essentielles telles que Keychain et Rendezvous sont supportées."
