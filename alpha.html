OpenBSD/alpha

History and Status:

The alpha port has no architecture specific changes from the NetBSD/alpha port. However, it's verified and running stably. Currently Niklas Hallqvist <niklas@openbsd.org> is maintaining the port. Like in NetBSD, the port won't compile with only in-tree tools, but that is being addressed.

Projects (in no particular order):

  • Build a binary snapshot release.
  • Make the port self-hosting (i.e. only using in-tree tools).
  • Add ISA DMA support (needed for floppies & wss).
  • Add floppy support.
  • Add WSS audio support.
  • DDB support.
