Hardware wanted.

The following list outlines hardware that the OpenBSD project could use. Some of this is being requested so that developers can improve or add driver support for devices which are rare; other requests are for hardware that will improve our development environment. In either case, we think we are being reasonable at asking for these devices.

In each case, please contact Theo de Raadt before simply shipping hardware around. (So that we can decide where the hardware needs to be shipped to).

ps. as an extra incentive, Theo is willing to trade quantities of stickers for some of these things...

  • Alpha hardware:
    • Some alpha hardware in Europe and North America.
    • We are especially looking for alphas better than the Multia and axppci33 models.
    • If we do not get some of these, we are not going to ship OpenBSD/alpha on the 2.5 CDROM (it isn't worth our effort).

  • A Pentium-2/450 motherboard.

  • Sparc gear:
    • Some SBUS ultrasparc machines
    • Some PCI ultrasparc machines
    • SBUS fas fast-wide scsi card (hme or non-hme card)
    • SBUS pcmcia adapters (using STP4020 or similar chipset).
    • SBUS bpp card
    • MBus ROSS HyperSPARC cpu module
    • A fast SS5 or SS20 machine.

  • PCI gear:
    • RealTek RTL8129 based ethernet card
    • PCI adaptec 789x-based scsi boards
    • Adaptec AIC-7890/7891/7895 and AIC-7860 programmers manual
    • ASIX AX88140A based ethernet cards, eg. ASIX AX88140A 10/100BaseTX
    • ADMtek AL981 based ethernet cards
    • Any supported PCI SCSI board with (at least) one narrow bus for RAIDframe testing.
    • NetGear GA620 gigabit ethernet card, or other Alteon-based board.
    • AdvanSys 3940U2W SCSI cards (preferably two: one for PC and one for Mac development).
    • S3 SonicVibes PCI audio card.

  • Infrastructure requirements:
    • A fast/wide differential scsi drive and cable
    • A small 1000mbit etherswitch, for gigabit network development.
    • a fairly large UPS (Calgary)
    • a medium sized UPS for the OpenBSD mail and lists machine (Boulder Colorado)

  • Amiga 68060 card, fit for an A2000 or A3000, preferably with fast scsi and slots for cheap memory.

  • BT445S VL SCSI card.
  • Intel i82586/i82596-based ethernet cards (i.e. BusLogic Bt763E EISA, Intel EtherExpress Pro/10 PCI, Intel Flash32 EISA, Cogent eMaster EM935/EM932 EISA, AT&T StarLAN 10, AT&T EN100, 3Com 3C507)

  • HPPA gear:
    • HP 9000 J2x0 w/ two cpus (J210 is HP A4081A, CPU is HP A4278A)
    • HP 9000/725/100 (upgrade for 725/50, HP A4146A) and/or 712/100 (upgrade for 712/80, HP A4222A)
    • HP 9000/712 2nd RS-232 port card (HP A4013A)
    • HP 9000/712 lan and RS-232 card (HP A4014A)
    • HP 9000/712 multiprotocol interface card (HP A4015A)
    • HP 9000/712 2nd graphics card (HP A2878A)
    • HP 9000/712 2nd graphics w/ 2nd LAN card (HP A4212A)
    • HP 9000/712 TeleShare POTS iface, analog RJ11C (HP A4012A, HP A4012AJ)
    • HP 9000/712 IEEE488 card (TAMS 50488, see TAMS)
    • Precision I/O Architecture Reference Specification (the latest version)

  • Motorola mvme68k/mvme88k gear:
    • MVME172 Single Board Computer. (68060 processor)
    • MVME177 Single Board Computer. (68060 processor)
    • MVME327 SCSI VME Board plus documentation.
    • Any type of VME 6U chassis.
    • 1 to 4 gig Non-Wide SCSI drives. (These would help enormously)

  • ISAPNP cards. All sorts wanted, including:
    • WSS audio card.
    • CSxxx audio card, that definately has MIDI support.
    We want as many as possible, so that our ISAPNP support can become very rock-solid.

  • PCMCIA: Any bizzare unused devices, including:
    • Xircom RealPort 10/100 ethernet + 56K modem + GSM
    • EtherExpress 100 pcmcia.
    • PC Card Specifications (PC Card Standard - Release 6.1)

  • Miscellaneous
    • External MII (PHY) interface (HPDB40 connector)
    • A MIDI playback device, so that the MIDI support can be kept solid.

  • POSIX.1 specifications document
