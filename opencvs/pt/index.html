OpenCVS será liberado em breve.

OpenCVS é uma implementação LIVRE do "Concurrent Versions System", o popular software de controle de revisões de código fonte aberto. Pode ser usado tanto como cliente como servidor para repositórios, e fornece acesso granular sobre os dados armazenados no repositório. Seu propósito é ser tão compatível quanto possível com outras implementações de CVS, exceto quando características particulares reduzirem a segurança total do sistema.

O projeto OpenCVS foi iniciado depois das discussões sobre as últimas vulnerabilidades divulgadas do CVS do projeto GNU. Embora o CVS seja amplamente utilizado, seu desenvolvimento estava paralizado nos últimos anos e muitos problemas de segurança surgiram desde então, tanto na implementação quanto nos mecanismos.

O OpenCVS é desenvolvido pelo Projeto OpenBSD. Este software é livre para ser usado por qualquer um sob a licença BSD.

O projeto OpenBSD vende CDs, Camisetas e Pôsteres. As vendas desses itens ajudam a financiar o desenvolvimento.
