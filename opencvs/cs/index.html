OpenCVS bude brzy vydáno.

OpenCVS je SVOBODNÁ implementace Concurrent Versions System, nejpopulárnějšího open source software pro správu verzí. Může být použito jako client i jako server pro repozitáře a nabízí kontrolu přístupu k datům uloženým v repozitáři. Snaží se být kompatibilní s ostatními CVS implementacemi, mimo případy, kdy určitá vlastnost snižuje celkovou bezpečnost systému.

OpenCVS projekt odstartoval po diskuzích týkajících se posledních zranitelností nalezených v GNU CVS. I přesto, že je CVS široce využíváno, tak jeho vývoj v posledních letech většinou stagnoval a objevilo se mnoho bezpečnostních incidentů jak v implementaci, tak i v samotných mechanismech.

OpenCVS je vyvíjen OpenBSD projektem. Software je volně použitelný a opakovaně využitelný kýmkoliv dle BSD licence.

OpenBSD projekt prodává CD, Trička a Plakáty. Prodej těchto předmětů pomáhá sponzorovat vývoj.
